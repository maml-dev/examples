# 436 - Theme Config

Design system theme configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  name: "extra-large Theme"
  # Color palette
  colors: {
    primary: "#beec3e"
    secondary: "#ca4766"
    accent: "#e78ef7"
    background: "#ffffff"
    text: "#e6edf3"

    error: "#dc3545"
    warning: "#ffc107"
    success: "#28a745"
  }

  # Typography
  typography: {
    font_family: "system-ui, sans-serif"
    base_size: 14
    line_height: 1.6
    headings: {
      font_family: "\"Playfair Display\", serif"
      weight: 700
    }
  }

  spacing: {
    unit: 8
    scale: [
      0
      4
      8
      16
      24
      32
      48
    ]
  }
  border_radius: 12
}
```
