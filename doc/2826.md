# 2826 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "roi"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 15 # seconds
    labels: {
      env: "prod"
      region: "eu-west"
      team: "sports"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 89
    memory_threshold: 92
    latency_p99_ms: 793
    error_rate: 0.01
    notification: {
      channel: "#ops-alerts"
      severity: "info"
    }
  }
}
```
