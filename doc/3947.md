# 3947 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "editor"
      permissions: [
        "delete"
        "view_analytics"
        "read"
      ]
      max_seats: null
    }

    {
      name: "billing"
      permissions: [
        "delete"
        "export_data"
        "read"
        "manage_billing"
      ]
      max_seats: null
    }

    {
      name: "moderator"
      permissions: [
        "read"
        "write"
        "manage_users"
        "export_data"
      ]
      max_seats: 22
    }

    {
      name: "support"
      permissions: [
        "manage_users"
        "delete"
      ]
      max_seats: null
    }
  ]
}
```
