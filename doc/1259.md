# 1259 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "jam-packed-lender.org"
  records: [
    {
      type: "NS"
      name: "mail"
      value: "160.255.243.181"
      ttl: 86400
    }

    {
      type: "AAAA"
      name: "api"
      value: "154.32.74.174"
      ttl: 3600
    }

    {
      type: "TXT"
      name: "staging"
      value: "hefty-heating.biz"
      ttl: 300
    }

    {
      type: "CNAME"
      name: "www"
      value: "v=spf1 include:_spf.medium-commodity.org ~all"
      ttl: 300
    }

    {
      type: "MX"
      name: "@"
      value: "which-reservation.net"
      ttl: 300
    }
  ]
}
```
