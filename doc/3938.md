# 3938 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.sociable-sand.org"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "1m"
    max_requests: 9725
    by: "api_key"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://possible-galoshes.name"
      "https://app.icy-yeast.net"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 86400
  }

  cache: {
    enabled: true
    ttl: 1218
    store: "memory"
  }
}
```
