# 455 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "billing"
      permissions: [
        "manage_users"
        "delete"
        "manage_billing"
        "export_data"
      ]
      max_seats: 35
    }

    {
      name: "editor"
      permissions: [
        "delete"
        "view_analytics"
        "manage_users"
        "manage_billing"
        "read"
      ]
      max_seats: null
    }

    {
      name: "billing"
      permissions: [
        "delete"
        "manage_billing"
        "export_data"
        "write"
        "view_analytics"
      ]
      max_seats: null
    }
  ]
}
```
