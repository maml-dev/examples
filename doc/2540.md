# 2540 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "flat-web"
    namespace: "production"
    labels: {
      app: "desk"
      tier: "data"
    }
  }

  # Deployment spec
  spec: {
    replicas: 4
    container: {
      image: "summary:6.10.9"
      port: 8080
      resources: {
        cpu_limit: "250m"
        memory_limit: "256Mi"
      }
    }

    health_check: {
      path: "/health"
      interval: 30
      timeout: 5
    }
  }
}
```
