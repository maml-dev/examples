# 840 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "beloved-gastropod.name"
  inbound: {
    mx_records: [
      "mx1.beloved-gastropod.name"
      "mx2.beloved-gastropod.name"
    ]
    spam_filter: {
      enabled: true
      threshold: 7
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.beloved-gastropod.name"
    port: 25
    tls: true
    rate_limit: {
      per_minute: 203
      per_hour: 1027
    }
  }
}
```
