# 1013 - Email Config

SMTP email service settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  smtp_host: "smtp.specific-saw.org"
  smtp_port: 25
  use_tls: true
  from_name: "Robel, Little and Marvin"
  from_email: "noreply@specific-saw.org"

  # Rate limiting
  max_batch_size: 206
  retry_interval: 74
}
```
