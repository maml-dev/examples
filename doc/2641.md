# 2641 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "vim"
  shortcuts: [
    {
      key: "Cmd+Shift+L"
      action: "save"
      when: "terminal"
    }

    {
      key: "Alt+Up"
      action: "command_palette"
      when: "terminal"
    }

    {
      key: "Ctrl+D"
      action: "toggle_comment"
      when: "global"
    }
  ]
}
```
