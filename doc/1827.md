# 1827 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "tender-resource.org"
  inbound: {
    mx_records: [
      "mx1.tender-resource.org"
      "mx2.tender-resource.org"
    ]
    spam_filter: {
      enabled: true
      threshold: 7
      action: "reject"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.tender-resource.org"
    port: 465
    tls: true
    rate_limit: {
      per_minute: 458
      per_hour: 4885
    }
  }
}
```
