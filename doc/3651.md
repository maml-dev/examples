# 3651 - Cron Jobs

Scheduled task definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  timezone: "Europe/London"
  jobs: [
    {
      name: "send-reports"
      schedule: "0 0 * * *"
      command: "node scripts/cleanup.js"
      enabled: true
    }

    {
      name: "rotate-logs"
      schedule: "0 0 * * *"
      command: "pg_dump $DATABASE_URL"
      enabled: false
    }

    {
      name: "cleanup-temp"
      schedule: "*/15 * * * *"
      command: "node scripts/cleanup.js"
      enabled: true
    }

    {
      name: "backup-db"
      schedule: "0 * * * *"
      command: "node scripts/cleanup.js"
      enabled: true
    }

    {
      name: "cleanup-temp"
      schedule: "30 6 * * 1-5"
      command: "./bin/sync --all"
      enabled: true
    }

    {
      name: "refresh-cache"
      schedule: "30 6 * * 1-5"
      command: "./bin/sync --all"
      enabled: false
    }
  ]
}
```
