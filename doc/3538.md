# 3538 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "pretty-import.org"
  inbound: {
    mx_records: [
      "mx1.pretty-import.org"
      "mx2.pretty-import.org"
    ]
    spam_filter: {
      enabled: true
      threshold: 7
      action: "quarantine"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.pretty-import.org"
    port: 465
    tls: true
    rate_limit: {
      per_minute: 110
      per_hour: 8223
    }
  }
}
```
