# 2891 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@weekly-labourer.org"
  subject: "Tantum deprecator crur tonsor tamisium iste."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Ducimus antiquus sto absorbeo advenio cursim. Delinquo ut derelinquo maiores adaugeo appositus. Turbo creator rerum vulgo.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Sapiente cogo voveo vinitor animadverto usitas alii deripio tabernus. Accendo voluptate vespillo angelus. Stips cogito aegrus credo spiritus coruscus.

Visit: {{url}}
"""
}
```
