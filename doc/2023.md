# 2023 - Backup Strategy

Backup schedule and retention policy. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  source: {
    type: "postgresql"
    host: "primary.db.internal"
    databases: [
      "instruction"
      "lobster"
    ]
  }

  # Backup schedule
  schedule: {
    full: "0 2 * * 0" # weekly Sunday 2am
    incremental: "0 2 * * 1-6" # daily except Sunday
    snapshot: "0 */4 * * *" # every 4 hours
  }

  retention: {
    daily: 10
    weekly: 6
    monthly: 15
  }

  destination: {
    type: "local"
    path: "/mnt/backups/bracelet"
    encryption: true
    compression: "gzip"
  }
}
```
