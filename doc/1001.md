# 1001 - DNS Config

DNS zone configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "diligent-verve.org"
  ttl: 86400 # seconds
  nameservers: [
    "ns1.bossy-agreement.name"
    "ns2.lumbering-reach.com"
  ]

  soa_email: "admin@super-cycle.name"
  refresh: 14400
  retry: 600
  expire: 604800
}
```
