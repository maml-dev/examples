# 535 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "convergence"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 60 # seconds
    labels: {
      env: "staging"
      region: "ap-south"
      team: "health"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 95
    memory_threshold: 87
    latency_p99_ms: 688
    error_rate: 0.05
    notification: {
      channel: "#incidents"
      severity: "info"
    }
  }
}
```
