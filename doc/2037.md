# 2037 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "expansion-worker"
    namespace: "production"
    labels: {
      app: "venom"
      tier: "data"
    }
  }

  # Deployment spec
  spec: {
    replicas: 5
    container: {
      image: "porter:4.5.10"
      port: 8080
      resources: {
        cpu_limit: "1000m"
        memory_limit: "512Mi"
      }
    }

    health_check: {
      path: "/healthz"
      interval: 10
      timeout: 3
    }
  }
}
```
