# 165 - Game Config

Video game settings and preferences. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  display: {
    resolution: "2560x1440"
    fullscreen: true
    vsync: true
    fps_limit: 120
  }

  graphics: {
    quality: "high"
    shadows: "high"
    anti_aliasing: "MSAA"
    render_distance: 28
    ray_tracing: true
  }

  audio: {
    master: 57
    music: 66
    sfx: 55
    voice: 93
  }

  controls: {
    sensitivity: 3.3
    invert_y: false
    vibration: false
  }
}
```
