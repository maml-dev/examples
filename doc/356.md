# 356 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "emacs"
  shortcuts: [
    {
      key: "Ctrl+S"
      action: "save"
      when: "editor"
    }

    {
      key: "F5"
      action: "search_project"
      when: "editor"
    }

    {
      key: "Alt+F7"
      action: "toggle_comment"
      when: "editor"
    }

    {
      key: "Alt+F7"
      action: "select_all_occurrences"
      when: "terminal"
    }

    {
      key: "Ctrl+Shift+P"
      action: "format"
      when: "terminal"
    }
  ]
}
```
