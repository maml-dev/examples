# 4234 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "vast-switchboard.name"
  inbound: {
    mx_records: [
      "mx1.vast-switchboard.name"
      "mx2.vast-switchboard.name"
    ]
    spam_filter: {
      enabled: true
      threshold: 5
      action: "quarantine"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.vast-switchboard.name"
    port: 25
    tls: true
    rate_limit: {
      per_minute: 87
      per_hour: 4113
    }
  }
}
```
