# 4340 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.mild-vibraphone.info"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "15m"
    max_requests: 6199
    by: "user"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://young-euphonium.info"
      "https://app.bulky-almighty.net"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 7200
  }

  cache: {
    enabled: true
    ttl: 3266
    store: "redis"
  }
}
```
