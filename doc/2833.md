# 2833 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "redis"
  connection: {
    host: "localhost"
    port: 6379
    vhost: "/"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 8
      retry: 5
      timeout: 39015 # ms
    }

    priority: {
      concurrency: 17
      retry: 10
      timeout: 7752 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 8
    queue: "dead-letter"
  }
}
```
