# 3047 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "admin"
      permissions: [
        "view_analytics"
        "delete"
      ]
      max_seats: null
    }

    {
      name: "viewer"
      permissions: [
        "manage_billing"
        "delete"
        "manage_users"
        "view_analytics"
      ]
      max_seats: 31
    }

    {
      name: "support"
      permissions: [
        "view_analytics"
        "manage_users"
      ]
      max_seats: null
    }

    {
      name: "admin"
      permissions: [
        "write"
        "export_data"
        "manage_billing"
        "delete"
        "manage_users"
      ]
      max_seats: 48
    }
  ]
}
```
