# 1531 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "lifetime value"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 10 # seconds
    labels: {
      env: "staging"
      region: "ap-south"
      team: "home"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 92
    memory_threshold: 94
    latency_p99_ms: 1692
    error_rate: 0.01
    notification: {
      channel: "#ops-alerts"
      severity: "critical"
    }
  }
}
```
