# 1349 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "billing"
      permissions: [
        "read"
        "write"
        "view_analytics"
      ]
      max_seats: 47
    }

    {
      name: "admin"
      permissions: [
        "read"
        "view_analytics"
        "manage_billing"
        "write"
      ]
      max_seats: null
    }

    {
      name: "billing"
      permissions: [
        "export_data"
        "manage_users"
      ]
      max_seats: 29
    }

    {
      name: "billing"
      permissions: [
        "view_analytics"
        "delete"
      ]
      max_seats: null
    }
  ]
}
```
