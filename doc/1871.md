# 1871 - Dashboard Layout

Dashboard widget layout configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  title: "Stanton Group Dashboard"
  refresh_interval: 30 # seconds
  widgets: [
    {
      type: "chart"
      title: "Revenue"
      source: "/api/metrics/traffic"
      chart_type: "bar"
      period: "90d"
    }

    {
      type: "stat"
      title: "MRR"
      source: "/api/stats/users"
      format: "percent"
    }

    {
      type: "table"
      title: "Latest Signups"
      source: "/api/recent/orders"
      limit: 18
    }
  ]

  theme: "dark"
}
```
