# 1141 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "editor"
      permissions: [
        "delete"
        "manage_users"
        "view_analytics"
        "write"
        "export_data"
      ]
      max_seats: null
    }

    {
      name: "moderator"
      permissions: [
        "write"
        "view_analytics"
        "manage_users"
        "manage_billing"
        "export_data"
      ]
      max_seats: null
    }

    {
      name: "support"
      permissions: [
        "delete"
        "write"
        "export_data"
        "manage_billing"
        "manage_users"
      ]
      max_seats: 21
    }
  ]
}
```
