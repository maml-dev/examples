# 1338 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "marksman-web"
    namespace: "staging"
    labels: {
      app: "request"
      tier: "backend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 3
    container: {
      image: "thunderbolt:1.5.17"
      port: 3000
      resources: {
        cpu_limit: "500m"
        memory_limit: "256Mi"
      }
    }

    health_check: {
      path: "/ready"
      interval: 30
      timeout: 10
    }
  }
}
```
