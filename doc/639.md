# 639 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "ashamed-lounge.name"
  inbound: {
    mx_records: [
      "mx1.ashamed-lounge.name"
      "mx2.ashamed-lounge.name"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.ashamed-lounge.name"
    port: 465
    tls: true
    rate_limit: {
      per_minute: 222
      per_hour: 3797
    }
  }
}
```
