# 3123 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "redis"
  connection: {
    host: "localhost"
    port: 6379
    vhost: "/production"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 2
      retry: 3
      timeout: 6235 # ms
    }

    priority: {
      concurrency: 9
      retry: 6
      timeout: 1920 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 3
    queue: "dead-letter"
  }
}
```
