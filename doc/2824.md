# 2824 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "quinoa-api"
    namespace: "production"
    labels: {
      app: "custody"
      tier: "backend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 1
    container: {
      image: "reach:6.17.10"
      port: 8000
      resources: {
        cpu_limit: "500m"
        memory_limit: "256Mi"
      }
    }

    health_check: {
      path: "/ready"
      interval: 30
      timeout: 5
    }
  }
}
```
