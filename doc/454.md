# 454 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "vim"
  shortcuts: [
    {
      key: "Alt+F7"
      action: "find_references"
      when: "global"
    }

    {
      key: "Ctrl+S"
      action: "format"
      when: "editor"
    }

    {
      key: "Ctrl+Shift+F"
      action: "toggle_comment"
      when: "global"
    }

    {
      key: "Cmd+Shift+L"
      action: "debug_run"
      when: "global"
    }

    {
      key: "Ctrl+B"
      action: "search_project"
      when: "terminal"
    }
  ]
}
```
