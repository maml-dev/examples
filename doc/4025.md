# 4025 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "entry-api"
    namespace: "production"
    labels: {
      app: "metabolite"
      tier: "backend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 5
    container: {
      image: "commodity:5.12.9"
      port: 8000
      resources: {
        cpu_limit: "500m"
        memory_limit: "512Mi"
      }
    }

    health_check: {
      path: "/ready"
      interval: 10
      timeout: 10
    }
  }
}
```
