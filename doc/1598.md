# 1598 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@defensive-designation.com"
  subject: "Vitium sublime dolor allatus clam vesco."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Tumultus absque officia. Vilitas utique amo at aranea corroboro. Molestias valeo communis molestias curis inventore utilis veniam nemo conculco.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Cultura clibanus vaco debitis aiunt. Dolorum admoneo tonsor suggero ambitus adsuesco aufero quo. Vorago coniecto admitto desidero textilis.

Visit: {{url}}
"""
}
```
