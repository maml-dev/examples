# 2868 - Dashboard Layout

Dashboard widget layout configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  title: "McGlynn - Deckow Dashboard"
  refresh_interval: 300 # seconds
  widgets: [
    {
      type: "chart"
      title: "Signups"
      source: "/api/metrics/traffic"
      chart_type: "line"
      period: "7d"
    }

    {
      type: "stat"
      title: "Uptime"
      source: "/api/stats/users"
      format: "number"
    }

    {
      type: "table"
      title: "Recent Orders"
      source: "/api/recent/signups"
      limit: 11
    }
  ]

  theme: "auto"
}
```
