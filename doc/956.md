# 956 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "vscode"
  shortcuts: [
    {
      key: "Ctrl+/"
      action: "command_palette"
      when: "debug"
    }

    {
      key: "Ctrl+D"
      action: "toggle_comment"
      when: "debug"
    }

    {
      key: "Alt+F7"
      action: "debug_run"
      when: "debug"
    }

    {
      key: "Alt+Up"
      action: "search_project"
      when: "global"
    }

    {
      key: "Ctrl+/"
      action: "debug_run"
      when: "debug"
    }

    {
      key: "Alt+F7"
      action: "search_project"
      when: "debug"
    }

    {
      key: "Ctrl+/"
      action: "toggle_terminal"
      when: "debug"
    }
  ]
}
```
