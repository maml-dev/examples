# 534 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "platforms"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 10 # seconds
    labels: {
      env: "prod"
      region: "ap-south"
      team: "automotive"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 85
    memory_threshold: 79
    latency_p99_ms: 1397
    error_rate: 0.01
    notification: {
      channel: "#ops-alerts"
      severity: "critical"
    }
  }
}
```
