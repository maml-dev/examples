# 3143 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "sublime"
  shortcuts: [
    {
      key: "Ctrl+S"
      action: "toggle_terminal"
      when: "terminal"
    }

    {
      key: "Ctrl+B"
      action: "search_project"
      when: "debug"
    }

    {
      key: "Ctrl+Shift+P"
      action: "search_project"
      when: "terminal"
    }

    {
      key: "Cmd+K"
      action: "toggle_sidebar"
      when: "terminal"
    }

    {
      key: "Alt+Up"
      action: "search_project"
      when: "debug"
    }

    {
      key: "Ctrl+Shift+F"
      action: "format"
      when: "editor"
    }

    {
      key: "F5"
      action: "toggle_sidebar"
      when: "terminal"
    }
  ]
}
```
