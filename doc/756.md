# 756 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "moderator"
      permissions: [
        "manage_users"
        "view_analytics"
        "read"
      ]
      max_seats: 18
    }

    {
      name: "admin"
      permissions: [
        "write"
        "view_analytics"
        "delete"
        "read"
        "export_data"
      ]
      max_seats: 14
    }

    {
      name: "admin"
      permissions: [
        "delete"
        "export_data"
        "manage_users"
        "write"
        "read"
      ]
      max_seats: 16
    }

    {
      name: "viewer"
      permissions: [
        "manage_users"
        "view_analytics"
        "delete"
      ]
      max_seats: 50
    }

    {
      name: "support"
      permissions: [
        "export_data"
        "manage_users"
        "read"
      ]
      max_seats: null
    }
  ]
}
```
