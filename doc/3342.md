# 3342 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "gracious-director.org"
  records: [
    {
      type: "MX"
      name: "api"
      value: "rubbery-scarification.net"
      ttl: 300
    }

    {
      type: "CNAME"
      name: "*"
      value: "ripe-climb.biz"
      ttl: 300
    }

    {
      type: "A"
      name: "mail"
      value: "corrupt-alb.name"
      ttl: 86400
    }

    {
      type: "AAAA"
      name: "*"
      value: "v=spf1 include:_spf.prickly-hyphenation.name ~all"
      ttl: 3600
    }
  ]
}
```
