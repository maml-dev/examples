# 3540 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "following-worker"
    namespace: "production"
    labels: {
      app: "teammate"
      tier: "data"
    }
  }

  # Deployment spec
  spec: {
    replicas: 5
    container: {
      image: "schedule:3.14.14"
      port: 8080
      resources: {
        cpu_limit: "500m"
        memory_limit: "256Mi"
      }
    }

    health_check: {
      path: "/ready"
      interval: 10
      timeout: 3
    }
  }
}
```
