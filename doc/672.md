# 672 - Dashboard Layout

Dashboard widget layout configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  title: "Boehm and Sons Dashboard"
  refresh_interval: 60 # seconds
  widgets: [
    {
      type: "chart"
      title: "Signups"
      source: "/api/metrics/revenue"
      chart_type: "bar"
      period: "7d"
    }

    {
      type: "stat"
      title: "MRR"
      source: "/api/stats/users"
      format: "currency"
    }

    {
      type: "table"
      title: "Top Pages"
      source: "/api/recent/orders"
      limit: 5
    }
  ]

  theme: "dark"
}
```
