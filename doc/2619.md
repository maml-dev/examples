# 2619 - Logger Config

Structured logging configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  level: "error"
  format: "pretty"
  timestamp: true
  caller: false # include file:line

  output: "stderr"
  rotation: false
  max_size_mb: 276
  max_age_days: 90
  compress: true
}
```
