# 723 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "awesome-deduction.info"
  inbound: {
    mx_records: [
      "mx1.awesome-deduction.info"
      "mx2.awesome-deduction.info"
    ]
    spam_filter: {
      enabled: true
      threshold: 7
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.awesome-deduction.info"
    port: 587
    tls: true
    rate_limit: {
      per_minute: 270
      per_hour: 1501
    }
  }
}
```
