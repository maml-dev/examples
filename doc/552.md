# 552 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "emacs"
  shortcuts: [
    {
      key: "Ctrl+D"
      action: "find_references"
      when: "terminal"
    }

    {
      key: "F5"
      action: "toggle_comment"
      when: "debug"
    }

    {
      key: "Ctrl+Shift+F"
      action: "debug_run"
      when: "terminal"
    }
  ]
}
```
