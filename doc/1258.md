# 1258 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "sublime"
  shortcuts: [
    {
      key: "Alt+Up"
      action: "toggle_comment"
      when: "editor"
    }

    {
      key: "Ctrl+S"
      action: "format"
      when: "editor"
    }

    {
      key: "Alt+F7"
      action: "select_all_occurrences"
      when: "editor"
    }

    {
      key: "Alt+Up"
      action: "search_project"
      when: "global"
    }

    {
      key: "Ctrl+Shift+P"
      action: "toggle_comment"
      when: "global"
    }

    {
      key: "Ctrl+D"
      action: "find_references"
      when: "editor"
    }
  ]
}
```
