# 3349 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "emacs"
  shortcuts: [
    {
      key: "Ctrl+D"
      action: "save"
      when: "debug"
    }

    {
      key: "Ctrl+Shift+F"
      action: "command_palette"
      when: "terminal"
    }

    {
      key: "Alt+Up"
      action: "toggle_terminal"
      when: "debug"
    }

    {
      key: "F5"
      action: "format"
      when: "terminal"
    }

    {
      key: "F5"
      action: "debug_run"
      when: "debug"
    }

    {
      key: "Alt+Up"
      action: "toggle_comment"
      when: "global"
    }

    {
      key: "Cmd+K"
      action: "find_references"
      when: "terminal"
    }
  ]
}
```
