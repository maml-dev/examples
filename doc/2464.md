# 2464 - Dashboard Layout

Dashboard widget layout configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  title: "Ziemann and Sons Dashboard"
  refresh_interval: 300 # seconds
  widgets: [
    {
      type: "chart"
      title: "Active Users"
      source: "/api/metrics/traffic"
      chart_type: "line"
      period: "7d"
    }

    {
      type: "stat"
      title: "Error Rate"
      source: "/api/stats/users"
      format: "currency"
    }

    {
      type: "table"
      title: "Latest Signups"
      source: "/api/recent/orders"
      limit: 12
    }
  ]

  theme: "dark"
}
```
