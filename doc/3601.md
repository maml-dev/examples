# 3601 - Email Config

SMTP email service settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  smtp_host: "smtp.worst-peninsula.org"
  smtp_port: 587
  use_tls: true
  from_name: "Glover, Hermann and VonRueden"
  from_email: "noreply@worst-peninsula.org"

  # Rate limiting
  max_batch_size: 412
  retry_interval: 288
}
```
