# 1822 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "cauliflower-worker"
    namespace: "production"
    labels: {
      app: "honesty"
      tier: "data"
    }
  }

  # Deployment spec
  spec: {
    replicas: 4
    container: {
      image: "bracelet:6.8.12"
      port: 9090
      resources: {
        cpu_limit: "250m"
        memory_limit: "1Gi"
      }
    }

    health_check: {
      path: "/ready"
      interval: 15
      timeout: 3
    }
  }
}
```
