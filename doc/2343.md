# 2343 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "caring-popularity.name"
  records: [
    {
      type: "TXT"
      name: "mail"
      value: "v=spf1 include:_spf.polite-thongs.com ~all"
      ttl: 300
    }

    {
      type: "CNAME"
      name: "cdn"
      value: "v=spf1 include:_spf.hateful-eggplant.name ~all"
      ttl: 3600
    }

    {
      type: "NS"
      name: "*"
      value: "worst-dredger.org"
      ttl: 300
    }

    {
      type: "A"
      name: "*"
      value: "121.166.84.134"
      ttl: 300
    }

    {
      type: "NS"
      name: "www"
      value: "glossy-reward.net"
      ttl: 300
    }

    {
      type: "NS"
      name: "www"
      value: "silent-scrap.net"
      ttl: 86400
    }
  ]
}
```
