# 4288 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@pure-cruelty.info"
  subject: "Abundans cado adfero."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Odio quisquam suscipio terga avaritia quas aspernatur odit deleniti. Tempora crapula vetus. Volo aeger callide praesentium vir vehemens votum sperno.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Adipiscor virtus terebro viscus timor confido arbustum. Asper universe combibo abundans tergum. Concedo arca aer tam deprimo cunabula terminatio.

Visit: {{url}}
"""
}
```
