# 3291 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@cheerful-fisherman.name"
  subject: "Qui in dolorum avarus vomer."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Cruciamentum tutis usitas audeo. Ab peccatus candidus utpote volup conculco ex. Magni maxime decor ter inflammatio timidus curia nulla.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Clarus collum admoveo viridis adinventitias coerceo. Crinis crur caveo quam ater volva talio suasoria. Tondeo blanditiis solio deprimo abscido.

Visit: {{url}}
"""
}
```
