# 2135 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "sqs"
  connection: {
    host: "localhost"
    port: 6379
    vhost: "/staging"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 5
      retry: 1
      timeout: 12648 # ms
    }

    priority: {
      concurrency: 5
      retry: 3
      timeout: 8777 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 6
    queue: "dead-letter"
  }
}
```
