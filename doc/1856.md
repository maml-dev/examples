# 1856 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "billing"
      permissions: [
        "manage_users"
        "view_analytics"
      ]
      max_seats: null
    }

    {
      name: "support"
      permissions: [
        "manage_users"
        "read"
        "export_data"
        "delete"
        "manage_billing"
      ]
      max_seats: null
    }

    {
      name: "editor"
      permissions: [
        "delete"
        "manage_billing"
        "manage_users"
        "view_analytics"
        "read"
      ]
      max_seats: 17
    }
  ]
}
```
