# 025 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "relationships"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 10 # seconds
    labels: {
      env: "prod"
      region: "us-east"
      team: "movies"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 93
    memory_threshold: 95
    latency_p99_ms: 1151
    error_rate: 0.05
    notification: {
      channel: "#incidents"
      severity: "info"
    }
  }
}
```
