# 1445 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "support"
      permissions: [
        "write"
        "delete"
      ]
      max_seats: 33
    }

    {
      name: "viewer"
      permissions: [
        "write"
        "delete"
        "export_data"
      ]
      max_seats: 13
    }

    {
      name: "admin"
      permissions: [
        "export_data"
        "read"
        "write"
        "delete"
      ]
      max_seats: null
    }

    {
      name: "billing"
      permissions: [
        "read"
        "delete"
        "manage_billing"
      ]
      max_seats: 37
    }

    {
      name: "support"
      permissions: [
        "delete"
        "view_analytics"
      ]
      max_seats: null
    }
  ]
}
```
