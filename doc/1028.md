# 1028 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "sqs"
  connection: {
    host: "localhost"
    port: 5672
    vhost: "/production"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 3
      retry: 1
      timeout: 9425 # ms
    }

    priority: {
      concurrency: 7
      retry: 4
      timeout: 9414 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 9
    queue: "dead-letter"
  }
}
```
