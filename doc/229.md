# 229 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "puzzled-bump.com"
  inbound: {
    mx_records: [
      "mx1.puzzled-bump.com"
      "mx2.puzzled-bump.com"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "reject"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.puzzled-bump.com"
    port: 25
    tls: true
    rate_limit: {
      per_minute: 341
      per_hour: 2009
    }
  }
}
```
