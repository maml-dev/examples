# 4028 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.cultivated-clamp.info"
    port: 8443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "15m"
    max_requests: 6348
    by: "ip"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://common-avalanche.org"
      "https://app.acidic-resolve.biz"
    ]

      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 7200
  }

  cache: {
    enabled: false
    ttl: 3283
    store: "memory"
  }
}
```
