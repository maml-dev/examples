# 2336 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "labourer-api"
    namespace: "default"
    labels: {
      app: "comestible"
      tier: "data"
    }
  }

  # Deployment spec
  spec: {
    replicas: 2
    container: {
      image: "draw:3.20.7"
      port: 8000
      resources: {
        cpu_limit: "250m"
        memory_limit: "512Mi"
      }
    }

    health_check: {
      path: "/healthz"
      interval: 15
      timeout: 10
    }
  }
}
```
