# 2223 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "kafka"
  connection: {
    host: "mq.internal"
    port: 6379
    vhost: "/staging"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 4
      retry: 4
      timeout: 39977 # ms
    }

    priority: {
      concurrency: 6
      retry: 9
      timeout: 6090 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 10
    queue: "dead-letter"
  }
}
```
