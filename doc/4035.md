# 4035 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "optimal-hydrant.name"
  inbound: {
    mx_records: [
      "mx1.optimal-hydrant.name"
      "mx2.optimal-hydrant.name"
    ]
    spam_filter: {
      enabled: true
      threshold: 7
      action: "quarantine"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.optimal-hydrant.name"
    port: 587
    tls: true
    rate_limit: {
      per_minute: 267
      per_hour: 7097
    }
  }
}
```
