# 4111 - Linter Config

Code linter rule configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  parser: "babel"
  env: "node"
  indent: 2
  semicolons: true
  quotes: "single"

  # Formatting
  max_line_length: 80
  trailing_comma: "all"
  no_unused_vars: true # error
  no_console: false # warn
}
```
