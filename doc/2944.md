# 2944 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v3"
  routes: [
    {
      method: "POST"
      path: "/comments"
      handler: "updateUsers"
      auth: false
    }

    {
      method: "PUT"
      path: "/orders/:id"
      handler: "updatePosts"
      auth: true
    }

    {
      method: "PATCH"
      path: "/comments/:id"
      handler: "createPosts"
      auth: false
    }
  ]
}
```
