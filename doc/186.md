# 186 - Quoted Keys

Configuration using special characters in keys. This is an example of a [MAML](https://maml.dev) document.

```maml
# HTTP Headers - keys need quoting
{
  Content-Type: "application/json"
  X-Request-ID: "058ad7a0-2881-422d-bdbd-0148e628383c"
  Accept-Language: "ja-JP"
  Cache-Control: "no-cache"
  X-Powered-By: "Koa"

  # Keys with spaces get quoted
  "headers with spaces": {
    "custom header": "depono"
    "another one": "sequi"
  }
}
```
