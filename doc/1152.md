# 1152 - Cron Jobs

Scheduled task definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  timezone: "America/New_York"
  jobs: [
    {
      name: "backup-db"
      schedule: "0 2 * * 0"
      command: "pg_dump $DATABASE_URL"
      enabled: false
    }

    {
      name: "send-reports"
      schedule: "0 0 * * *"
      command: "node scripts/cleanup.js"
      enabled: true
    }

    {
      name: "cleanup-temp"
      schedule: "0 * * * *"
      command: "python manage.py report"
      enabled: false
    }

    {
      name: "rotate-logs"
      schedule: "*/15 * * * *"
      command: "python manage.py report"
      enabled: true
    }

    {
      name: "rotate-logs"
      schedule: "30 6 * * 1-5"
      command: "./bin/sync --all"
      enabled: true
    }

    {
      name: "rotate-logs"
      schedule: "*/15 * * * *"
      command: "node scripts/cleanup.js"
      enabled: false
    }
  ]
}
```
