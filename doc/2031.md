# 2031 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "lace-web"
    namespace: "staging"
    labels: {
      app: "recovery"
      tier: "backend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 2
    container: {
      image: "pillow:0.15.16"
      port: 9090
      resources: {
        cpu_limit: "1000m"
        memory_limit: "1Gi"
      }
    }

    health_check: {
      path: "/healthz"
      interval: 30
      timeout: 10
    }
  }
}
```
