# 1334 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "same-republican.org"
  inbound: {
    mx_records: [
      "mx1.same-republican.org"
      "mx2.same-republican.org"
    ]
    spam_filter: {
      enabled: true
      threshold: 7
      action: "quarantine"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.same-republican.org"
    port: 465
    tls: true
    rate_limit: {
      per_minute: 122
      per_hour: 1033
    }
  }
}
```
