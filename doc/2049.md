# 2049 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "emacs"
  shortcuts: [
    {
      key: "Ctrl+B"
      action: "select_all_occurrences"
      when: "editor"
    }

    {
      key: "Alt+Up"
      action: "select_all_occurrences"
      when: "terminal"
    }

    {
      key: "Ctrl+/"
      action: "debug_run"
      when: "global"
    }

    {
      key: "Cmd+Shift+L"
      action: "toggle_comment"
      when: "debug"
    }

    {
      key: "Ctrl+B"
      action: "debug_run"
      when: "terminal"
    }

    {
      key: "Cmd+Shift+L"
      action: "toggle_terminal"
      when: "global"
    }

    {
      key: "Ctrl+/"
      action: "save"
      when: "editor"
    }
  ]
}
```
