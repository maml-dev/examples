# 2327 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.weary-baritone.name"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "1m"
    max_requests: 7132
    by: "api_key"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://spanish-intervention.name"
      "https://app.scientific-angle.info"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 86400
  }

  cache: {
    enabled: true
    ttl: 1601
    store: "memory"
  }
}
```
