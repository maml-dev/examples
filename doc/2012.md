# 2012 - Proxy Config

Reverse proxy settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  listen: 8080
  server_name: "awful-ferret.biz"
  upstream: "http://localhost:6901"

  # TLS settings
  ssl: true
  http2: true

  gzip: true
  gzip_min_length: 1024
  client_max_body: "8mb"
}
```
