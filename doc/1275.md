# 1275 - Dashboard Layout

Dashboard widget layout configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  title: "Bradtke, Gutkowski and Beer Dashboard"
  refresh_interval: 60 # seconds
  widgets: [
    {
      type: "chart"
      title: "Traffic"
      source: "/api/metrics/traffic"
      chart_type: "area"
      period: "7d"
    }

    {
      type: "stat"
      title: "MRR"
      source: "/api/stats/users"
      format: "number"
    }

    {
      type: "table"
      title: "Recent Orders"
      source: "/api/recent/orders"
      limit: 8
    }
  ]

  theme: "dark"
}
```
