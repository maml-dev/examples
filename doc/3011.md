# 3011 - DNS Config

DNS zone configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "soggy-valentine.net"
  ttl: 600 # seconds
  nameservers: [
    "ns1.ignorant-cleaner.com"
    "ns2.gracious-singing.com"
  ]

  soa_email: "admin@cavernous-cuckoo.net"
  refresh: 7200
  retry: 600
  expire: 604800
}
```
