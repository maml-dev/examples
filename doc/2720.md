# 2720 - DNS Config

DNS zone configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "corny-reasoning.biz"
  ttl: 86400 # seconds
  nameservers: [
    "ns1.velvety-collectivization.name"
    "ns2.unwelcome-tenement.com"
  ]

  soa_email: "admin@happy-ruin.name"
  refresh: 3600
  retry: 1800
  expire: 604800
}
```
