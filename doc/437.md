# 437 - Theme Config

Design system theme configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  name: "utter Theme"
  # Color palette
  colors: {
    primary: "#6aae17"
    secondary: "#c51ef0"
    accent: "#52dffd"
    background: "#ffffff"
    text: "#333333"

    error: "#dc3545"
    warning: "#ffc107"
    success: "#28a745"
  }

  # Typography
  typography: {
    font_family: "system-ui, sans-serif"
    base_size: 16
    line_height: 1.6
    headings: {
      font_family: "Inter, sans-serif"
      weight: 800
    }
  }

  spacing: {
    unit: 8
    scale: [
      0
      4
      8
      16
      24
      32
      48
    ]
  }
  border_radius: 6
}
```
