# 232 - Theme Config

Design system theme configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  name: "amazing Theme"
  # Color palette
  colors: {
    primary: "#ddf9a3"
    secondary: "#0d866a"
    accent: "#942bd1"
    background: "#f8f9fa"
    text: "#333333"

    error: "#dc3545"
    warning: "#ffc107"
    success: "#28a745"
  }

  # Typography
  typography: {
    font_family: "\"JetBrains Mono\", monospace"
    base_size: 18
    line_height: 1.5
    headings: {
      font_family: "\"Playfair Display\", serif"
      weight: 700
    }
  }

  spacing: {
    unit: 8
    scale: [
      0
      4
      8
      16
      24
      32
      48
    ]
  }
  border_radius: 8
}
```
