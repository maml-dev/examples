# 2457 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v1"
  routes: [
    {
      method: "POST"
      path: "/posts/:id"
      handler: "createProducts"
      auth: true
    }

    {
      method: "PATCH"
      path: "/orders/:id"
      handler: "updatePosts"
      auth: false
    }

    {
      method: "PUT"
      path: "/users/:id"
      handler: "updateUsers"
      auth: false
    }
  ]
}
```
