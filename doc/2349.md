# 2349 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "moderator"
      permissions: [
        "delete"
        "read"
        "write"
      ]
      max_seats: null
    }

    {
      name: "billing"
      permissions: [
        "view_analytics"
        "read"
        "delete"
        "export_data"
        "write"
      ]
      max_seats: 20
    }

    {
      name: "editor"
      permissions: [
        "manage_billing"
        "write"
        "export_data"
        "manage_users"
      ]
      max_seats: 21
    }
  ]
}
```
