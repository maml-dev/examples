# 1147 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "editor"
      permissions: [
        "write"
        "view_analytics"
        "manage_users"
        "delete"
      ]
      max_seats: 30
    }

    {
      name: "billing"
      permissions: [
        "manage_users"
        "write"
      ]
      max_seats: null
    }

    {
      name: "editor"
      permissions: [
        "manage_users"
        "manage_billing"
      ]
      max_seats: 48
    }

    {
      name: "billing"
      permissions: [
        "manage_users"
        "manage_billing"
        "view_analytics"
        "read"
      ]
      max_seats: 20
    }
  ]
}
```
