# 788 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@eminent-saloon.info"
  subject: "Adhuc vestigium repellendus adhaero uxor."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Vilicus angulus trado coepi canonicus aliqua. Adhuc aut admoneo dedico summopere. Laboriosam assentator debitis facilis toties trucido cibus crustulum volva.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Rerum temporibus demergo coepi crapula abutor addo tutamen. Deprecator curtus articulus taceo excepturi tracto doloribus vicissitudo supellex qui. Bos tempus patruus viduo sunt.

Visit: {{url}}
"""
}
```
