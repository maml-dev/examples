# 2630 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "gifted-unit.info"
  inbound: {
    mx_records: [
      "mx1.gifted-unit.info"
      "mx2.gifted-unit.info"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "reject"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.gifted-unit.info"
    port: 465
    tls: true
    rate_limit: {
      per_minute: 460
      per_hour: 2769
    }
  }
}
```
