# 1880 - Dashboard Layout

Dashboard widget layout configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  title: "Gleichner, Moore-Heller and Jacobi Dashboard"
  refresh_interval: 300 # seconds
  widgets: [
    {
      type: "chart"
      title: "Revenue"
      source: "/api/metrics/revenue"
      chart_type: "area"
      period: "7d"
    }

    {
      type: "stat"
      title: "Uptime"
      source: "/api/stats/users"
      format: "percent"
    }

    {
      type: "table"
      title: "Top Pages"
      source: "/api/recent/orders"
      limit: 17
    }
  ]

  theme: "dark"
}
```
