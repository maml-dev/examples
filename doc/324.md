# 324 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "agitated-step.biz"
  inbound: {
    mx_records: [
      "mx1.agitated-step.biz"
      "mx2.agitated-step.biz"
    ]
    spam_filter: {
      enabled: true
      threshold: 5
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.agitated-step.biz"
    port: 25
    tls: true
    rate_limit: {
      per_minute: 102
      per_hour: 5813
    }
  }
}
```
