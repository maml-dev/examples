# 1044 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "vim"
  shortcuts: [
    {
      key: "Cmd+K"
      action: "format"
      when: "editor"
    }

    {
      key: "Ctrl+Shift+P"
      action: "toggle_terminal"
      when: "terminal"
    }

    {
      key: "Cmd+Shift+L"
      action: "command_palette"
      when: "global"
    }
  ]
}
```
