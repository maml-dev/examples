# 3616 - Proxy Config

Reverse proxy settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  listen: 80
  server_name: "speedy-cemetery.name"
  upstream: "http://127.0.0.1:7265"

  # TLS settings
  ssl: true
  http2: true

  gzip: true
  gzip_min_length: 1024
  client_max_body: "25mb"
}
```
