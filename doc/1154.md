# 1154 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "viewer"
      permissions: [
        "read"
        "write"
        "view_analytics"
      ]
      max_seats: null
    }

    {
      name: "viewer"
      permissions: [
        "write"
        "manage_billing"
      ]
      max_seats: null
    }

    {
      name: "billing"
      permissions: [
        "delete"
        "read"
        "write"
        "manage_billing"
      ]
      max_seats: 27
    }
  ]
}
```
