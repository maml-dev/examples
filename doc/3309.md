# 3309 - Logger Config

Structured logging configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  level: "info"
  format: "pretty"
  timestamp: true
  caller: false # include file:line

  output: "stderr"
  rotation: true
  max_size_mb: 60
  max_age_days: 89
  compress: true
}
```
