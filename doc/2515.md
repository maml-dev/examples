# 2515 - Logger Config

Structured logging configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  level: "trace"
  format: "logfmt"
  timestamp: true
  caller: true # include file:line

  output: "stderr"
  rotation: false
  max_size_mb: 161
  max_age_days: 76
  compress: true
}
```
