# 3339 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "eminent-tabletop.org"
  inbound: {
    mx_records: [
      "mx1.eminent-tabletop.org"
      "mx2.eminent-tabletop.org"
    ]
    spam_filter: {
      enabled: true
      threshold: 7
      action: "reject"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.eminent-tabletop.org"
    port: 25
    tls: true
    rate_limit: {
      per_minute: 310
      per_hour: 4617
    }
  }
}
```
