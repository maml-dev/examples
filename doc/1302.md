# 1302 - Linter Config

Code linter rule configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  parser: "babel"
  env: "browser"
  indent: 2
  semicolons: true
  quotes: "single"

  # Formatting
  max_line_length: 100
  trailing_comma: "none"
  no_unused_vars: true # error
  no_console: true # warn
}
```
