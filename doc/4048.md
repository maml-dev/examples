# 4048 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v1"
  routes: [
    {
      method: "PUT"
      path: "/orders"
      handler: "deleteUsers"
      auth: false
    }

    {
      method: "DELETE"
      path: "/comments"
      handler: "listOrders"
      auth: true
    }

    {
      method: "GET"
      path: "/posts/:id"
      handler: "createPosts"
      auth: false
    }

    {
      method: "PUT"
      path: "/comments/:id"
      handler: "updateOrders"
      auth: false
    }

    {
      method: "PATCH"
      path: "/comments/:id"
      handler: "getPosts"
      auth: true
    }
  ]
}
```
