# 2840 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "mortise-worker"
    namespace: "default"
    labels: {
      app: "statue"
      tier: "frontend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 3
    container: {
      image: "polarisation:3.6.9"
      port: 8080
      resources: {
        cpu_limit: "250m"
        memory_limit: "256Mi"
      }
    }

    health_check: {
      path: "/healthz"
      interval: 15
      timeout: 10
    }
  }
}
```
