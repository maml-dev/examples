# 1748 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "vim"
  shortcuts: [
    {
      key: "Ctrl+B"
      action: "format"
      when: "global"
    }

    {
      key: "Alt+Up"
      action: "toggle_terminal"
      when: "editor"
    }

    {
      key: "Ctrl+Shift+P"
      action: "search_project"
      when: "terminal"
    }
  ]
}
```
