# 154 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "shimmering-dredger.biz"
  records: [
    {
      type: "TXT"
      name: "www"
      value: "v=spf1 include:_spf.cluttered-deck.net ~all"
      ttl: 86400
    }

    {
      type: "NS"
      name: "staging"
      value: "dark-sprinkles.name"
      ttl: 300
    }

    {
      type: "A"
      name: "@"
      value: "witty-retention.biz"
      ttl: 86400
    }

    {
      type: "MX"
      name: "www"
      value: "shameful-kick.org"
      ttl: 300
    }
  ]
}
```
