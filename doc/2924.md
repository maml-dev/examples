# 2924 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "crooked-handful.name"
  inbound: {
    mx_records: [
      "mx1.crooked-handful.name"
      "mx2.crooked-handful.name"
    ]
    spam_filter: {
      enabled: true
      threshold: 5
      action: "reject"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.crooked-handful.name"
    port: 25
    tls: true
    rate_limit: {
      per_minute: 492
      per_hour: 5894
    }
  }
}
```
