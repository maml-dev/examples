# 3224 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "sqs"
  connection: {
    host: "localhost"
    port: 6379
    vhost: "/production"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 2
      retry: 3
      timeout: 27653 # ms
    }

    priority: {
      concurrency: 9
      retry: 7
      timeout: 7749 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 10
    queue: "dead-letter"
  }
}
```
