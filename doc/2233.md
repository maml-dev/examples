# 2233 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "frank-flu.name"
  inbound: {
    mx_records: [
      "mx1.frank-flu.name"
      "mx2.frank-flu.name"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "reject"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.frank-flu.name"
    port: 25
    tls: true
    rate_limit: {
      per_minute: 267
      per_hour: 6906
    }
  }
}
```
