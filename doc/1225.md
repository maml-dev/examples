# 1225 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "models"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 15 # seconds
    labels: {
      env: "prod"
      region: "us-east"
      team: "clothing"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 90
    memory_threshold: 76
    latency_p99_ms: 1193
    error_rate: 0.05
    notification: {
      channel: "#incidents"
      severity: "warning"
    }
  }
}
```
