# 631 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "sqs"
  connection: {
    host: "mq.internal"
    port: 9092
    vhost: "/production"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 2
      retry: 1
      timeout: 29545 # ms
    }

    priority: {
      concurrency: 12
      retry: 8
      timeout: 4752 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 3
    queue: "dead-letter"
  }
}
```
