# 2248 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "vim"
  shortcuts: [
    {
      key: "Cmd+Shift+L"
      action: "debug_run"
      when: "debug"
    }

    {
      key: "Ctrl+Shift+P"
      action: "toggle_terminal"
      when: "editor"
    }

    {
      key: "Alt+F7"
      action: "command_palette"
      when: "debug"
    }

    {
      key: "Alt+F7"
      action: "save"
      when: "debug"
    }

    {
      key: "Alt+Up"
      action: "debug_run"
      when: "debug"
    }
  ]
}
```
