# 543 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "firm-commercial.net"
  records: [
    {
      type: "CNAME"
      name: "www"
      value: "252.96.179.251"
      ttl: 86400
    }

    {
      type: "A"
      name: "@"
      value: "108.85.25.31"
      ttl: 300
    }

    {
      type: "MX"
      name: "@"
      value: "grim-yogurt.org"
      ttl: 86400
    }

    {
      type: "MX"
      name: "*"
      value: "v=spf1 include:_spf.irresponsible-hepatitis.com ~all"
      ttl: 86400
    }

    {
      type: "AAAA"
      name: "@"
      value: "254.251.29.179"
      ttl: 300
    }

    {
      type: "CNAME"
      name: "staging"
      value: "mixed-distinction.org"
      ttl: 3600
    }

    {
      type: "CNAME"
      name: "cdn"
      value: "bogus-providence.com"
      ttl: 86400
    }

    {
      type: "NS"
      name: "@"
      value: "97.11.54.230"
      ttl: 300
    }
  ]
}
```
