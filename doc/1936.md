# 1936 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "summary-worker"
    namespace: "staging"
    labels: {
      app: "vibraphone"
      tier: "frontend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 4
    container: {
      image: "maintainer:3.1.6"
      port: 8000
      resources: {
        cpu_limit: "1000m"
        memory_limit: "256Mi"
      }
    }

    health_check: {
      path: "/ready"
      interval: 10
      timeout: 5
    }
  }
}
```
