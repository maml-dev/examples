# 1639 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "kafka"
  connection: {
    host: "localhost"
    port: 5672
    vhost: "/production"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 5
      retry: 2
      timeout: 36234 # ms
    }

    priority: {
      concurrency: 18
      retry: 3
      timeout: 6853 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 6
    queue: "dead-letter"
  }
}
```
