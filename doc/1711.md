# 1711 - Proxy Config

Reverse proxy settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  listen: 8080
  server_name: "rotating-dividend.name"
  upstream: "http://127.0.0.1:5960"

  # TLS settings
  ssl: false
  http2: true

  gzip: true
  gzip_min_length: 1024
  client_max_body: "32mb"
}
```
