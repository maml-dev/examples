# 746 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "admin"
      permissions: [
        "manage_users"
        "export_data"
        "read"
      ]
      max_seats: 30
    }

    {
      name: "billing"
      permissions: [
        "manage_billing"
        "write"
        "export_data"
      ]
      max_seats: null
    }

    {
      name: "admin"
      permissions: [
        "export_data"
        "write"
        "delete"
        "manage_billing"
      ]
      max_seats: 25
    }
  ]
}
```
