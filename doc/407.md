# 407 - Logger Config

Structured logging configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  level: "warn"
  format: "logfmt"
  timestamp: true
  caller: false # include file:line

  output: "stdout"
  rotation: false
  max_size_mb: 159
  max_age_days: 26
  compress: true
}
```
