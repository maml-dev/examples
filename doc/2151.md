# 2151 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "vim"
  shortcuts: [
    {
      key: "Ctrl+D"
      action: "toggle_terminal"
      when: "terminal"
    }

    {
      key: "Alt+F7"
      action: "toggle_terminal"
      when: "global"
    }

    {
      key: "Ctrl+B"
      action: "format"
      when: "terminal"
    }

    {
      key: "Ctrl+/"
      action: "debug_run"
      when: "debug"
    }
  ]
}
```
