# 823 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "smart contracts"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 10 # seconds
    labels: {
      env: "prod"
      region: "us-east"
      team: "baby"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 95
    memory_threshold: 79
    latency_p99_ms: 242
    error_rate: 0.01
    notification: {
      channel: "#oncall"
      severity: "critical"
    }
  }
}
```
