# 3532 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "exhausted-marathon.biz"
  inbound: {
    mx_records: [
      "mx1.exhausted-marathon.biz"
      "mx2.exhausted-marathon.biz"
    ]
    spam_filter: {
      enabled: true
      threshold: 7
      action: "reject"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.exhausted-marathon.biz"
    port: 465
    tls: true
    rate_limit: {
      per_minute: 333
      per_hour: 5564
    }
  }
}
```
