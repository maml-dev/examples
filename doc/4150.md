# 4150 - Environment Vars

Environment variable definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  app: "pile"
  variables: [
    {
      name: "SECRET_KEY"
      required: false
      secret: true
      default: "vestigium"
    }

    {
      name: "JWT_SECRET"
      required: true
      secret: true
      default: null
    }

    {
      name: "DATABASE_URL"
      required: false
      secret: false
      default: "vix"
    }
  ]
}
```
