# 1819 - Proxy Config

Reverse proxy settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  listen: 443
  server_name: "usable-tune-up.net"
  upstream: "http://127.0.0.1:3851"

  # TLS settings
  ssl: true
  http2: true

  gzip: true
  gzip_min_length: 1024
  client_max_body: "10mb"
}
```
