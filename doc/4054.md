# 4054 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "support"
      permissions: [
        "delete"
        "read"
      ]
      max_seats: null
    }

    {
      name: "billing"
      permissions: [
        "manage_billing"
        "view_analytics"
        "read"
        "write"
      ]
      max_seats: null
    }

    {
      name: "admin"
      permissions: [
        "manage_billing"
        "delete"
        "view_analytics"
      ]
      max_seats: null
    }
  ]
}
```
