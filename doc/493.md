# 493 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@rotating-space.name"
  subject: "Validus beatae patrocinor sui."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Cogito bellum tantillus. Demens vix caritas adduco coniuratio curis. Deinde vetus curriculum.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Depopulo arx aestus custodia. Quis vehemens tempore expedita textilis cicuta acquiro coadunatio. Taedium teneo sui statua sequi laboriosam advoco.

Visit: {{url}}
"""
}
```
