# 3939 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "cow-worker"
    namespace: "default"
    labels: {
      app: "affiliate"
      tier: "backend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 1
    container: {
      image: "lobster:7.14.15"
      port: 3000
      resources: {
        cpu_limit: "250m"
        memory_limit: "1Gi"
      }
    }

    health_check: {
      path: "/health"
      interval: 15
      timeout: 5
    }
  }
}
```
