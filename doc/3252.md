# 3252 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "negative-descendant.com"
  records: [
    {
      type: "AAAA"
      name: "*"
      value: "our-hyena.org"
      ttl: 3600
    }

    {
      type: "NS"
      name: "@"
      value: "zany-ethyl.org"
      ttl: 3600
    }

    {
      type: "AAAA"
      name: "mail"
      value: "100.99.115.157"
      ttl: 300
    }

    {
      type: "A"
      name: "@"
      value: "sleepy-mountain.name"
      ttl: 86400
    }

    {
      type: "NS"
      name: "staging"
      value: "119.237.36.100"
      ttl: 86400
    }
  ]
}
```
