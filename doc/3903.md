# 3903 - Logger Config

Structured logging configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  level: "warn"
  format: "logfmt"
  timestamp: true
  caller: true # include file:line

  output: "/var/log/app.log"
  rotation: true
  max_size_mb: 389
  max_age_days: 9
  compress: true
}
```
