# 3234 - Backup Strategy

Backup schedule and retention policy. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  source: {
    type: "filesystem"
    host: "localhost"
    databases: [
      "handover"
      "finding"
    ]
  }

  # Backup schedule
  schedule: {
    full: "0 2 * * 0" # weekly Sunday 2am
    incremental: "0 2 * * 1-6" # daily except Sunday
    snapshot: "0 */4 * * *" # every 4 hours
  }

  retention: {
    daily: 11
    weekly: 7
    monthly: 7
  }

  destination: {
    type: "local"
    path: "/mnt/backups/tennis"
    encryption: true
    compression: "lz4"
  }
}
```
