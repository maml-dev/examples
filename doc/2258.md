# 2258 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "emacs"
  shortcuts: [
    {
      key: "Ctrl+`"
      action: "format"
      when: "editor"
    }

    {
      key: "Ctrl+S"
      action: "select_all_occurrences"
      when: "terminal"
    }

    {
      key: "Ctrl+/"
      action: "save"
      when: "global"
    }
  ]
}
```
