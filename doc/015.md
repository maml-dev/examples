# 015 - Locale Config

Regional and locale settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  language: "de"
  country: "MK"
  timezone: "America/Argentina/La_Rioja"
  currency: "SYP"

  date_format: "YYYY-MM-DD"
  time_format: "24h"
  first_day_of_week: "monday"
  decimal_separator: ","
}
```
