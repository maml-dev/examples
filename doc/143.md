# 143 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "muddy-husband.org"
  records: [
    {
      type: "CNAME"
      name: "@"
      value: "royal-traditionalism.com"
      ttl: 86400
    }

    {
      type: "TXT"
      name: "cdn"
      value: "160.129.124.95"
      ttl: 3600
    }

    {
      type: "AAAA"
      name: "www"
      value: "202.143.240.187"
      ttl: 300
    }

    {
      type: "TXT"
      name: "www"
      value: "176.195.167.217"
      ttl: 3600
    }

    {
      type: "TXT"
      name: "@"
      value: "145.247.189.75"
      ttl: 3600
    }

    {
      type: "TXT"
      name: "*"
      value: "ruddy-interviewer.net"
      ttl: 86400
    }

    {
      type: "MX"
      name: "@"
      value: "v=spf1 include:_spf.greedy-awareness.name ~all"
      ttl: 3600
    }

    {
      type: "NS"
      name: "www"
      value: "left-volleyball.com"
      ttl: 3600
    }
  ]
}
```
