# 160 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v3"
  routes: [
    {
      method: "GET"
      path: "/posts/:id"
      handler: "createUsers"
      auth: true
    }

    {
      method: "POST"
      path: "/users"
      handler: "listProducts"
      auth: true
    }

    {
      method: "POST"
      path: "/users"
      handler: "listProducts"
      auth: false
    }

    {
      method: "PUT"
      path: "/products/:id"
      handler: "createUsers"
      auth: true
    }

    {
      method: "POST"
      path: "/posts"
      handler: "getProducts"
      auth: true
    }

    {
      method: "GET"
      path: "/posts/:id"
      handler: "getProducts"
      auth: true
    }

    {
      method: "POST"
      path: "/posts/:id"
      handler: "deletePosts"
      auth: true
    }
  ]
}
```
