# 230 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.front-e-mail.info"
    port: 8443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "1m"
    max_requests: 5519
    by: "user"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://distinct-signature.net"
      "https://app.lazy-behest.org"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 86400
  }

  cache: {
    enabled: false
    ttl: 2194
    store: "memory"
  }
}
```
