# 358 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "support"
      permissions: [
        "manage_users"
        "read"
        "export_data"
      ]
      max_seats: null
    }

    {
      name: "admin"
      permissions: [
        "read"
        "export_data"
      ]
      max_seats: null
    }

    {
      name: "admin"
      permissions: [
        "export_data"
        "view_analytics"
        "manage_users"
        "manage_billing"
        "write"
      ]
      max_seats: null
    }
  ]
}
```
