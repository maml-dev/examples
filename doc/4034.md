# 4034 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "unlucky-deed.org"
  inbound: {
    mx_records: [
      "mx1.unlucky-deed.org"
      "mx2.unlucky-deed.org"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "reject"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.unlucky-deed.org"
    port: 25
    tls: true
    rate_limit: {
      per_minute: 339
      per_hour: 1429
    }
  }
}
```
