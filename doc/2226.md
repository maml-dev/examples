# 2226 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "redis"
  connection: {
    host: "mq.internal"
    port: 6379
    vhost: "/"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 4
      retry: 2
      timeout: 57667 # ms
    }

    priority: {
      concurrency: 17
      retry: 3
      timeout: 6884 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 10
    queue: "dead-letter"
  }
}
```
