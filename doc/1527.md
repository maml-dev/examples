# 1527 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "models"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 15 # seconds
    labels: {
      env: "staging"
      region: "us-east"
      team: "movies"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 91
    memory_threshold: 88
    latency_p99_ms: 121
    error_rate: 0.02
    notification: {
      channel: "#ops-alerts"
      severity: "info"
    }
  }
}
```
