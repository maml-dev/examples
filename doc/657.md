# 657 - Cron Jobs

Scheduled task definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  timezone: "UTC"
  jobs: [
    {
      name: "sync-data"
      schedule: "*/15 * * * *"
      command: "node scripts/cleanup.js"
      enabled: true
    }

    {
      name: "cleanup-temp"
      schedule: "0 * * * *"
      command: "python manage.py report"
      enabled: false
    }

    {
      name: "refresh-cache"
      schedule: "0 * * * *"
      command: "python manage.py report"
      enabled: true
    }

    {
      name: "cleanup-temp"
      schedule: "0 * * * *"
      command: "./bin/sync --all"
      enabled: false
    }

    {
      name: "refresh-cache"
      schedule: "0 0 * * *"
      command: "./bin/sync --all"
      enabled: true
    }

    {
      name: "rotate-logs"
      schedule: "0 * * * *"
      command: "pg_dump $DATABASE_URL"
      enabled: true
    }
  ]
}
```
