# 3223 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.new-object.biz"
    port: 8443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "1m"
    max_requests: 3382
    by: "api_key"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://self-assured-asset.info"
      "https://app.formal-intellect.com"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 7200
  }

  cache: {
    enabled: true
    ttl: 2581
    store: "redis"
  }
}
```
