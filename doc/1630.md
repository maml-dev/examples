# 1630 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "substantial-slide.biz"
  inbound: {
    mx_records: [
      "mx1.substantial-slide.biz"
      "mx2.substantial-slide.biz"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.substantial-slide.biz"
    port: 465
    tls: true
    rate_limit: {
      per_minute: 268
      per_hour: 6260
    }
  }
}
```
