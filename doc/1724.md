# 1724 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "communities"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 30 # seconds
    labels: {
      env: "staging"
      region: "us-east"
      team: "shoes"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 83
    memory_threshold: 81
    latency_p99_ms: 864
    error_rate: 0.01
    notification: {
      channel: "#oncall"
      severity: "info"
    }
  }
}
```
