# 3044 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "support"
      permissions: [
        "delete"
        "read"
        "manage_billing"
        "write"
      ]
      max_seats: null
    }

    {
      name: "admin"
      permissions: [
        "manage_billing"
        "view_analytics"
      ]
      max_seats: null
    }

    {
      name: "moderator"
      permissions: [
        "export_data"
        "write"
      ]
      max_seats: null
    }

    {
      name: "viewer"
      permissions: [
        "delete"
        "write"
        "export_data"
        "read"
        "view_analytics"
      ]
      max_seats: null
    }
  ]
}
```
