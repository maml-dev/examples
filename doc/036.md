# 036 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "other-seal.biz"
  inbound: {
    mx_records: [
      "mx1.other-seal.biz"
      "mx2.other-seal.biz"
    ]
    spam_filter: {
      enabled: true
      threshold: 7
      action: "reject"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.other-seal.biz"
    port: 465
    tls: true
    rate_limit: {
      per_minute: 200
      per_hour: 9069
    }
  }
}
```
