# 3798 - Quoted Keys

Configuration using special characters in keys. This is an example of a [MAML](https://maml.dev) document.

```maml
# HTTP Headers
{
  Content-Type: "application/xml"
  X-Request-ID: "a3cf9ba7-c7d0-4bda-a037-be1061b9e844"
  Accept-Language: "fr-FR"
  Cache-Control: "no-cache"
  X-Powered-By: "Fastify"

  # Keys with spaces get quoted
  "headers with spaces": {
    "custom header": "saepe"
    "another one": "defungo"
  }
}
```
