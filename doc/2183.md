# 2183 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@sticky-window.net"
  subject: "Comes aestus auctor."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Virga cohibeo canis qui placeat umquam. Voluptate vapulus defleo ascit cometes socius suscipit viriliter vorago. Termes brevis aliquid baiulus tamisium admoneo.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Venustas cicuta degenero verus argentum benevolentia tepidus. Culpo delectatio cursus. Tubineus speciosus carbo.

Visit: {{url}}
"""
}
```
