# 431 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "sqs"
  connection: {
    host: "localhost"
    port: 5672
    vhost: "/"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 8
      retry: 3
      timeout: 34312 # ms
    }

    priority: {
      concurrency: 18
      retry: 3
      timeout: 4491 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 6
    queue: "dead-letter"
  }
}
```
