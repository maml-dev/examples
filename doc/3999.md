# 3999 - Quoted Keys

Configuration using special characters in keys. This is an example of a [MAML](https://maml.dev) document.

```maml
# HTTP Headers
{
  Content-Type: "text/html"
  X-Request-ID: "fb93b469-b5ef-4ce3-adc8-6d5d460fc186"
  Accept-Language: "en-US"
  Cache-Control: "public, max-age=86400"
  X-Powered-By: "Koa"

  # Keys with spaces get quoted
  "headers with spaces": {
    "custom header": "deserunt"
    "another one": "vetus"
  }
}
```
