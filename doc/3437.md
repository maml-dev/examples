# 3437 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "understated-tentacle.com"
  inbound: {
    mx_records: [
      "mx1.understated-tentacle.com"
      "mx2.understated-tentacle.com"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.understated-tentacle.com"
    port: 465
    tls: true
    rate_limit: {
      per_minute: 207
      per_hour: 4516
    }
  }
}
```
