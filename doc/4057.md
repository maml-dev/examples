# 4057 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "vim"
  shortcuts: [
    {
      key: "Alt+F7"
      action: "toggle_terminal"
      when: "global"
    }

    {
      key: "Ctrl+S"
      action: "save"
      when: "editor"
    }

    {
      key: "Alt+F7"
      action: "toggle_comment"
      when: "terminal"
    }

    {
      key: "Ctrl+Shift+F"
      action: "debug_run"
      when: "debug"
    }

    {
      key: "F5"
      action: "save"
      when: "editor"
    }

    {
      key: "Ctrl+/"
      action: "search_project"
      when: "global"
    }
  ]
}
```
