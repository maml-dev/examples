# 1517 - Proxy Config

Reverse proxy settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  listen: 8443
  server_name: "shrill-ferret.biz"
  upstream: "http://127.0.0.1:7510"

  # TLS settings
  ssl: false
  http2: true

  gzip: true
  gzip_min_length: 256
  client_max_body: "5mb"
}
```
