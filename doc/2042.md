# 2042 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "jam-packed-steak.biz"
  records: [
    {
      type: "CNAME"
      name: "cdn"
      value: "130.0.154.101"
      ttl: 3600
    }

    {
      type: "A"
      name: "www"
      value: "v=spf1 include:_spf.realistic-dress.name ~all"
      ttl: 300
    }

    {
      type: "TXT"
      name: "www"
      value: "electric-barracks.info"
      ttl: 300
    }

    {
      type: "CNAME"
      name: "*"
      value: "creative-cd.org"
      ttl: 86400
    }
  ]
}
```
