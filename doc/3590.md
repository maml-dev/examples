# 3590 - Quoted Keys

Configuration using special characters in keys. This is an example of a [MAML](https://maml.dev) document.

```maml
# HTTP Headers
{
  Content-Type: "text/html"
  X-Request-ID: "21d01ccd-6c2d-4d78-801a-bfb33bf5941c"
  Accept-Language: "fr-FR"
  Cache-Control: "no-cache"
  X-Powered-By: "Hono"

  # Keys with spaces get quoted
  "headers with spaces": {
    "custom header": "impedit"
    "another one": "tendo"
  }
}
```
