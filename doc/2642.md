# 2642 - Cron Jobs

Scheduled task definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  timezone: "UTC"
  jobs: [
    {
      name: "sync-data"
      schedule: "0 0 * * *"
      command: "pg_dump $DATABASE_URL"
      enabled: true
    }

    {
      name: "send-reports"
      schedule: "*/15 * * * *"
      command: "pg_dump $DATABASE_URL"
      enabled: true
    }

    {
      name: "send-reports"
      schedule: "0 2 * * 0"
      command: "python manage.py report"
      enabled: true
    }

    {
      name: "send-reports"
      schedule: "0 * * * *"
      command: "./bin/sync --all"
      enabled: false
    }

    {
      name: "rotate-logs"
      schedule: "30 6 * * 1-5"
      command: "python manage.py report"
      enabled: false
    }
  ]
}
```
