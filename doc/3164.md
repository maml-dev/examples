# 3164 - Dashboard Layout

Dashboard widget layout configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  title: "Kuhlman - Bauch Dashboard"
  refresh_interval: 30 # seconds
  widgets: [
    {
      type: "chart"
      title: "Active Users"
      source: "/api/metrics/revenue"
      chart_type: "bar"
      period: "30d"
    }

    {
      type: "stat"
      title: "Total Users"
      source: "/api/stats/revenue"
      format: "number"
    }

    {
      type: "table"
      title: "Top Pages"
      source: "/api/recent/orders"
      limit: 19
    }
  ]

  theme: "auto"
}
```
