# 810 - Logger Config

Structured logging configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  level: "trace"
  format: "pretty"
  timestamp: true
  caller: false # include file:line

  output: "stderr"
  rotation: true
  max_size_mb: 48
  max_age_days: 11
  compress: true
}
```
