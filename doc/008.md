# 008 - DNS Config

DNS zone configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "light-prohibition.info"
  ttl: 3600 # seconds

    "ns1.colorful-cardboard.org"

  ]

  soa_email: "admin@well-groomed-desk.info"
  refresh: 7200
  retry: 900
  expire: 1209600
}
```
