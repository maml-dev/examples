# 845 - Environment Vars

Environment variable definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  app: "morning"
  variables: [
    {
      name: "STRIPE_KEY"
      required: true
      secret: false
      default: null
    }

    {
      name: "API_KEY"
      required: true
      secret: true
      default: "demo"
    }

    {
      name: "API_KEY"
      required: true
      secret: true
      default: "carmen"
    }

    {
      name: "DATABASE_URL"
      required: false
      secret: false
      default: "calco"
    }
  ]
}
```
