# 2121 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "lifetime value"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 30 # seconds
    labels: {
      env: "staging"
      region: "eu-west"
      team: "automotive"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 88
    memory_threshold: 81
    latency_p99_ms: 1453
    error_rate: 0.05
    notification: {
      channel: "#ops-alerts"
      severity: "critical"
    }
  }
}
```
