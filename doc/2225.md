# 2225 - Theme Config

Design system theme configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  name: "familiar Theme"
  # Color palette
  colors: {
    primary: "#9b3dc2"
    secondary: "#9eaabf"
    accent: "#ec9c6d"
    background: "#f8f9fa"
    text: "#333333"

    error: "#dc3545"
    warning: "#ffc107"
    success: "#28a745"
  }

  # Typography
  typography: {
    font_family: "system-ui, sans-serif"
    base_size: 18
    line_height: 1.75
    headings: {
      font_family: "Inter, sans-serif"
      weight: 700
    }
  }

  spacing: {
    unit: 4
    scale: [
      0
      4
      8
      16
      24
      32
      48
    ]
  }
  border_radius: 8
}
```
