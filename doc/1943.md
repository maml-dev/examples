# 1943 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "sublime"
  shortcuts: [
    {
      key: "Alt+Up"
      action: "toggle_terminal"
      when: "global"
    }

    {
      key: "Ctrl+`"
      action: "search_project"
      when: "editor"
    }

    {
      key: "Ctrl+/"
      action: "command_palette"
      when: "editor"
    }

    {
      key: "Cmd+K"
      action: "debug_run"
      when: "global"
    }

    {
      key: "Ctrl+Shift+F"
      action: "command_palette"
      when: "editor"
    }
  ]
}
```
