# 644 - Cron Jobs

Scheduled task definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  timezone: "UTC"
  jobs: [
    {
      name: "rotate-logs"
      schedule: "*/15 * * * *"
      command: "./bin/sync --all"
      enabled: true
    }

    {
      name: "refresh-cache"
      schedule: "0 0 * * *"
      command: "pg_dump $DATABASE_URL"
      enabled: false
    }

    {
      name: "cleanup-temp"
      schedule: "0 0 * * *"
      command: "./bin/sync --all"
      enabled: true
    }
  ]
}
```
