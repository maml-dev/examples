# 3982 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@ironclad-mobility.com"
  subject: "Curatio vociferor aqua spargo curiositas caveo."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Aveho sollers villa. At assentator succurro adulescens confero porro corpus strenuus cupiditate. Dapifer cinis sustineo quis vitiosus debilito ago tubineus vespillo adsidue.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Ex cattus adsidue solvo trepide crastinus veritatis consectetur reiciendis. Crur congregatio calco bos certe accusamus denego. Adstringo angustus avarus vinculum terra.

Visit: {{url}}
"""
}
```
