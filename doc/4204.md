# 4204 - Proxy Config

Reverse proxy settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  listen: 8443
  server_name: "spiffy-plain.info"
  upstream: "http://127.0.0.1:8630"

  # TLS settings
  ssl: false
  http2: true

  gzip: true
  gzip_min_length: 512
  client_max_body: "26mb"
}
```
