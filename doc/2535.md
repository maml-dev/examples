# 2535 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "metrics"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 30 # seconds
    labels: {
      env: "prod"
      region: "us-east"
      team: "movies"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 78
    memory_threshold: 76
    latency_p99_ms: 1624
    error_rate: 0.02
    notification: {
      channel: "#incidents"
      severity: "warning"
    }
  }
}
```
