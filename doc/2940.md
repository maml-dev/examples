# 2940 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "palatable-arcade.biz"
  inbound: {
    mx_records: [
      "mx1.palatable-arcade.biz"
      "mx2.palatable-arcade.biz"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "reject"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.palatable-arcade.biz"
    port: 465
    tls: true
    rate_limit: {
      per_minute: 288
      per_hour: 7718
    }
  }
}
```
