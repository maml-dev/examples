# 2954 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "vscode"
  shortcuts: [
    {
      key: "Ctrl+D"
      action: "search_project"
      when: "debug"
    }

    {
      key: "Cmd+Shift+L"
      action: "toggle_sidebar"
      when: "global"
    }

    {
      key: "Ctrl+D"
      action: "command_palette"
      when: "editor"
    }

    {
      key: "Ctrl+B"
      action: "debug_run"
      when: "terminal"
    }
  ]
}
```
