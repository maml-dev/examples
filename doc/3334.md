# 3334 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "metrics"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 30 # seconds
    labels: {
      env: "prod"
      region: "ap-south"
      team: "computers"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 82
    memory_threshold: 90
    latency_p99_ms: 827
    error_rate: 0.05
    notification: {
      channel: "#incidents"
      severity: "critical"
    }
  }
}
```
