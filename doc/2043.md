# 2043 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "self-reliant-chasuble.name"
  records: [
    {
      type: "NS"
      name: "@"
      value: "mean-vicinity.com"
      ttl: 86400
    }

    {
      type: "A"
      name: "mail"
      value: "248.86.12.69"
      ttl: 86400
    }

    {
      type: "MX"
      name: "cdn"
      value: "v=spf1 include:_spf.intelligent-godparent.info ~all"
      ttl: 300
    }

    {
      type: "TXT"
      name: "*"
      value: "36.30.61.184"
      ttl: 3600
    }
  ]
}
```
