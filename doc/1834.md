# 1834 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "meadow-web"
    namespace: "staging"
    labels: {
      app: "cod"
      tier: "backend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 2
    container: {
      image: "surface:0.17.5"
      port: 8000
      resources: {
        cpu_limit: "500m"
        memory_limit: "512Mi"
      }
    }

    health_check: {
      path: "/health"
      interval: 10
      timeout: 10
    }
  }
}
```
