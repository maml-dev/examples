# 3347 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "vim"
  shortcuts: [
    {
      key: "Ctrl+D"
      action: "save"
      when: "editor"
    }

    {
      key: "Ctrl+`"
      action: "format"
      when: "global"
    }

    {
      key: "Cmd+K"
      action: "toggle_sidebar"
      when: "global"
    }

    {
      key: "Ctrl+`"
      action: "toggle_terminal"
      when: "debug"
    }

    {
      key: "Ctrl+Shift+P"
      action: "toggle_terminal"
      when: "terminal"
    }

    {
      key: "Ctrl+Shift+F"
      action: "format"
      when: "terminal"
    }
  ]
}
```
