# 2525 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "users"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 30 # seconds
    labels: {
      env: "staging"
      region: "eu-west"
      team: "home"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 95
    memory_threshold: 84
    latency_p99_ms: 974
    error_rate: 0.05
    notification: {
      channel: "#incidents"
      severity: "warning"
    }
  }
}
```
