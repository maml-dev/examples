# 647 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "vim"
  shortcuts: [
    {
      key: "Ctrl+Shift+F"
      action: "find_references"
      when: "global"
    }

    {
      key: "Ctrl+`"
      action: "command_palette"
      when: "editor"
    }

    {
      key: "Ctrl+Shift+F"
      action: "format"
      when: "global"
    }

    {
      key: "Ctrl+B"
      action: "search_project"
      when: "editor"
    }
  ]
}
```
