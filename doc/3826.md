# 3826 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "trick-web"
    namespace: "default"
    labels: {
      app: "newsstand"
      tier: "frontend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 2
    container: {
      image: "cap:4.4.18"
      port: 8080
      resources: {
        cpu_limit: "1000m"
        memory_limit: "512Mi"
      }
    }

    health_check: {
      path: "/ready"
      interval: 15
      timeout: 10
    }
  }
}
```
