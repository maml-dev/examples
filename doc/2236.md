# 2236 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "birth-worker"
    namespace: "staging"
    labels: {
      app: "fledgling"
      tier: "frontend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 4
    container: {
      image: "relative:8.3.2"
      port: 8080
      resources: {
        cpu_limit: "500m"
        memory_limit: "512Mi"
      }
    }

    health_check: {
      path: "/ready"
      interval: 15
      timeout: 10
    }
  }
}
```
