# 2114 - Proxy Config

Reverse proxy settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  listen: 443
  server_name: "responsible-toothbrush.com"
  upstream: "http://localhost:4538"

  # TLS settings
  ssl: true
  http2: true

  gzip: true
  gzip_min_length: 256
  client_max_body: "46mb"
}
```
