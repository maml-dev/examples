# 3991 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@enlightened-elver.com"
  subject: "Aperte supellex soluta."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Ancilla unde admoveo cursim arcesso quia substantia talus. Defero pariatur umquam aqua traho. Abeo barba crapula toties alienus aggero carus laudantium doloribus conduco.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Tempore cohaero tollo ex. Aegrus colligo strenuus nihil attonbitus ipsa succedo currus denego communis. Agnitio territo sunt suffragium cumque cursim vulticulus viscus.

Visit: {{url}}
"""
}
```
