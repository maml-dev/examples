# 1011 - Logger Config

Structured logging configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  level: "info"
  format: "pretty"
  timestamp: true
  caller: true # include file:line

  output: "stderr"
  rotation: true
  max_size_mb: 18
  max_age_days: 44
  compress: true
}
```
