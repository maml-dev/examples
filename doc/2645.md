# 2645 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "plump-help.org"
  records: [
    {
      type: "CNAME"
      name: "@"
      value: "dim-wheel.net"
      ttl: 86400
    }

    {
      type: "CNAME"
      name: "api"
      value: "v=spf1 include:_spf.flickering-simple.name ~all"
      ttl: 3600
    }

    {
      type: "CNAME"
      name: "cdn"
      value: "166.129.165.189"
      ttl: 3600
    }

    {
      type: "TXT"
      name: "mail"
      value: "rotating-steeple.info"
      ttl: 300
    }
  ]
}
```
