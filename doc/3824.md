# 3824 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "paradigms"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 60 # seconds
    labels: {
      env: "prod"
      region: "us-east"
      team: "shoes"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 80
    memory_threshold: 89
    latency_p99_ms: 789
    error_rate: 0.02
    notification: {
      channel: "#ops-alerts"
      severity: "critical"
    }
  }
}
```
