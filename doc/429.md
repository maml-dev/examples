# 429 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "redis"
  connection: {
    host: "mq.internal"
    port: 5672
    vhost: "/production"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 1
      retry: 4
      timeout: 32886 # ms
    }

    priority: {
      concurrency: 13
      retry: 9
      timeout: 3949 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 8
    queue: "dead-letter"
  }
}
```
