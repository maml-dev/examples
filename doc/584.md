# 584 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@suburban-repeat.net"
  subject: "Vulariter cursim valeo decerno crux."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Validus impedit suasoria. Admiratio accusantium tendo perspiciatis dedecor cuppedia vae. Crepusculum turba agnitio veritas claudeo harum aqua.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Sortitus aequus a talio suasoria ancilla cribro laboriosam. Color substantia tripudio clementia autem ulciscor. Vapulus ubi terreo ustulo tribuo tubineus degenero bibo.

Visit: {{url}}
"""
}
```
