# 3249 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "emacs"
  shortcuts: [
    {
      key: "Ctrl+B"
      action: "find_references"
      when: "terminal"
    }

    {
      key: "Ctrl+D"
      action: "debug_run"
      when: "editor"
    }

    {
      key: "Ctrl+`"
      action: "search_project"
      when: "editor"
    }

    {
      key: "Ctrl+Shift+P"
      action: "command_palette"
      when: "terminal"
    }

    {
      key: "Ctrl+B"
      action: "search_project"
      when: "editor"
    }

    {
      key: "Ctrl+/"
      action: "select_all_occurrences"
      when: "global"
    }

    {
      key: "Ctrl+S"
      action: "search_project"
      when: "editor"
    }
  ]
}
```
