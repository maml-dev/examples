# 3787 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@limping-schedule.info"
  subject: "Candidus copiose curia cicuta."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Quas ara utpote aspicio ventus beatus. Terror defessus vorago cohaero veniam auxilium ascit sulum facere. Sopor caelum cilicium curo amor tandem blanditiis subito quo.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Amicitia agnosco stips stipes viridis. Aggredior abutor stipes assentator dens amitto cupressus cohors. Vallum asperiores arbor abbas absque sonitus deludo delectatio admitto enim.

Visit: {{url}}
"""
}
```
