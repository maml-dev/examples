# 1128 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.impassioned-heartbeat.info"
    port: 8443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "15m"
    max_requests: 4355
    by: "user"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://lost-final.com"
      "https://app.likable-chow.net"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 86400
  }

  cache: {
    enabled: true
    ttl: 1510
    store: "redis"
  }
}
```
