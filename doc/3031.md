# 3031 - Terraform Resource

Infrastructure as code resource definition. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  provider: "digitalocean"
  region: "us-east-1"
  resources: {
    # Network
    vpc: {
      cidr_block: "10.234.0.0/16"
      enable_dns: true
      tags: {
        Name: "packaging-vpc"
        Environment: "staging"
      }
    }

    # Compute
    instance: {
      ami: "ami-5be6aed5"
      type: "m5.large"
      key_name: "curl-key"
      monitoring: true
      root_volume: {
        size: 50
        type: "gp3"
        encrypted: true
      }
    }

    # Security
    security_group: {
      name: "allow-ssh"
      ingress_port: 80
      ingress_cidr: "0.0.0.0/0"
    }
  }
}
```
