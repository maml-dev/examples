# 182 - Quoted Keys

Configuration using special characters in keys. This is an example of a [MAML](https://maml.dev) document.

```maml
# HTTP Headers
{
  Content-Type: "application/json"
  X-Request-ID: "6b40565c-29b9-4934-9a5a-5fc081a37ce2"
  Accept-Language: "de-DE"
  Cache-Control: "public, max-age=86400"
  X-Powered-By: "Koa"

  # Keys with spaces get quoted
  "headers with spaces": {
    "custom header": "sunt"
    "another one": "viridis"
  }
}
```
