# 2942 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "admin"
      permissions: [
        "view_analytics"
        "read"
        "manage_billing"
      ]
      max_seats: null
    }

    {
      name: "support"
      permissions: [
        "write"
        "read"
        "export_data"
        "manage_users"
      ]
      max_seats: 15
    }

    {
      name: "admin"
      permissions: [
        "manage_billing"
        "export_data"
        "read"
        "manage_users"
      ]
      max_seats: null
    }

    {
      name: "admin"
      permissions: [
        "write"
        "read"
        "manage_billing"
        "manage_users"
      ]
      max_seats: 40
    }

    {
      name: "support"
      permissions: [
        "view_analytics"
        "write"
        "manage_users"
      ]
      max_seats: 15
    }
  ]
}
```
