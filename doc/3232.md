# 3232 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "paradigms"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 30 # seconds
    labels: {
      env: "staging"
      region: "us-east"
      team: "jewelry"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 87
    memory_threshold: 83
    latency_p99_ms: 1876
    error_rate: 0.02
    notification: {
      channel: "#oncall"
      severity: "warning"
    }
  }
}
```
