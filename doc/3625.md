# 3625 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "experiences"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 30 # seconds
    labels: {
      env: "prod"
      region: "us-east"
      team: "industrial"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 83
    memory_threshold: 87
    latency_p99_ms: 252
    error_rate: 0.01
    notification: {
      channel: "#ops-alerts"
      severity: "critical"
    }
  }
}
```
