# 1230 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.soggy-swine.org"
    port: 8443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "5m"
    max_requests: 849
    by: "ip"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://worldly-nudge.biz"
      "https://app.dearest-boyfriend.com"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 7200
  }

  cache: {
    enabled: false
    ttl: 2206
    store: "memory"
  }
}
```
