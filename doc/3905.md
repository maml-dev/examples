# 3905 - Proxy Config

Reverse proxy settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  listen: 443
  server_name: "grimy-premium.com"
  upstream: "http://localhost:8521"

  # TLS settings
  ssl: false
  http2: true

  gzip: true
  gzip_min_length: 1024
  client_max_body: "33mb"
}
```
