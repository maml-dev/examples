# 2723 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "silver-insolence.name"
  inbound: {
    mx_records: [
      "mx1.silver-insolence.name"
      "mx2.silver-insolence.name"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.silver-insolence.name"
    port: 25
    tls: true
    rate_limit: {
      per_minute: 119
      per_hour: 9391
    }
  }
}
```
