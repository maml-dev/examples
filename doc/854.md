# 854 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "parallel-antelope.org"
  records: [
    {
      type: "CNAME"
      name: "www"
      value: "flashy-rim.name"
      ttl: 86400
    }

    {
      type: "A"
      name: "@"
      value: "v=spf1 include:_spf.intelligent-maintainer.org ~all"
      ttl: 86400
    }

    {
      type: "CNAME"
      name: "mail"
      value: "38.162.171.11"
      ttl: 3600
    }

    {
      type: "AAAA"
      name: "api"
      value: "v=spf1 include:_spf.unlucky-nudge.name ~all"
      ttl: 3600
    }

    {
      type: "CNAME"
      name: "*"
      value: "careless-dash.com"
      ttl: 86400
    }
  ]
}
```
