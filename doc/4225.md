# 4225 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "hornet-worker"
    namespace: "production"
    labels: {
      app: "stool"
      tier: "backend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 5
    container: {
      image: "marketplace:2.12.17"
      port: 9090
      resources: {
        cpu_limit: "1000m"
        memory_limit: "512Mi"
      }
    }

    health_check: {
      path: "/ready"
      interval: 10
      timeout: 10
    }
  }
}
```
