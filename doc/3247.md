# 3247 - Cron Jobs

Scheduled task definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  timezone: "Europe/London"
  jobs: [
    {
      name: "rotate-logs"
      schedule: "*/15 * * * *"
      command: "./bin/sync --all"
      enabled: false
    }

    {
      name: "backup-db"
      schedule: "30 6 * * 1-5"
      command: "python manage.py report"
      enabled: false
    }

    {
      name: "cleanup-temp"
      schedule: "0 0 * * *"
      command: "python manage.py report"
      enabled: true
    }

    {
      name: "rotate-logs"
      schedule: "0 0 * * *"
      command: "node scripts/cleanup.js"
      enabled: false
    }

    {
      name: "refresh-cache"
      schedule: "0 0 * * *"
      command: "pg_dump $DATABASE_URL"
      enabled: false
    }
  ]
}
```
