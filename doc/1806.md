# 1806 - DNS Config

DNS zone configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "male-providence.net"
  ttl: 86400 # seconds
  nameservers: [
    "ns1.annual-bathrobe.org"
    "ns2.weekly-gallery.com"
  ]

  soa_email: "admin@worst-yarmulke.org"
  refresh: 3600
  retry: 1800
  expire: 604800
}
```
