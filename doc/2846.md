# 2846 - Cron Jobs

Scheduled task definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  timezone: "America/New_York"
  jobs: [
    {
      name: "refresh-cache"
      schedule: "30 6 * * 1-5"
      command: "python manage.py report"
      enabled: true
    }

    {
      name: "cleanup-temp"
      schedule: "0 2 * * 0"
      command: "pg_dump $DATABASE_URL"
      enabled: false
    }

    {
      name: "refresh-cache"
      schedule: "0 0 * * *"
      command: "pg_dump $DATABASE_URL"
      enabled: true
    }

    {
      name: "cleanup-temp"
      schedule: "*/15 * * * *"
      command: "node scripts/cleanup.js"
      enabled: false
    }
  ]
}
```
