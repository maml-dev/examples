# 537 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "rationale-web"
    namespace: "production"
    labels: {
      app: "chairperson"
      tier: "backend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 1
    container: {
      image: "poppy:4.19.14"
      port: 9090
      resources: {
        cpu_limit: "1000m"
        memory_limit: "512Mi"
      }
    }

    health_check: {
      path: "/healthz"
      interval: 10
      timeout: 10
    }
  }
}
```
