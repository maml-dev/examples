# 1434 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "vicinity-web"
    namespace: "production"
    labels: {
      app: "tomography"
      tier: "frontend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 4
    container: {
      image: "tomatillo:1.7.9"
      port: 3000
      resources: {
        cpu_limit: "500m"
        memory_limit: "256Mi"
      }
    }

    health_check: {
      path: "/health"
      interval: 15
      timeout: 5
    }
  }
}
```
