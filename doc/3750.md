# 3750 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "vscode"
  shortcuts: [
    {
      key: "Alt+F7"
      action: "debug_run"
      when: "terminal"
    }

    {
      key: "Ctrl+Shift+P"
      action: "command_palette"
      when: "debug"
    }

    {
      key: "Ctrl+/"
      action: "toggle_sidebar"
      when: "debug"
    }

    {
      key: "Cmd+K"
      action: "select_all_occurrences"
      when: "terminal"
    }

    {
      key: "Ctrl+D"
      action: "search_project"
      when: "global"
    }

    {
      key: "Alt+F7"
      action: "find_references"
      when: "debug"
    }
  ]
}
```
