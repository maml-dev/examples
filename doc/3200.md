# 3200 - Quoted Keys

Configuration using special characters in keys. This is an example of a [MAML](https://maml.dev) document.

```maml
# HTTP Headers
{
  Content-Type: "application/xml"
  X-Request-ID: "5f900c16-425f-44f6-ba19-866e75a71f6c"
  Accept-Language: "ja-JP"
  Cache-Control: "public, max-age=86400"
  X-Powered-By: "Fastify"

  # Keys with spaces get quoted
  "headers with spaces": {
    "custom header": "argumentum"
    "another one": "tricesimus"
  }
}
```
