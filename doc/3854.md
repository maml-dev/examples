# 3854 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v1"
  routes: [
    {
      method: "GET"
      path: "/users"
      handler: "updateOrders"
      auth: true
    }

    {
      method: "GET"
      path: "/posts"
      handler: "getPosts"
      auth: false
    }

    {
      method: "DELETE"
      path: "/orders/:id"
      handler: "listUsers"
      auth: true
    }

    {
      method: "POST"
      path: "/users/:id"
      handler: "createOrders"
      auth: true
    }
  ]
}
```
