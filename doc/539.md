# 539 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "applications"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 15 # seconds
    labels: {
      env: "staging"
      region: "ap-south"
      team: "baby"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 89
    memory_threshold: 83
    latency_p99_ms: 1428
    error_rate: 0.02
    notification: {
      channel: "#ops-alerts"
      severity: "info"
    }
  }
}
```
