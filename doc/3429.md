# 3429 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "numeracy-worker"
    namespace: "default"
    labels: {
      app: "submitter"
      tier: "frontend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 3
    container: {
      image: "zebra:6.6.15"
      port: 3000
      resources: {
        cpu_limit: "250m"
        memory_limit: "512Mi"
      }
    }

    health_check: {
      path: "/health"
      interval: 15
      timeout: 5
    }
  }
}
```
