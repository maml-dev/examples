# 4315 - Logger Config

Structured logging configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  level: "fatal"
  format: "pretty"
  timestamp: true
  caller: true # include file:line

  output: "stderr"
  rotation: false
  max_size_mb: 126
  max_age_days: 86
  compress: true
}
```
