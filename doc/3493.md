# 3493 - Quoted Keys

Configuration using special characters in keys. This is an example of a [MAML](https://maml.dev) document.

```maml
# HTTP Headers
{
  Content-Type: "application/json"
  X-Request-ID: "99f34372-c5b0-416c-a894-3c33d946775c"
  Accept-Language: "de-DE"
  Cache-Control: "max-age=3600"
  X-Powered-By: "Hono"

  # Keys with spaces get quoted
  "headers with spaces": {
    "custom header": "conservo"
    "another one": "contigo"
  }
}
```
