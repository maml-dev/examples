# 838 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "dental-transparency.info"
  inbound: {
    mx_records: [
      "mx1.dental-transparency.info"
      "mx2.dental-transparency.info"
    ]
    spam_filter: {
      enabled: true
      threshold: 5
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.dental-transparency.info"
    port: 25
    tls: true
    rate_limit: {
      per_minute: 287
      per_hour: 1957
    }
  }
}
```
