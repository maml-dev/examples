# 4131 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "redis"
  connection: {
    host: "mq.internal"
    port: 5672
    vhost: "/"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 1
      retry: 3
      timeout: 10069 # ms
    }

    priority: {
      concurrency: 15
      retry: 10
      timeout: 9259 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 6
    queue: "dead-letter"
  }
}
```
