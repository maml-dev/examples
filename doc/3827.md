# 3827 - Backup Strategy

Backup schedule and retention policy. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  source: {
    type: "postgresql"
    host: "db.internal"
    databases: [
      "joy"
      "cinder"
    ]
  }

  # Backup schedule
  schedule: {
    full: "0 2 * * 0" # weekly Sunday 2am
    incremental: "0 2 * * 1-6" # daily except Sunday
    snapshot: "0 */4 * * *" # every 4 hours
  }

  retention: {
    daily: 9
    weekly: 6
    monthly: 6
  }

  destination: {
    type: "local"
    path: "/mnt/backups/charlatan"
    encryption: true
    compression: "lz4"
  }
}
```
