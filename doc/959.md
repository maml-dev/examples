# 959 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v3"
  routes: [
    {
      method: "POST"
      path: "/orders"
      handler: "deleteUsers"
      auth: true
    }

    {
      method: "DELETE"
      path: "/comments"
      handler: "createProducts"
      auth: true
    }

    {
      method: "PATCH"
      path: "/comments"
      handler: "createProducts"
      auth: true
    }

    {
      method: "POST"
      path: "/products/:id"
      handler: "listOrders"
      auth: true
    }

    {
      method: "POST"
      path: "/users"
      handler: "updateProducts"
      auth: false
    }

    {
      method: "GET"
      path: "/comments/:id"
      handler: "updateOrders"
      auth: false
    }

    {
      method: "DELETE"
      path: "/products"
      handler: "deletePosts"
      auth: true
    }
  ]
}
```
