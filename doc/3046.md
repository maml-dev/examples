# 3046 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "emacs"
  shortcuts: [
    {
      key: "Alt+F7"
      action: "format"
      when: "editor"
    }

    {
      key: "Ctrl+B"
      action: "search_project"
      when: "debug"
    }

    {
      key: "Alt+Up"
      action: "debug_run"
      when: "terminal"
    }

    {
      key: "Ctrl+D"
      action: "save"
      when: "editor"
    }

    {
      key: "Ctrl+D"
      action: "toggle_comment"
      when: "global"
    }

    {
      key: "Ctrl+Shift+P"
      action: "debug_run"
      when: "editor"
    }
  ]
}
```
