# 3546 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "billing"
      permissions: [
        "manage_billing"
        "write"
        "view_analytics"
        "delete"
        "manage_users"
      ]
      max_seats: null
    }

    {
      name: "moderator"
      permissions: [
        "manage_billing"
        "write"
        "manage_users"
        "delete"
      ]
      max_seats: 22
    }

    {
      name: "editor"
      permissions: [
        "write"
        "manage_billing"
        "delete"
        "manage_users"
        "export_data"
      ]
      max_seats: 32
    }

    {
      name: "admin"
      permissions: [
        "delete"
        "export_data"
      ]
      max_seats: null
    }
  ]
}
```
