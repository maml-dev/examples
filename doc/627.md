# 627 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "lifetime value"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 15 # seconds
    labels: {
      env: "staging"
      region: "ap-south"
      team: "grocery"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 79
    memory_threshold: 75
    latency_p99_ms: 1981
    error_rate: 0.02
    notification: {
      channel: "#incidents"
      severity: "critical"
    }
  }
}
```
