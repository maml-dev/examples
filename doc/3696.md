# 3696 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@pricey-overheard.org"
  subject: "Consuasor surculus summisse surculus."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Cibus praesentium esse sunt via tandem tabernus ventosus. Callide tendo maiores antepono tempora utilis comitatus. Armarium totus termes blandior nobis cito.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Ater aveho cilicium demum vulgo spoliatio coma fuga vobis. Iure quae valetudo autem canonicus est. Tres cultellus paulatim iure ulciscor ambulo.

Visit: {{url}}
"""
}
```
