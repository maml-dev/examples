# 2958 - Cron Jobs

Scheduled task definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  timezone: "America/New_York"
  jobs: [
    {
      name: "send-reports"
      schedule: "*/15 * * * *"
      command: "./bin/sync --all"
      enabled: false
    }

    {
      name: "send-reports"
      schedule: "*/15 * * * *"
      command: "node scripts/cleanup.js"
      enabled: false
    }

    {
      name: "cleanup-temp"
      schedule: "*/15 * * * *"
      command: "pg_dump $DATABASE_URL"
      enabled: false
    }

    {
      name: "cleanup-temp"
      schedule: "0 0 * * *"
      command: "./bin/sync --all"
      enabled: false
    }
  ]
}
```
