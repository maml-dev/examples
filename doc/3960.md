# 3960 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "emacs"
  shortcuts: [
    {
      key: "Cmd+K"
      action: "format"
      when: "terminal"
    }

    {
      key: "Alt+F7"
      action: "toggle_sidebar"
      when: "terminal"
    }

    {
      key: "Ctrl+`"
      action: "debug_run"
      when: "debug"
    }

    {
      key: "Ctrl+/"
      action: "format"
      when: "terminal"
    }
  ]
}
```
