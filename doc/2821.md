# 2821 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "users"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 60 # seconds
    labels: {
      env: "prod"
      region: "eu-west"
      team: "music"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 73
    memory_threshold: 82
    latency_p99_ms: 1645
    error_rate: 0.05
    notification: {
      channel: "#ops-alerts"
      severity: "info"
    }
  }
}
```
