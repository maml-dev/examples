# 3857 - Cron Jobs

Scheduled task definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  timezone: "UTC"
  jobs: [
    {
      name: "cleanup-temp"
      schedule: "0 0 * * *"
      command: "python manage.py report"
      enabled: false
    }

    {
      name: "sync-data"
      schedule: "0 * * * *"
      command: "python manage.py report"
      enabled: true
    }

    {
      name: "refresh-cache"
      schedule: "0 2 * * 0"
      command: "python manage.py report"
      enabled: true
    }

    {
      name: "sync-data"
      schedule: "*/15 * * * *"
      command: "python manage.py report"
      enabled: false
    }

    {
      name: "cleanup-temp"
      schedule: "30 6 * * 1-5"
      command: "./bin/sync --all"
      enabled: true
    }
  ]
}
```
