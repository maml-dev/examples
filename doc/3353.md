# 3353 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "admin"
      permissions: [
        "delete"
        "write"
        "read"
      ]
      max_seats: 36
    }

    {
      name: "support"
      permissions: [
        "view_analytics"
        "read"
        "write"
        "manage_billing"
        "delete"
      ]
      max_seats: 50
    }

    {
      name: "moderator"
      permissions: [
        "view_analytics"
        "read"
        "export_data"
      ]
      max_seats: null
    }
  ]
}
```
