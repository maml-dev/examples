# 2810 - Logger Config

Structured logging configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  level: "fatal"
  format: "pretty"
  timestamp: true
  caller: false # include file:line

  output: "stdout"
  rotation: false
  max_size_mb: 229
  max_age_days: 32
  compress: true
}
```
