# 134 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "kafka"
  connection: {
    host: "localhost"
    port: 9092
    vhost: "/"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 2
      retry: 4
      timeout: 45579 # ms
    }

    priority: {
      concurrency: 5
      retry: 8
      timeout: 9457 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 10
    queue: "dead-letter"
  }
}
```
