# 1842 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "moderator"
      permissions: [
        "write"
        "read"
      ]
      max_seats: 20
    }

    {
      name: "editor"
      permissions: [
        "write"
        "delete"
        "view_analytics"
        "export_data"
        "read"
      ]
      max_seats: 42
    }

    {
      name: "editor"
      permissions: [
        "export_data"
        "manage_users"
        "delete"
        "write"
      ]
      max_seats: null
    }
  ]
}
```
