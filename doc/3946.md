# 3946 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "moderator"
      permissions: [
        "export_data"
        "manage_users"
        "view_analytics"
        "read"
        "manage_billing"
      ]
      max_seats: 42
    }

    {
      name: "viewer"
      permissions: [
        "export_data"
        "delete"
      ]
      max_seats: 23
    }

    {
      name: "admin"
      permissions: [
        "manage_billing"
        "read"
        "view_analytics"
      ]
      max_seats: 50
    }
  ]
}
```
