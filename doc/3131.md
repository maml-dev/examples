# 3131 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "antelope-api"
    namespace: "production"
    labels: {
      app: "alert"
      tier: "frontend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 2
    container: {
      image: "chapel:8.11.10"
      port: 8000
      resources: {
        cpu_limit: "1000m"
        memory_limit: "1Gi"
      }
    }

    health_check: {
      path: "/healthz"
      interval: 10
      timeout: 3
    }
  }
}
```
