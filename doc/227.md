# 227 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "winding-solution.info"
  inbound: {
    mx_records: [
      "mx1.winding-solution.info"
      "mx2.winding-solution.info"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.winding-solution.info"
    port: 587
    tls: true
    rate_limit: {
      per_minute: 381
      per_hour: 3924
    }
  }
}
```
