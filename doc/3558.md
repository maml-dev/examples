# 3558 - Cron Jobs

Scheduled task definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  timezone: "UTC"
  jobs: [
    {
      name: "rotate-logs"
      schedule: "0 * * * *"
      command: "python manage.py report"
      enabled: true
    }

    {
      name: "refresh-cache"
      schedule: "*/15 * * * *"
      command: "python manage.py report"
      enabled: false
    }

    {
      name: "sync-data"
      schedule: "30 6 * * 1-5"
      command: "pg_dump $DATABASE_URL"
      enabled: true
    }

    {
      name: "refresh-cache"
      schedule: "*/15 * * * *"
      command: "./bin/sync --all"
      enabled: false
    }

    {
      name: "send-reports"
      schedule: "0 2 * * 0"
      command: "./bin/sync --all"
      enabled: false
    }

    {
      name: "send-reports"
      schedule: "*/15 * * * *"
      command: "python manage.py report"
      enabled: false
    }
  ]
}
```
