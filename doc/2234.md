# 2234 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.unwilling-tray.info"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "1m"
    max_requests: 7544
    by: "ip"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://queasy-release.com"
      "https://app.hungry-gray.biz"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 7200
  }

  cache: {
    enabled: true
    ttl: 77
    store: "redis"
  }
}
```
