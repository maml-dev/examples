# 1558 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "sublime"
  shortcuts: [
    {
      key: "Alt+F7"
      action: "select_all_occurrences"
      when: "global"
    }

    {
      key: "Ctrl+Shift+P"
      action: "select_all_occurrences"
      when: "terminal"
    }

    {
      key: "Alt+Up"
      action: "toggle_sidebar"
      when: "editor"
    }

    {
      key: "Ctrl+B"
      action: "toggle_comment"
      when: "editor"
    }

    {
      key: "Ctrl+S"
      action: "find_references"
      when: "global"
    }

    {
      key: "Alt+Up"
      action: "command_palette"
      when: "editor"
    }
  ]
}
```
