# 2401 - Email Config

SMTP email service settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  smtp_host: "smtp.dapper-schnitzel.biz"
  smtp_port: 465
  use_tls: true
  from_name: "Mayer-Vandervort Group"
  from_email: "noreply@dapper-schnitzel.biz"

  # Rate limiting
  max_batch_size: 405
  retry_interval: 281
}
```
