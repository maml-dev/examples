# 2143 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "support"
      permissions: [
        "delete"
        "manage_billing"
        "manage_users"
        "export_data"
      ]
      max_seats: 13
    }

    {
      name: "editor"
      permissions: [
        "delete"
        "manage_billing"
      ]
      max_seats: 17
    }

    {
      name: "billing"
      permissions: [
        "view_analytics"
        "manage_users"
        "export_data"
      ]
      max_seats: null
    }

    {
      name: "editor"
      permissions: [
        "write"
        "manage_users"
        "view_analytics"
        "manage_billing"
        "export_data"
      ]
      max_seats: 13
    }

    {
      name: "viewer"
      permissions: [
        "export_data"
        "view_analytics"
        "write"
        "read"
      ]
      max_seats: 21
    }
  ]
}
```
