# 260 - Cron Jobs

Scheduled task definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  timezone: "UTC"
  jobs: [
    {
      name: "cleanup-temp"
      schedule: "0 * * * *"
      command: "pg_dump $DATABASE_URL"
      enabled: false
    }

    {
      name: "rotate-logs"
      schedule: "*/15 * * * *"
      command: "node scripts/cleanup.js"
      enabled: false
    }

    {
      name: "cleanup-temp"
      schedule: "0 * * * *"
      command: "pg_dump $DATABASE_URL"
      enabled: false
    }

    {
      name: "rotate-logs"
      schedule: "0 * * * *"
      command: "./bin/sync --all"
      enabled: false
    }
  ]
}
```
