# 3330 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "programme-web"
    namespace: "default"
    labels: {
      app: "dish"
      tier: "data"
    }
  }

  # Deployment spec
  spec: {
    replicas: 4
    container: {
      image: "singing:5.19.8"
      port: 3000
      resources: {
        cpu_limit: "1000m"
        memory_limit: "1Gi"
      }
    }

    health_check: {
      path: "/healthz"
      interval: 15
      timeout: 10
    }
  }
}
```
