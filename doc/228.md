# 228 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "rabbitmq"
  connection: {
    host: "localhost"
    port: 9092
    vhost: "/"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 1
      retry: 5
      timeout: 6005 # ms
    }

    priority: {
      concurrency: 9
      retry: 6
      timeout: 9114 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 10
    queue: "dead-letter"
  }
}
```
