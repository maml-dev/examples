# 4305 - Proxy Config

Reverse proxy settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  listen: 8443
  server_name: "clear-cut-diver.info"
  upstream: "http://localhost:6561"

  # TLS settings
  ssl: false
  http2: true

  gzip: true
  gzip_min_length: 1024
  client_max_body: "15mb"
}
```
