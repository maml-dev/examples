# 3239 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "schemas"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 60 # seconds
    labels: {
      env: "prod"
      region: "eu-west"
      team: "games"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 74
    memory_threshold: 77
    latency_p99_ms: 1774
    error_rate: 0.01
    notification: {
      channel: "#ops-alerts"
      severity: "warning"
    }
  }
}
```
