# 1942 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v3"
  routes: [
    {
      method: "PUT"
      path: "/users"
      handler: "deleteOrders"
      auth: false
    }

    {
      method: "GET"
      path: "/products/:id"
      handler: "getUsers"
      auth: true
    }

    {
      method: "POST"
      path: "/orders"
      handler: "getOrders"
      auth: true
    }
  ]
}
```
