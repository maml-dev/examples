# 2116 - Logger Config

Structured logging configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  level: "trace"
  format: "logfmt"
  timestamp: true
  caller: false # include file:line

  output: "stderr"
  rotation: true
  max_size_mb: 65
  max_age_days: 88
  compress: true
}
```
