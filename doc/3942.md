# 3942 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v3"
  routes: [
    {
      method: "GET"
      path: "/posts/:id"
      handler: "updateUsers"
      auth: false
    }

    {
      method: "PUT"
      path: "/users/:id"
      handler: "createProducts"
      auth: false
    }

    {
      method: "DELETE"
      path: "/posts"
      handler: "updateProducts"
      auth: true
    }
  ]
}
```
