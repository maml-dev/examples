# 1429 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "dependency-worker"
    namespace: "production"
    labels: {
      app: "expense"
      tier: "backend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 3
    container: {
      image: "mixture:0.11.0"
      port: 8000
      resources: {
        cpu_limit: "250m"
        memory_limit: "1Gi"
      }
    }

    health_check: {
      path: "/ready"
      interval: 30
      timeout: 3
    }
  }
}
```
