# 3722 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "users"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 60 # seconds
    labels: {
      env: "staging"
      region: "eu-west"
      team: "toys"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 82
    memory_threshold: 76
    latency_p99_ms: 815
    error_rate: 0.01
    notification: {
      channel: "#ops-alerts"
      severity: "info"
    }
  }
}
```
