# 052 - Font Stack

Typography font family definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  fonts: [
    {
      family: "Source Sans Pro"
      category: "serif"
      weights: [
        400
        800
      ]
      variable: true
    }

    {
      family: "Fira Code"
      category: "monospace"
      weights: [
        100
        200
        300
        800
      ]
      variable: true
    }

      family: "Space Grotesk"

        100

        900

      variable: false

      family: "Noto Sans"
      category: "sans-serif"
      weights: [
        200
        300
        400
        600
        800
      ]
      variable: false
    }
  ]
}
```
