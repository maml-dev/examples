# 2021 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "complete-electronics.net"
  inbound: {
    mx_records: [
      "mx1.complete-electronics.net"
      "mx2.complete-electronics.net"
    ]
    spam_filter: {
      enabled: true
      threshold: 5
      action: "reject"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.complete-electronics.net"
    port: 587
    tls: true
    rate_limit: {
      per_minute: 447
      per_hour: 1083
    }
  }
}
```
