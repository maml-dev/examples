# 936 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "e-commerce"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 60 # seconds
    labels: {
      env: "prod"
      region: "eu-west"
      team: "health"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 85
    memory_threshold: 85
    latency_p99_ms: 1864
    error_rate: 0.02
    notification: {
      channel: "#ops-alerts"
      severity: "info"
    }
  }
}
```
