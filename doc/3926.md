# 3926 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "riser-web"
    namespace: "default"
    labels: {
      app: "presume"
      tier: "frontend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 1
    container: {
      image: "travel:0.15.6"
      port: 8000
      resources: {
        cpu_limit: "500m"
        memory_limit: "1Gi"
      }
    }

    health_check: {
      path: "/health"
      interval: 30
      timeout: 10
    }
  }
}
```
