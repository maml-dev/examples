# 1017 - Proxy Config

Reverse proxy settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  listen: 8443
  server_name: "quiet-birdcage.net"
  upstream: "http://localhost:4608"

  # TLS settings
  ssl: false
  http2: true

  gzip: true
  gzip_min_length: 512
  client_max_body: "46mb"
}
```
