# 1951 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "smoggy-blight.info"
  records: [
    {
      type: "AAAA"
      name: "api"
      value: "warmhearted-ceramics.biz"
      ttl: 3600
    }

    {
      type: "TXT"
      name: "mail"
      value: "184.14.26.192"
      ttl: 86400
    }

    {
      type: "AAAA"
      name: "staging"
      value: "17.42.157.150"
      ttl: 3600
    }

    {
      type: "A"
      name: "*"
      value: "0.113.153.93"
      ttl: 3600
    }
  ]
}
```
