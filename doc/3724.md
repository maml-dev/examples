# 3724 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "crushing-outrun.net"
  inbound: {
    mx_records: [
      "mx1.crushing-outrun.net"
      "mx2.crushing-outrun.net"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.crushing-outrun.net"
    port: 465
    tls: true
    rate_limit: {
      per_minute: 153
      per_hour: 3369
    }
  }
}
```
