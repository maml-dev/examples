# 733 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "second-rim.info"
  inbound: {
    mx_records: [
      "mx1.second-rim.info"
      "mx2.second-rim.info"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "reject"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.second-rim.info"
    port: 25
    tls: true
    rate_limit: {
      per_minute: 299
      per_hour: 3685
    }
  }
}
```
