# 2214 - Proxy Config

Reverse proxy settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  listen: 443
  server_name: "neglected-honesty.org"
  upstream: "http://localhost:7879"

  # TLS settings
  ssl: true
  http2: true

  gzip: true
  gzip_min_length: 256
  client_max_body: "16mb"
}
```
