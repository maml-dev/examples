# 2560 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "admin"
      permissions: [
        "manage_billing"
        "manage_users"
      ]
      max_seats: 37
    }

    {
      name: "support"
      permissions: [
        "manage_users"
        "read"
      ]
      max_seats: null
    }

    {
      name: "support"
      permissions: [
        "write"
        "export_data"
        "manage_users"
        "manage_billing"
      ]
      max_seats: 46
    }

    {
      name: "viewer"
      permissions: [
        "manage_billing"
        "view_analytics"
        "manage_users"
      ]
      max_seats: null
    }
  ]
}
```
