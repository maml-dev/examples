# 158 - Font Stack

Typography font family definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  fonts: [
    {
      family: "Berkeley Mono"
      category: "serif"
      weights: [
        100
        200
        600
        700
        900
      ]
      variable: true
    }

    {
      family: "Roboto"
      category: "monospace"
      weights: [
        200
        300
        400
        500
        900
      ]
      variable: false
    }

    {
      family: "Source Sans Pro"
      category: "serif"
      weights: [
        100
        400
      ]
      variable: false
    }
  ]
}
```
