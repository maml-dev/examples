# 3301 - Locale Config

```maml
  language: "en"
  country: "NG"
  timezone: "Indian/Chagos"
  currency: "USD"

  date_format: "MM/DD/YYYY"
  time_format: "12h"
  first_day_of_week: "monday"
  decimal_separator: ","
}
```
