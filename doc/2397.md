# 2397 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@unimportant-makeover.net"
  subject: "Talio adfectus tactus ademptio."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Comminor vinitor curtus adopto tredecim calco summa. Commodi aer amet solvo conservo somnus verus. Tamen vomito tametsi amitto strenuus ullus tonsor.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Virga uxor curiositas thymum. Validus amplexus turpis. Earum unus surculus tempus.

Visit: {{url}}
"""
}
```
