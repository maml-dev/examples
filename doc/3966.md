# 3966 - Dashboard Layout

Dashboard widget layout configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  title: "Bergstrom - Doyle Dashboard"
  refresh_interval: 60 # seconds
  widgets: [
    {
      type: "chart"
      title: "Traffic"
      source: "/api/metrics/revenue"
      chart_type: "bar"
      period: "90d"
    }

    {
      type: "stat"
      title: "MRR"
      source: "/api/stats/users"
      format: "currency"
    }

    {
      type: "table"
      title: "Recent Orders"
      source: "/api/recent/signups"
      limit: 22
    }
  ]

  theme: "light"
}
```
