# 152 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "hot-contrail.info"
  records: [
    {
      type: "TXT"
      name: "staging"
      value: "75.148.28.160"
      ttl: 3600
    }

    {
      type: "TXT"
      name: "@"
      value: "foolish-decryption.biz"
      ttl: 86400
    }

    {
      type: "TXT"
      name: "api"
      value: "waterlogged-roadway.net"
      ttl: 86400
    }

    {
      type: "CNAME"
      name: "www"
      value: "sniveling-management.net"
      ttl: 300
    }
  ]
}
```
