# 1635 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "liquid-fishery.name"
  inbound: {
    mx_records: [
      "mx1.liquid-fishery.name"
      "mx2.liquid-fishery.name"
    ]
    spam_filter: {
      enabled: true
      threshold: 7
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.liquid-fishery.name"
    port: 25
    tls: true
    rate_limit: {
      per_minute: 456
      per_hour: 8184
    }
  }
}
```
