# 4311 - Logger Config

Structured logging configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  level: "fatal"
  format: "pretty"
  timestamp: true
  caller: false # include file:line

  output: "stdout"
  rotation: true
  max_size_mb: 293
  max_age_days: 18
  compress: true
}
```
