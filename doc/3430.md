# 3430 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "models"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 10 # seconds
    labels: {
      env: "prod"
      region: "eu-west"
      team: "health"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 77
    memory_threshold: 75
    latency_p99_ms: 1133
    error_rate: 0.05
    notification: {
      channel: "#incidents"
      severity: "info"
    }
  }
}
```
