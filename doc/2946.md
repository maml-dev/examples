# 2946 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "sublime"
  shortcuts: [
    {
      key: "Cmd+Shift+L"
      action: "toggle_comment"
      when: "editor"
    }

    {
      key: "Cmd+Shift+L"
      action: "format"
      when: "debug"
    }

    {
      key: "Cmd+Shift+L"
      action: "find_references"
      when: "terminal"
    }

    {
      key: "Ctrl+D"
      action: "search_project"
      when: "global"
    }
  ]
}
```
