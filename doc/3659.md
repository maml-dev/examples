# 3659 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "moderator"
      permissions: [
        "export_data"
        "view_analytics"
        "manage_billing"
      ]
      max_seats: 39
    }

    {
      name: "viewer"
      permissions: [
        "delete"
        "read"
        "view_analytics"
        "manage_billing"
        "write"
      ]
      max_seats: null
    }

    {
      name: "editor"
      permissions: [
        "manage_users"
        "write"
        "view_analytics"
        "manage_billing"
        "read"
      ]
      max_seats: 5
    }
  ]
}
```
