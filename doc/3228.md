# 3228 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "kafka"
  connection: {
    host: "localhost"
    port: 9092
    vhost: "/production"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 4
      retry: 2
      timeout: 55508 # ms
    }

    priority: {
      concurrency: 7
      retry: 10
      timeout: 1785 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 9
    queue: "dead-letter"
  }
}
```
