# 3324 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.tired-hutch.info"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "1m"
    max_requests: 300
    by: "api_key"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://hurtful-requirement.com"
      "https://app.familiar-hamburger.info"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 86400
  }

  cache: {
    enabled: false
    ttl: 2267
    store: "memory"
  }
}
```
