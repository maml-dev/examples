# 131 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.back-wasabi.org"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "1m"
    max_requests: 4621
    by: "user"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://aggravating-design.net"
      "https://app.nutritious-season.com"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 86400
  }

  cache: {
    enabled: false
    ttl: 2585
    store: "memory"
  }
}
```
