# 3122 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "adolescent-remark.biz"
  inbound: {
    mx_records: [
      "mx1.adolescent-remark.biz"
      "mx2.adolescent-remark.biz"
    ]
    spam_filter: {
      enabled: true
      threshold: 7
      action: "quarantine"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.adolescent-remark.biz"
    port: 465
    tls: true
    rate_limit: {
      per_minute: 259
      per_hour: 8984
    }
  }
}
```
