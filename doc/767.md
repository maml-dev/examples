# 767 - Dashboard Layout

Dashboard widget layout configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  title: "Witting - Volkman Dashboard"
  refresh_interval: 60 # seconds
  widgets: [
    {
      type: "chart"
      title: "Signups"
      source: "/api/metrics/revenue"
      chart_type: "line"
      period: "7d"
    }

    {
      type: "stat"
      title: "Total Users"
      source: "/api/stats/revenue"
      format: "percent"
    }

    {
      type: "table"
      title: "Top Pages"
      source: "/api/recent/orders"
      limit: 12
    }
  ]

  theme: "light"
}
```
