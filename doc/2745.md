# 2745 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v2"
  routes: [
    {
      method: "PATCH"
      path: "/posts/:id"
      handler: "listPosts"
      auth: false
    }

    {
      method: "DELETE"
      path: "/products/:id"
      handler: "createUsers"
      auth: true
    }

    {
      method: "PUT"
      path: "/users/:id"
      handler: "getProducts"
      auth: true
    }
  ]
}
```
