# 2636 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.messy-affiliate.info"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "1m"
    max_requests: 6725
    by: "api_key"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://soupy-onset.name"
      "https://app.darling-millet.info"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 3600
  }

  cache: {
    enabled: false
    ttl: 2535
    store: "redis"
  }
}
```
