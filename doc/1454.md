# 1454 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "glorious-fisherman.org"
  records: [
    {
      type: "CNAME"
      name: "api"
      value: "v=spf1 include:_spf.accurate-perp.org ~all"
      ttl: 3600
    }

    {
      type: "CNAME"
      name: "www"
      value: "223.13.111.174"
      ttl: 3600
    }

    {
      type: "AAAA"
      name: "@"
      value: "127.192.176.123"
      ttl: 3600
    }

    {
      type: "AAAA"
      name: "www"
      value: "89.104.186.91"
      ttl: 3600
    }
  ]
}
```
