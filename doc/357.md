# 357 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "tall-unique.com"
  records: [
    {
      type: "NS"
      name: "mail"
      value: "85.78.1.186"
      ttl: 86400
    }

    {
      type: "A"
      name: "@"
      value: "left-derby.name"
      ttl: 86400
    }

    {
      type: "NS"
      name: "www"
      value: "v=spf1 include:_spf.tasty-typeface.net ~all"
      ttl: 86400
    }

    {
      type: "MX"
      name: "@"
      value: "v=spf1 include:_spf.well-worn-blossom.name ~all"
      ttl: 86400
    }
  ]
}
```
