# 2951 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "sublime"
  shortcuts: [
    {
      key: "Ctrl+Shift+F"
      action: "save"
      when: "debug"
    }

    {
      key: "Ctrl+`"
      action: "save"
      when: "terminal"
    }

    {
      key: "Ctrl+Shift+F"
      action: "save"
      when: "editor"
    }

    {
      key: "Ctrl+/"
      action: "save"
      when: "terminal"
    }

    {
      key: "F5"
      action: "find_references"
      when: "global"
    }

    {
      key: "Ctrl+D"
      action: "command_palette"
      when: "debug"
    }

    {
      key: "Ctrl+Shift+P"
      action: "toggle_terminal"
      when: "debug"
    }
  ]
}
```
