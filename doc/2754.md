# 2754 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v3"
  routes: [
    {
      method: "PATCH"
      path: "/orders/:id"
      handler: "deleteOrders"
      auth: true
    }

    {
      method: "PUT"
      path: "/orders/:id"
      handler: "deleteUsers"
      auth: false
    }

    {
      method: "PUT"
      path: "/products"
      handler: "updatePosts"
      auth: false
    }

    {
      method: "DELETE"
      path: "/products/:id"
      handler: "updateOrders"
      auth: true
    }

    {
      method: "PUT"
      path: "/posts/:id"
      handler: "deletePosts"
      auth: true
    }

    {
      method: "DELETE"
      path: "/orders/:id"
      handler: "deleteProducts"
      auth: true
    }
  ]
}
```
