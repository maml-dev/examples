# 3957 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "grave-calculus.net"
  records: [
    {
      type: "NS"
      name: "cdn"
      value: "23.232.159.243"
      ttl: 300
    }

    {
      type: "TXT"
      name: "api"
      value: "19.114.145.180"
      ttl: 3600
    }

    {
      type: "NS"
      name: "mail"
      value: "53.184.157.55"
      ttl: 86400
    }

    {
      type: "TXT"
      name: "@"
      value: "254.57.110.251"
      ttl: 300
    }

    {
      type: "TXT"
      name: "api"
      value: "v=spf1 include:_spf.plain-video.info ~all"
      ttl: 86400
    }
  ]
}
```
