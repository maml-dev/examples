# 2134 - Theme Config

Design system theme configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  name: "mild Theme"
  # Color palette
  colors: {
    primary: "#778b3e"
    secondary: "#c28c0b"
    accent: "#aba058"
    background: "#0d1117"
    text: "#333333"

    error: "#dc3545"
    warning: "#ffc107"
    success: "#28a745"
  }

  # Typography
  typography: {
    font_family: "Inter, sans-serif"
    base_size: 14
    line_height: 1.5
    headings: {
      font_family: "\"Space Grotesk\", sans-serif"
      weight: 800
    }
  }

  spacing: {
    unit: 4
    scale: [
      0
      4
      8
      16
      24
      32
      48
    ]
  }
  border_radius: 4
}
```
