# 2252 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v3"
  routes: [
    {
      method: "GET"
      path: "/comments/:id"
      handler: "updateProducts"
      auth: true
    }

    {
      method: "PATCH"
      path: "/users/:id"
      handler: "getProducts"
      auth: true
    }

    {
      method: "PATCH"
      path: "/users"
      handler: "deletePosts"
      auth: true
    }

    {
      method: "PUT"
      path: "/posts/:id"
      handler: "getUsers"
      auth: true
    }

    {
      method: "DELETE"
      path: "/comments"
      handler: "deleteUsers"
      auth: false
    }
  ]
}
```
