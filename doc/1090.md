# 1090 - Quoted Keys

Configuration using special characters in keys. This is an example of a [MAML](https://maml.dev) document.

```maml
# HTTP Headers
{
  Content-Type: "text/html"
  X-Request-ID: "91db1425-9cd5-4247-bd09-6264aabe3073"
  Accept-Language: "en-US"
  Cache-Control: "no-cache"
  X-Powered-By: "Hono"

  # Keys with spaces get quoted
  "headers with spaces": {
    "custom header": "facere"
    "another one": "earum"
  }
}
```
