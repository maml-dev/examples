# 2947 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v1"
  routes: [
    {
      method: "POST"
      path: "/posts/:id"
      handler: "getUsers"
      auth: true
    }

    {
      method: "GET"
      path: "/users/:id"
      handler: "updateProducts"
      auth: false
    }

    {
      method: "DELETE"
      path: "/comments"
      handler: "createPosts"
      auth: false
    }

    {
      method: "POST"
      path: "/comments"
      handler: "deleteOrders"
      auth: false
    }

    {
      method: "GET"
      path: "/posts"
      handler: "createOrders"
      auth: false
    }
  ]
}
```
