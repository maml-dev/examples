# 2242 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v3"
  routes: [
    {
      method: "PATCH"
      path: "/comments/:id"
      handler: "listProducts"
      auth: false
    }

    {
      method: "POST"
      path: "/orders/:id"
      handler: "deletePosts"
      auth: false
    }

    {
      method: "GET"
      path: "/comments/:id"
      handler: "listOrders"
      auth: true
    }
  ]
}
```
