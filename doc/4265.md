# 4265 - Dashboard Layout

Dashboard widget layout configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  title: "Balistreri - Kertzmann Dashboard"
  refresh_interval: 30 # seconds
  widgets: [
    {
      type: "chart"
      title: "Active Users"
      source: "/api/metrics/revenue"
      chart_type: "line"
      period: "90d"
    }

    {
      type: "stat"
      title: "Error Rate"
      source: "/api/stats/revenue"
      format: "currency"
    }

    {
      type: "table"
      title: "Latest Signups"
      source: "/api/recent/signups"
      limit: 12
    }
  ]

  theme: "auto"
}
```
