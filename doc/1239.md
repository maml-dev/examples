# 1239 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.austere-testimonial.info"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "15m"
    max_requests: 5523
    by: "ip"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://damp-schnitzel.name"
      "https://app.cumbersome-awareness.net"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 86400
  }

  cache: {
    enabled: true
    ttl: 3382
    store: "redis"
  }
}
```
