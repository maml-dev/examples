# 4238 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "schemas"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 60 # seconds
    labels: {
      env: "staging"
      region: "eu-west"
      team: "grocery"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 92
    memory_threshold: 89
    latency_p99_ms: 1145
    error_rate: 0.02
    notification: {
      channel: "#incidents"
      severity: "info"
    }
  }
}
```
