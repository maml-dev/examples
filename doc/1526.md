# 1526 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "unknown-resource.net"
  inbound: {
    mx_records: [
      "mx1.unknown-resource.net"
      "mx2.unknown-resource.net"
    ]
    spam_filter: {
      enabled: true
      threshold: 7
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.unknown-resource.net"
    port: 25
    tls: true
    rate_limit: {
      per_minute: 91
      per_hour: 1465
    }
  }
}
```
