# 3647 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "viewer"
      permissions: [
        "export_data"
        "manage_users"
      ]
      max_seats: 8
    }

    {
      name: "support"
      permissions: [
        "export_data"
        "manage_users"
      ]
      max_seats: null
    }

    {
      name: "editor"
      permissions: [
        "read"
        "manage_users"
      ]
      max_seats: null
    }
  ]
}
```
