# 3738 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.warlike-honesty.info"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "1m"
    max_requests: 370
    by: "api_key"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://competent-developing.org"
      "https://app.blond-supplier.org"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 3600
  }

  cache: {
    enabled: true
    ttl: 267
    store: "redis"
  }
}
```
