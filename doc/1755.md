# 1755 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "support"
      permissions: [
        "manage_users"
        "read"
        "write"
        "delete"
        "view_analytics"
      ]
      max_seats: 3
    }

    {
      name: "admin"
      permissions: [
        "view_analytics"
        "export_data"
        "manage_users"
      ]
      max_seats: 41
    }

    {
      name: "support"
      permissions: [
        "delete"
        "read"
      ]
      max_seats: 11
    }

    {
      name: "moderator"
      permissions: [
        "write"
        "delete"
      ]
      max_seats: 33
    }
  ]
}
```
