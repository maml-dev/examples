# 3635 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "experiences"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 60 # seconds
    labels: {
      env: "staging"
      region: "eu-west"
      team: "garden"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 84
    memory_threshold: 75
    latency_p99_ms: 1642
    error_rate: 0.05
    notification: {
      channel: "#incidents"
      severity: "critical"
    }
  }
}
```
