# 833 - Terraform Resource

Infrastructure as code resource definition. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  provider: "google"
  region: "asia-southeast1"
  resources: {
    # Network
    vpc: {
      cidr_block: "10.52.0.0/16"
      enable_dns: true
      tags: {
        Name: "haversack-vpc"
        Environment: "staging"
      }
    }

    # Compute
    instance: {
      ami: "ami-c1bbeddb"
      type: "t3.medium"
      key_name: "drive-key"
      monitoring: true
      root_volume: {
        size: 100
        type: "gp3"
        encrypted: true
      }
    }

    # Security
    security_group: {
      name: "allow-ssh"
      ingress_port: 8080
      ingress_cidr: "0.0.0.0/0"
    }
  }
}
```
