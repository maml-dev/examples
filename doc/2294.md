# 2294 - Quoted Keys

Configuration using special characters in keys. This is an example of a [MAML](https://maml.dev) document.

```maml
# HTTP Headers
{
  Content-Type: "text/html"
  X-Request-ID: "8bbd0874-55f3-4199-96b4-847f72d730d6"
  Accept-Language: "en-US"
  Cache-Control: "max-age=3600"
  X-Powered-By: "Express"

  # Keys with spaces get quoted
  "headers with spaces": {
    "custom header": "vos"
    "another one": "dedecor"
  }
}
```
