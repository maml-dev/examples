# 426 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "rabbitmq"
  connection: {
    host: "mq.internal"
    port: 6379
    vhost: "/"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 5
      retry: 5
      timeout: 44931 # ms
    }

    priority: {
      concurrency: 5
      retry: 4
      timeout: 9388 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 5
    queue: "dead-letter"
  }
}
```
