# 1627 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "foolish-director.info"
  inbound: {
    mx_records: [
      "mx1.foolish-director.info"
      "mx2.foolish-director.info"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.foolish-director.info"
    port: 465
    tls: true
    rate_limit: {
      per_minute: 392
      per_hour: 2077
    }
  }
}
```
