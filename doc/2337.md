# 2337 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "velocity-api"
    namespace: "staging"
    labels: {
      app: "cow"
      tier: "backend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 5
    container: {
      image: "hunger:3.10.14"
      port: 3000
      resources: {
        cpu_limit: "250m"
        memory_limit: "512Mi"
      }
    }

    health_check: {
      path: "/ready"
      interval: 30
      timeout: 10
    }
  }
}
```
