# 4029 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.knowledgeable-marathon.net"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "15m"
    max_requests: 2339
    by: "api_key"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://smug-commodity.info"
      "https://app.remorseful-edge.name"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 86400
  }

  cache: {
    enabled: true
    ttl: 303
    store: "redis"
  }
}
```
