# 300 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@harmful-unit.net"
  subject: "Volaticus credo uredo."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Expedita quae antea vicinus comprehendo trepide tot. Copia aequitas defaeco cibus usus contego natus bardus defendo. Praesentium ocer nulla tabesco arbustum cavus canis custodia peccatus incidunt.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Sed umquam custodia corporis vox benevolentia vinitor cornu. Socius claudeo vesper ullus totus titulus tot clementia. Sollers ullus pauci caelestis vulgus bellum ago subseco.

Visit: {{url}}
"""
}
```
