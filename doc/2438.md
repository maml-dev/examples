# 2438 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "amused-reservation.net"
  inbound: {
    mx_records: [
      "mx1.amused-reservation.net"
      "mx2.amused-reservation.net"
    ]
    spam_filter: {
      enabled: true
      threshold: 5
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.amused-reservation.net"
    port: 25
    tls: true
    rate_limit: {
      per_minute: 273
      per_hour: 2481
    }
  }
}
```
