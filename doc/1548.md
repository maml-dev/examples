# 1548 - Cron Jobs

Scheduled task definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  timezone: "UTC"
  jobs: [
    {
      name: "send-reports"
      schedule: "*/15 * * * *"
      command: "node scripts/cleanup.js"
      enabled: true
    }

    {
      name: "rotate-logs"
      schedule: "0 0 * * *"
      command: "node scripts/cleanup.js"
      enabled: false
    }

    {
      name: "rotate-logs"
      schedule: "0 * * * *"
      command: "node scripts/cleanup.js"
      enabled: true
    }

    {
      name: "cleanup-temp"
      schedule: "30 6 * * 1-5"
      command: "node scripts/cleanup.js"
      enabled: true
    }

    {
      name: "cleanup-temp"
      schedule: "30 6 * * 1-5"
      command: "node scripts/cleanup.js"
      enabled: false
    }

    {
      name: "sync-data"
      schedule: "*/15 * * * *"
      command: "python manage.py report"
      enabled: true
    }
  ]
}
```
