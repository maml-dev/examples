# 1193 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@assured-provider.net"
  subject: "Via peccatus articulus nesciunt cursus custodia."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Temptatio suscipit vesica. Amplitudo comparo cursim numquam advoco possimus. Temeritas vis bos consectetur volaticus statua amoveo coniuratio tenetur cauda.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Tergeo vociferor tabernus dedecor solitudo desidero aeternus vomer speciosus arto. Adulatio tripudio nobis copiose vulnero accendo uxor numquam repellendus ipsum. Vorago titulus sint adsidue decipio adsum nobis cohaero.

Visit: {{url}}
"""
}
```
