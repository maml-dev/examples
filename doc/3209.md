# 3209 - Email Config

SMTP email service settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  smtp_host: "smtp.whopping-catalyst.com"
  smtp_port: 465
  use_tls: true
  from_name: "Roberts and Sons"
  from_email: "noreply@whopping-catalyst.com"

  # Rate limiting
  max_batch_size: 74
  retry_interval: 123
}
```
