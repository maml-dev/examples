# 1035 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "improbable-reboot.org"
  inbound: {
    mx_records: [
      "mx1.improbable-reboot.org"
      "mx2.improbable-reboot.org"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "quarantine"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.improbable-reboot.org"
    port: 587
    tls: true
    rate_limit: {
      per_minute: 151
      per_hour: 7700
    }
  }
}
```
