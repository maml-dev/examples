# 2822 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "distinction-web"
    namespace: "production"
    labels: {
      app: "deduction"
      tier: "backend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 4
    container: {
      image: "perp:9.15.20"
      port: 9090
      resources: {
        cpu_limit: "250m"
        memory_limit: "256Mi"
      }
    }

    health_check: {
      path: "/health"
      interval: 15
      timeout: 10
    }
  }
}
```
