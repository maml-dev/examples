# 2189 - Quoted Keys

Configuration using special characters in keys. This is an example of a [MAML](https://maml.dev) document.

```maml
# HTTP Headers
{
  Content-Type: "text/html"
  X-Request-ID: "548f95a6-0edd-4bd4-ae77-65a143832def"
  Accept-Language: "fr-FR"
  Cache-Control: "public, max-age=86400"
  X-Powered-By: "Express"

  # Keys with spaces get quoted
  "headers with spaces": {
    "custom header": "admiratio"
    "another one": "aggredior"
  }
}
```
