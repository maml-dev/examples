# 1745 - Cron Jobs

Scheduled task definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  timezone: "UTC"
  jobs: [
    {
      name: "backup-db"
      schedule: "0 2 * * 0"
      command: "./bin/sync --all"
      enabled: false
    }

    {
      name: "refresh-cache"
      schedule: "0 2 * * 0"
      command: "node scripts/cleanup.js"
      enabled: true
    }

    {
      name: "sync-data"
      schedule: "0 * * * *"
      command: "./bin/sync --all"
      enabled: true
    }
  ]
}
```
