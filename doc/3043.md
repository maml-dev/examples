# 3043 - Cron Jobs

Scheduled task definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  timezone: "America/New_York"
  jobs: [
    {
      name: "rotate-logs"
      schedule: "0 2 * * 0"
      command: "./bin/sync --all"
      enabled: true
    }

    {
      name: "backup-db"
      schedule: "*/15 * * * *"
      command: "node scripts/cleanup.js"
      enabled: false
    }

    {
      name: "sync-data"
      schedule: "0 0 * * *"
      command: "./bin/sync --all"
      enabled: true
    }

    {
      name: "backup-db"
      schedule: "0 2 * * 0"
      command: "python manage.py report"
      enabled: true
    }
  ]
}
```
