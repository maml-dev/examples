# 2056 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "viewer"
      permissions: [
        "manage_users"
        "view_analytics"
      ]
      max_seats: 10
    }

    {
      name: "billing"
      permissions: [
        "write"
        "export_data"
      ]
      max_seats: null
    }

    {
      name: "moderator"
      permissions: [
        "delete"
        "write"
        "manage_users"
        "view_analytics"
        "export_data"
      ]
      max_seats: 10
    }

    {
      name: "support"
      permissions: [
        "manage_billing"
        "view_analytics"
        "manage_users"
        "write"
      ]
      max_seats: 41
    }

    {
      name: "viewer"
      permissions: [
        "view_analytics"
        "write"
      ]
      max_seats: null
    }
  ]
}
```
