# 2695 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@dim-hydrolyze.info"
  subject: "Caterva volo cogito repellat adsum."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Apto in deinde sublime beatae theca totus. Tui tristis dedecor delego abduco vae. Somniculosus aestus vindico caritas solum vehemens non delectatio.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Auctor tergiversatio peccatus volo aliquid cognatus. Terra umquam amplitudo. Cicuta artificiose non pectus debitis.

Visit: {{url}}
"""
}
```
