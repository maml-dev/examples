# 3021 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "voluminous-haversack.org"
  inbound: {
    mx_records: [
      "mx1.voluminous-haversack.org"
      "mx2.voluminous-haversack.org"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.voluminous-haversack.org"
    port: 587
    tls: true
    rate_limit: {
      per_minute: 360
      per_hour: 1364
    }
  }
}
```
