# 1993 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@rubbery-tinderbox.biz"
  subject: "Spectaculum alii animi adopto virtus aestivus."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Nesciunt illo atque audentia vomer vicinus. Omnis dedecor demitto audio arcesso. Demum ante commodo.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Temperantia verbera amplus cornu angelus tenax accedo officia. Corpus solus velit votum ex arbor averto. Temptatio audentia vomica.

Visit: {{url}}
"""
}
```
