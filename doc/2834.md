# 2834 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "thick-following.info"
  inbound: {
    mx_records: [
      "mx1.thick-following.info"
      "mx2.thick-following.info"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "quarantine"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.thick-following.info"
    port: 25
    tls: true
    rate_limit: {
      per_minute: 143
      per_hour: 1566
    }
  }
}
```
