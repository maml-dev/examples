# 2052 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "emacs"
  shortcuts: [
    {
      key: "Alt+Up"
      action: "toggle_terminal"
      when: "global"
    }

    {
      key: "Cmd+Shift+L"
      action: "format"
      when: "debug"
    }

    {
      key: "Cmd+Shift+L"
      action: "debug_run"
      when: "editor"
    }

    {
      key: "Alt+F7"
      action: "toggle_sidebar"
      when: "debug"
    }
  ]
}
```
