# 1418 - Logger Config

Structured logging configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  level: "error"
  format: "json"
  timestamp: true
  caller: true # include file:line

  output: "stdout"
  rotation: false
  max_size_mb: 54
  max_age_days: 20
  compress: true
}
```
