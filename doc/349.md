# 349 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "billing"
      permissions: [
        "manage_billing"
        "export_data"
        "delete"
      ]
      max_seats: null
    }

    {
      name: "billing"
      permissions: [
        "view_analytics"
        "export_data"
        "delete"
      ]
      max_seats: 23
    }

    {
      name: "viewer"
      permissions: [
        "view_analytics"
        "export_data"
        "manage_billing"
        "read"
        "delete"
      ]
      max_seats: 7
    }

    {
      name: "support"
      permissions: [
        "manage_users"
        "delete"
        "view_analytics"
        "read"
        "export_data"
      ]
      max_seats: 40
    }
  ]
}
```
