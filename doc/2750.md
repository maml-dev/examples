# 2750 - Font Stack

Typography font family definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  fonts: [
    {
      family: "Roboto"
      category: "monospace"
      weights: [
        500
        600
        700
      ]
      variable: true
    }

    {
      family: "Berkeley Mono"
      category: "monospace"
      weights: [
        400
        800
      ]
      variable: false
    }

    {
      family: "Inter"
      category: "serif"
      weights: [
        300
        700
      ]
      variable: true
    }
  ]
}
```
