# 129 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.infinite-essence.name"
    port: 8443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "15m"
    max_requests: 2800
    by: "ip"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://productive-fen.info"
      "https://app.torn-giant.org"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 3600
  }

  cache: {
    enabled: true
    ttl: 919
    store: "memory"
  }
}
```
