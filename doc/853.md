# 853 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v3"
  routes: [
    {
      method: "POST"
      path: "/posts/:id"
      handler: "getPosts"
      auth: true
    }

    {
      method: "PATCH"
      path: "/posts/:id"
      handler: "deleteOrders"
      auth: true
    }

    {
      method: "PUT"
      path: "/orders/:id"
      handler: "updateProducts"
      auth: false
    }

    {
      method: "PUT"
      path: "/products"
      handler: "updateProducts"
      auth: true
    }

    {
      method: "GET"
      path: "/users"
      handler: "getProducts"
      auth: false
    }

    {
      method: "PUT"
      path: "/comments/:id"
      handler: "createProducts"
      auth: false
    }
  ]
}
```
