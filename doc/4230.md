# 4230 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.self-assured-technician.name"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "5m"
    max_requests: 8555
    by: "api_key"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://unconscious-soliloquy.biz"
      "https://app.emotional-editor.net"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 7200
  }

  cache: {
    enabled: true
    ttl: 2292
    store: "memory"
  }
}
```
