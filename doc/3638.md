# 3638 - Backup Strategy

Backup schedule and retention policy. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  source: {
    type: "filesystem"
    host: "db.internal"
    databases: [
      "cake"
      "passport"
    ]
  }

  # Backup schedule
  schedule: {
    full: "0 2 * * 0" # weekly Sunday 2am
    incremental: "0 2 * * 1-6" # daily except Sunday
    snapshot: "0 */4 * * *" # every 4 hours
  }

  retention: {
    daily: 13
    weekly: 6
    monthly: 11
  }

  destination: {
    type: "s3"
    path: "s3://backups-galoshes"
    encryption: true
    compression: "zstd"
  }
}
```
