# 3645 - Cron Jobs

Scheduled task definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  timezone: "UTC"
  jobs: [
    {
      name: "send-reports"
      schedule: "0 * * * *"
      command: "./bin/sync --all"
      enabled: false
    }

    {
      name: "send-reports"
      schedule: "0 * * * *"
      command: "pg_dump $DATABASE_URL"
      enabled: true
    }

    {
      name: "send-reports"
      schedule: "0 * * * *"
      command: "python manage.py report"
      enabled: false
    }

    {
      name: "backup-db"
      schedule: "*/15 * * * *"
      command: "./bin/sync --all"
      enabled: false
    }

    {
      name: "refresh-cache"
      schedule: "0 0 * * *"
      command: "python manage.py report"
      enabled: false
    }

    {
      name: "refresh-cache"
      schedule: "0 * * * *"
      command: "python manage.py report"
      enabled: true
    }
  ]
}
```
