# 1421 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "austere-summary.name"
  inbound: {
    mx_records: [
      "mx1.austere-summary.name"
      "mx2.austere-summary.name"
    ]
    spam_filter: {
      enabled: true
      threshold: 7
      action: "reject"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.austere-summary.name"
    port: 465
    tls: true
    rate_limit: {
      per_minute: 104
      per_hour: 6260
    }
  }
}
```
