# 3507 - Proxy Config

Reverse proxy settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  listen: 8080
  server_name: "funny-quart.org"
  upstream: "http://127.0.0.1:5820"

  # TLS settings
  ssl: true
  http2: true

  gzip: true
  gzip_min_length: 1024
  client_max_body: "20mb"
}
```
