# 1952 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "fantastic-operating.name"
  records: [
    {
      type: "TXT"
      name: "cdn"
      value: "v=spf1 include:_spf.velvety-crest.info ~all"
      ttl: 3600
    }

    {
      type: "AAAA"
      name: "api"
      value: "amused-farm.biz"
      ttl: 86400
    }

    {
      type: "NS"
      name: "api"
      value: "184.36.114.105"
      ttl: 300
    }

    {
      type: "A"
      name: "mail"
      value: "silky-quinoa.info"
      ttl: 86400
    }
  ]
}
```
