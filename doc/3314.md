# 3314 - Logger Config

Structured logging configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  level: "trace"
  format: "logfmt"
  timestamp: true
  caller: false # include file:line

  output: "stderr"
  rotation: true
  max_size_mb: 361
  max_age_days: 19
  compress: true
}
```
