# 1836 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "plugin-web"
    namespace: "default"
    labels: {
      app: "leading"
      tier: "data"
    }
  }

  # Deployment spec
  spec: {
    replicas: 1
    container: {
      image: "toaster:6.6.17"
      port: 9090
      resources: {
        cpu_limit: "500m"
        memory_limit: "1Gi"
      }
    }

    health_check: {
      path: "/ready"
      interval: 30
      timeout: 5
    }
  }
}
```
