# 4336 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.icy-advancement.name"
    port: 8443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "5m"
    max_requests: 399
    by: "user"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://handy-railway.net"
      "https://app.multicolored-brief.com"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 3600
  }

  cache: {
    enabled: true
    ttl: 3017
    store: "memory"
  }
}
```
