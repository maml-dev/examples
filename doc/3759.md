# 3759 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v1"
  routes: [
    {
      method: "GET"
      path: "/posts"
      handler: "getUsers"
      auth: false
    }

    {
      method: "PATCH"
      path: "/users"
      handler: "createPosts"
      auth: false
    }

    {
      method: "PATCH"
      path: "/products/:id"
      handler: "deletePosts"
      auth: true
    }

    {
      method: "GET"
      path: "/posts"
      handler: "createPosts"
      auth: false
    }
  ]
}
```
