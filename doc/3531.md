# 3531 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "wry-markup.com"
  inbound: {
    mx_records: [
      "mx1.wry-markup.com"
      "mx2.wry-markup.com"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "reject"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.wry-markup.com"
    port: 25
    tls: true
    rate_limit: {
      per_minute: 494
      per_hour: 5212
    }
  }
}
```
