# 2506 - DNS Config

DNS zone configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "scaly-draft.name"
  ttl: 600 # seconds
  nameservers: [
    "ns1.frank-bog.net"
    "ns2.juvenile-obligation.net"
  ]

  soa_email: "admin@dismal-scenario.info"
  refresh: 3600
  retry: 600
  expire: 604800
}
```
