# 339 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "final-worker"
    namespace: "production"
    labels: {
      app: "rubric"
      tier: "backend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 3
    container: {
      image: "populist:5.8.11"
      port: 9090
      resources: {
        cpu_limit: "1000m"
        memory_limit: "256Mi"
      }
    }

    health_check: {
      path: "/health"
      interval: 15
      timeout: 5
    }
  }
}
```
