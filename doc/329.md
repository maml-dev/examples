# 329 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "synergy-api"
    namespace: "staging"
    labels: {
      app: "planula"
      tier: "backend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 4
    container: {
      image: "embossing:2.15.12"
      port: 9090
      resources: {
        cpu_limit: "500m"
        memory_limit: "1Gi"
      }
    }

    health_check: {
      path: "/ready"
      interval: 15
      timeout: 5
    }
  }
}
```
