# 671 - Game Config

Video game settings and preferences. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  display: {
    resolution: "1920x1080"
    fullscreen: true
    vsync: true
    fps_limit: 144
  }

  graphics: {
    quality: "ultra"
    shadows: "high"
    anti_aliasing: "FXAA"
    render_distance: 11
    ray_tracing: true
  }

  audio: {
    master: 79
    music: 80
    sfx: 58
    voice: 74
  }

  controls: {
    sensitivity: 2.6
    invert_y: false
    vibration: false
  }
}
```
