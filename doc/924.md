# 924 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "pure-schedule.net"
  inbound: {
    mx_records: [
      "mx1.pure-schedule.net"
      "mx2.pure-schedule.net"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.pure-schedule.net"
    port: 587
    tls: true
    rate_limit: {
      per_minute: 451
      per_hour: 6994
    }
  }
}
```
