# 213 - Locale Config

Regional and locale settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  language: "en"
  country: "LS"
  timezone: "Europe/Zagreb"
  currency: "JMD"

  date_format: "DD/MM/YYYY"
  time_format: "12h"
  first_day_of_week: "monday"
  decimal_separator: ","
}
```
