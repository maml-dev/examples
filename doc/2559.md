# 2559 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "editor"
      permissions: [
        "manage_users"
        "manage_billing"
        "view_analytics"
        "export_data"
        "delete"
      ]
      max_seats: 43
    }

    {
      name: "billing"
      permissions: [
        "view_analytics"
        "read"
        "manage_billing"
      ]
      max_seats: 27
    }

    {
      name: "billing"
      permissions: [
        "read"
        "manage_billing"
        "write"
        "manage_users"
      ]
      max_seats: 44
    }

    {
      name: "billing"
      permissions: [
        "manage_users"
        "manage_billing"
      ]
      max_seats: 22
    }
  ]
}
```
