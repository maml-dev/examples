# 2842 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "rich-guard.com"
  records: [
    {
      type: "A"
      name: "www"
      value: "lavish-disk.com"
      ttl: 300
    }

    {
      type: "A"
      name: "cdn"
      value: "mixed-relative.biz"
      ttl: 86400
    }

    {
      type: "NS"
      name: "cdn"
      value: "31.8.238.219"
      ttl: 86400
    }

    {
      type: "NS"
      name: "staging"
      value: "173.140.237.160"
      ttl: 86400
    }
  ]
}
```
