# 583 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@irresponsible-cruelty.com"
  subject: "Beneficium curtus deludo aestus vulgus cubitum."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Explicabo alii curriculum deduco traho corrumpo id adeptio. Sonitus accommodo deleniti comes. Sollers nostrum theatrum vox accusator tres altus.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Valde clamo optio depereo corrumpo abundans cursim. Accedo textus crustulum consequuntur solio cado optio virgo omnis. Tredecim dedecor harum patruus utrum.

Visit: {{url}}
"""
}
```
