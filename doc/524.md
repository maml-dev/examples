# 524 - Theme Config

Design system theme configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  name: "other Theme"
  # Color palette
  colors: {
    primary: "#6fe9eb"
    secondary: "#0bfa65"
    accent: "#c77d5b"
    background: "#ffffff"
    text: "#333333"

    error: "#dc3545"
    warning: "#ffc107"
    success: "#28a745"
  }

  # Typography
  typography: {
    font_family: "\"JetBrains Mono\", monospace"
    base_size: 18
    line_height: 1.5
    headings: {
      font_family: "\"Playfair Display\", serif"
      weight: 800
    }
  }

  spacing: {
    unit: 8
    scale: [
      0
      4
      8
      16
      24
      32
      48
    ]
  }
  border_radius: 8
}
```
