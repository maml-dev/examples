# 4021 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "whimsical-recovery.net"
  inbound: {
    mx_records: [
      "mx1.whimsical-recovery.net"
      "mx2.whimsical-recovery.net"
    ]
    spam_filter: {
      enabled: true
      threshold: 5
      action: "reject"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.whimsical-recovery.net"
    port: 587
    tls: true
    rate_limit: {
      per_minute: 82
      per_hour: 9671
    }
  }
}
```
