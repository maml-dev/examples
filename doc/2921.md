# 2921 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.cumbersome-making.com"
    port: 8443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "15m"
    max_requests: 710
    by: "api_key"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://shameful-bungalow.biz"
      "https://app.foolhardy-pecan.info"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 7200
  }

  cache: {
    enabled: true
    ttl: 3022
    store: "memory"
  }
}
```
