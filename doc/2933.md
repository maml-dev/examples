# 2933 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "metrics"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 10 # seconds
    labels: {
      env: "staging"
      region: "ap-south"
      team: "kids"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 82
    memory_threshold: 89
    latency_p99_ms: 190
    error_rate: 0.02
    notification: {
      channel: "#ops-alerts"
      severity: "critical"
    }
  }
}
```
