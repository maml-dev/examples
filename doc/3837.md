# 3837 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "siege-worker"
    namespace: "production"
    labels: {
      app: "gallery"
      tier: "frontend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 4
    container: {
      image: "e-mail:6.5.17"
      port: 8080
      resources: {
        cpu_limit: "1000m"
        memory_limit: "512Mi"
      }
    }

    health_check: {
      path: "/ready"
      interval: 30
      timeout: 3
    }
  }
}
```
