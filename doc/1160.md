# 1160 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "editor"
      permissions: [
        "manage_billing"
        "view_analytics"
        "write"
        "manage_users"
        "read"
      ]
      max_seats: null
    }

    {
      name: "viewer"
      permissions: [
        "read"
        "write"
        "view_analytics"
        "export_data"
      ]
      max_seats: 21
    }

    {
      name: "editor"
      permissions: [
        "view_analytics"
        "export_data"
        "manage_users"
        "read"
        "manage_billing"
      ]
      max_seats: 30
    }
  ]
}
```
