# 2715 - Proxy Config

Reverse proxy settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  listen: 80
  server_name: "fluffy-gloom.biz"
  upstream: "http://localhost:3385"

  # TLS settings
  ssl: false
  http2: true

  gzip: true
  gzip_min_length: 1024
  client_max_body: "43mb"
}
```
