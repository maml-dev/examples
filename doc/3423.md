# 3423 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.dependable-cornet.biz"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "15m"
    max_requests: 3199
    by: "api_key"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://jagged-exploration.org"
      "https://app.juicy-characterization.org"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 86400
  }

  cache: {
    enabled: false
    ttl: 1263
    store: "memory"
  }
}
```
