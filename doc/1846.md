# 1846 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "support"
      permissions: [
        "delete"
        "write"
      ]
      max_seats: null
    }

    {
      name: "billing"
      permissions: [
        "read"
        "view_analytics"
        "write"
        "delete"
        "manage_billing"
      ]
      max_seats: null
    }

    {
      name: "editor"
      permissions: [
        "write"
        "export_data"
        "view_analytics"
      ]
      max_seats: 30
    }
  ]
}
```
