# 2681 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@whispered-airmail.name"
  subject: "Pariatur autem vehemens molestiae."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Agnitio laborum tabesco sollicito sortitus custodia umquam ait vix catena. Accedo utor cogito defetiscor sublime denuncio. Capio contra capitulus corporis sumo tabesco uredo vero villa tandem.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Despecto est cauda admitto termes odio admoneo uredo curto speculum. Absens tener coruscus saepe subiungo quasi ter quam. Conculco dolor volva urbanus.

Visit: {{url}}
"""
}
```
