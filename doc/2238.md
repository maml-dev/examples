# 2238 - Backup Strategy

Backup schedule and retention policy. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  source: {
    type: "mysql"
    host: "db.internal"
    databases: [
      "decongestant"
      "leading"
    ]
  }

  # Backup schedule
  schedule: {
    full: "0 2 * * 0" # weekly Sunday 2am
    incremental: "0 2 * * 1-6" # daily except Sunday
    snapshot: "0 */4 * * *" # every 4 hours
  }

  retention: {
    daily: 11
    weekly: 8
    monthly: 8
  }

  destination: {
    type: "s3"
    path: "s3://backups-pension"
    encryption: true
    compression: "zstd"
  }
}
```
