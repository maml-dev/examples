# 3542 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "dim-resolve.info"
  records: [
    {
      type: "AAAA"
      name: "cdn"
      value: "abandoned-bookcase.biz"
      ttl: 3600
    }

    {
      type: "CNAME"
      name: "api"
      value: "v=spf1 include:_spf.serene-spring.info ~all"
      ttl: 300
    }

    {
      type: "NS"
      name: "*"
      value: "v=spf1 include:_spf.vivid-deck.name ~all"
      ttl: 86400
    }

    {
      type: "AAAA"
      name: "www"
      value: "57.177.112.83"
      ttl: 300
    }

    {
      type: "MX"
      name: "mail"
      value: "portly-milestone.com"
      ttl: 3600
    }
  ]
}
```
