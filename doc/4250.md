# 4250 - Environment Vars

Environment variable definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  app: "valentine"
  variables: [
    {
      name: "STRIPE_KEY"
      required: false
      secret: false
      default: "certus"
    }

    {
      name: "GITHUB_TOKEN"
      required: true
      secret: false
      default: null
    }

    {
      name: "API_KEY"
      required: false
      secret: true
      default: null
    }

    {
      name: "SENTRY_DSN"
      required: true
      secret: true
      default: "rem"
    }
  ]
}
```
