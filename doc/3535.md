# 3535 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "order-api"
    namespace: "default"
    labels: {
      app: "best-seller"
      tier: "frontend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 5
    container: {
      image: "scholarship:2.10.11"
      port: 8000
      resources: {
        cpu_limit: "250m"
        memory_limit: "512Mi"
      }
    }

    health_check: {
      path: "/ready"
      interval: 15
      timeout: 3
    }
  }
}
```
