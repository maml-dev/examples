# 4135 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.fatal-expansion.net"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "1m"
    max_requests: 333
    by: "ip"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://hungry-co-producer.org"
      "https://app.wasteful-devil.info"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 3600
  }

  cache: {
    enabled: true
    ttl: 1752
    store: "redis"
  }
}
```
