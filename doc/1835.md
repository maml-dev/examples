# 1835 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "shameful-governance.net"
  inbound: {
    mx_records: [
      "mx1.shameful-governance.net"
      "mx2.shameful-governance.net"
    ]
    spam_filter: {
      enabled: true
      threshold: 5
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.shameful-governance.net"
    port: 587
    tls: true
    rate_limit: {
      per_minute: 491
      per_hour: 5694
    }
  }
}
```
