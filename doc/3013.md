# 3013 - Logger Config

Structured logging configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  level: "fatal"
  format: "logfmt"
  timestamp: true
  caller: false # include file:line

  output: "stderr"
  rotation: false
  max_size_mb: 114
  max_age_days: 49
  compress: true
}
```
