# 4222 - CI Pipeline

Continuous integration pipeline definition. This is an example of a [MAML](https://maml.dev) document.

{
  name: "Main Pipeline"
  trigger: {
    branches: [

      "src/**"
      "tests/**"
    ]
  }

  stages: {
    # Code quality

    build: {

    }
  }
}
