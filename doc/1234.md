# 1234 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.french-cannon.biz"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "5m"
    max_requests: 5654
    by: "user"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://joyous-brace.org"
      "https://app.insidious-loyalty.name"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 3600
  }

  cache: {
    enabled: false
    ttl: 931
    store: "redis"
  }
}
```
