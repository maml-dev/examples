# 2398 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@thorough-millet.info"
  subject: "Uredo ceno derelinquo defetiscor crux demonstro."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Averto callide delego cupio itaque speculum voluptatem tot. Denique cado alias subvenio umbra blanditiis vel abstergo acquiro. Tempora turbo subiungo adipisci somnus spargo libero decet dolores.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Recusandae alioqui adsuesco soleo aliqua adopto. Valens audentia brevis beatus suppono. Dapifer capto cresco torqueo defaeco video arma coruscus torrens caveo.

Visit: {{url}}
"""
}
```
