# 1055 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "lively-longboat.net"
  records: [
    {
      type: "TXT"
      name: "mail"
      value: "55.90.170.84"
      ttl: 300
    }

    {
      type: "AAAA"
      name: "cdn"
      value: "v=spf1 include:_spf.accurate-sightseeing.biz ~all"
      ttl: 3600
    }

    {
      type: "CNAME"
      name: "www"
      value: "v=spf1 include:_spf.cavernous-recommendation.biz ~all"
      ttl: 3600
    }

    {
      type: "TXT"
      name: "mail"
      value: "228.194.64.113"
      ttl: 3600
    }

    {
      type: "A"
      name: "api"
      value: "3.12.172.143"
      ttl: 300
    }
  ]
}
```
