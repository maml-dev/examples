# 2742 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "tender-availability.biz"
  records: [
    {
      type: "TXT"
      name: "@"
      value: "14.77.126.97"
      ttl: 86400
    }

    {
      type: "AAAA"
      name: "staging"
      value: "impractical-asset.com"
      ttl: 86400
    }

    {
      type: "CNAME"
      name: "mail"
      value: "grimy-disconnection.com"
      ttl: 86400
    }

    {
      type: "MX"
      name: "api"
      value: "v=spf1 include:_spf.alarmed-king.org ~all"
      ttl: 3600
    }

    {
      type: "CNAME"
      name: "staging"
      value: "215.234.39.148"
      ttl: 86400
    }
  ]
}
```
