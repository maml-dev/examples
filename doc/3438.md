# 3438 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.deficient-possession.info"
    port: 8443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "15m"
    max_requests: 6173
    by: "api_key"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://dull-pearl.info"
      "https://app.mundane-flint.name"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 86400
  }

  cache: {
    enabled: false
    ttl: 434
    store: "memory"
  }
}
```
