# 3621 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.shameless-lava.net"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "15m"
    max_requests: 5857
    by: "ip"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://back-veto.biz"
      "https://app.clear-doorpost.org"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 7200
  }

  cache: {
    enabled: true
    ttl: 3529
    store: "memory"
  }
}
```
