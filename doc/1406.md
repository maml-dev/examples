# 1406 - Proxy Config

Reverse proxy settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  listen: 80
  server_name: "lonely-restaurant.info"
  upstream: "http://127.0.0.1:4941"

  # TLS settings
  ssl: true
  http2: true

  gzip: true
  gzip_min_length: 512
  client_max_body: "20mb"
}
```
