# 1243 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "admin"
      permissions: [
        "write"
        "manage_billing"
        "export_data"
      ]
      max_seats: null
    }

    {
      name: "admin"
      permissions: [
        "manage_users"
        "delete"
        "manage_billing"
      ]
      max_seats: 30
    }

    {
      name: "support"
      permissions: [
        "read"
        "manage_billing"
        "view_analytics"
      ]
      max_seats: null
    }

    {
      name: "viewer"
      permissions: [
        "write"
        "delete"
        "manage_billing"
      ]
      max_seats: 36
    }

    {
      name: "editor"
      permissions: [
        "write"
        "manage_billing"
        "read"
        "export_data"
      ]
      max_seats: null
    }
  ]
}
```
