# 4236 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "wrathful-derby.com"
  inbound: {
    mx_records: [
      "mx1.wrathful-derby.com"
      "mx2.wrathful-derby.com"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "quarantine"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.wrathful-derby.com"
    port: 465
    tls: true
    rate_limit: {
      per_minute: 234
      per_hour: 2327
    }
  }
}
```
