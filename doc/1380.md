# 1380 - Dashboard Layout

Dashboard widget layout configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  title: "Abshire, Towne and Reinger Dashboard"
  refresh_interval: 30 # seconds
  widgets: [
    {
      type: "chart"
      title: "Signups"
      source: "/api/metrics/revenue"
      chart_type: "line"
      period: "30d"
    }

    {
      type: "stat"
      title: "MRR"
      source: "/api/stats/revenue"
      format: "currency"
    }

    {
      type: "table"
      title: "Recent Orders"
      source: "/api/recent/orders"
      limit: 17
    }
  ]

  theme: "auto"
}
```
