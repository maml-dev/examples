# 1524 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "networks"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 10 # seconds
    labels: {
      env: "staging"
      region: "eu-west"
      team: "electronics"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 85
    memory_threshold: 84
    latency_p99_ms: 594
    error_rate: 0.01
    notification: {
      channel: "#oncall"
      severity: "warning"
    }
  }
}
```
