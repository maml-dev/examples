# 2308 - Proxy Config

Reverse proxy settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  listen: 443
  server_name: "descriptive-meander.biz"
  upstream: "http://127.0.0.1:8057"

  # TLS settings
  ssl: true
  http2: true

  gzip: true
  gzip_min_length: 512
  client_max_body: "12mb"
}
```
