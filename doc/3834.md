# 3834 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.mediocre-insolence.info"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "15m"
    max_requests: 6190
    by: "user"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://sizzling-parsnip.net"
      "https://app.familiar-vision.net"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 3600
  }

  cache: {
    enabled: true
    ttl: 2430
    store: "redis"
  }
}
```
