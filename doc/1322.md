# 1322 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "large language models"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 60 # seconds
    labels: {
      env: "prod"
      region: "us-east"
      team: "tools"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 93
    memory_threshold: 90
    latency_p99_ms: 1471
    error_rate: 0.05
    notification: {
      channel: "#ops-alerts"
      severity: "warning"
    }
  }
}
```
