# 632 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "sqs"
  connection: {
    host: "localhost"
    port: 9092
    vhost: "/staging"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 2
      retry: 1
      timeout: 24331 # ms
    }

    priority: {
      concurrency: 20
      retry: 9
      timeout: 2894 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 10
    queue: "dead-letter"
  }
}
```
