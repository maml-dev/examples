# 411 - Email Config

SMTP email service settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  smtp_host: "smtp.old-finer.com"
  smtp_port: 465
  use_tls: true
  from_name: "Terry, Jerde and Haley"
  from_email: "noreply@old-finer.com"

  # Rate limiting
  max_batch_size: 96
  retry_interval: 222
}
```
