# 3154 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "emacs"
  shortcuts: [
    {
      key: "Ctrl+Shift+F"
      action: "command_palette"
      when: "terminal"
    }

    {
      key: "Ctrl+/"
      action: "search_project"
      when: "debug"
    }

    {
      key: "F5"
      action: "save"
      when: "terminal"
    }

    {
      key: "Alt+Up"
      action: "toggle_sidebar"
      when: "editor"
    }

    {
      key: "Alt+F7"
      action: "format"
      when: "terminal"
    }
  ]
}
```
