# 1658 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v2"
  routes: [
    {
      method: "POST"
      path: "/orders/:id"
      handler: "createProducts"
      auth: false
    }

    {
      method: "PATCH"
      path: "/posts"
      handler: "updatePosts"
      auth: false
    }

    {
      method: "PATCH"
      path: "/posts"
      handler: "listProducts"
      auth: true
    }

    {
      method: "GET"
      path: "/comments/:id"
      handler: "listPosts"
      auth: true
    }
  ]
}
```
