# 3735 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "blockchains"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 30 # seconds
    labels: {
      env: "prod"
      region: "eu-west"
      team: "garden"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 84
    memory_threshold: 83
    latency_p99_ms: 1289
    error_rate: 0.05
    notification: {
      channel: "#oncall"
      severity: "warning"
    }
  }
}
```
