# 435 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "clueless-brush.org"
  inbound: {
    mx_records: [
      "mx1.clueless-brush.org"
      "mx2.clueless-brush.org"
    ]
    spam_filter: {
      enabled: true
      threshold: 5
      action: "reject"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.clueless-brush.org"
    port: 465
    tls: true
    rate_limit: {
      per_minute: 350
      per_hour: 8893
    }
  }
}
```
