# 120 - Linter Config

Code linter rule configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  parser: "typescript"
  env: "node"
  indent: 4
  semicolons: true
  quotes: "single"

  # Formatting
  max_line_length: 100
  trailing_comma: "all"
  no_unused_vars: true # error
  no_console: true # warn
}
```
