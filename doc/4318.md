# 4318 - Logger Config

Structured logging configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  level: "trace"
  format: "logfmt"
  timestamp: true
  caller: true # include file:line

  output: "stdout"
  rotation: false
  max_size_mb: 336
  max_age_days: 15
  compress: true
}
```
