# 038 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "platforms"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 30 # seconds
    labels: {
      env: "prod"
      region: "ap-south"
      team: "home"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 89
    memory_threshold: 79
    latency_p99_ms: 918
    error_rate: 0.05
    notification: {
      channel: "#oncall"
      severity: "info"
    }
  }
}
```
