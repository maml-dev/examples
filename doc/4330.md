# 4330 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "lifetime value"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 10 # seconds
    labels: {
      env: "prod"
      region: "eu-west"
      team: "computers"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 77
    memory_threshold: 94
    latency_p99_ms: 1444
    error_rate: 0.02
    notification: {
      channel: "#ops-alerts"
      severity: "critical"
    }
  }
}
```
