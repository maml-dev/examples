# 1190 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@profuse-hope.net"
  subject: "Quibusdam averto temperantia tamdiu spiritus error."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Spes nobis trepide laboriosam constans ter. Patruus asperiores decimus. Capto cursus timor.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Quisquam distinctio triumphus voveo. Communis appositus xiphias caput laudantium cavus. Delicate aequus thesaurus sortitus damnatio.

Visit: {{url}}
"""
}
```
