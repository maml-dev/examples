# 2279 - Dashboard Layout

Dashboard widget layout configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  title: "O'Connell, Kessler and Connelly Dashboard"
  refresh_interval: 300 # seconds
  widgets: [
    {
      type: "chart"
      title: "Traffic"
      source: "/api/metrics/revenue"
      chart_type: "line"
      period: "90d"
    }

    {
      type: "stat"
      title: "Uptime"
      source: "/api/stats/revenue"
      format: "number"
    }

    {
      type: "table"
      title: "Recent Orders"
      source: "/api/recent/signups"
      limit: 23
    }
  ]

  theme: "auto"
}
```
