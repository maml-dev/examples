# 2481 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@unfinished-reward.info"
  subject: "Cursus animi canto."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Coadunatio peccatus textilis color adamo suasoria copia. Stillicidium adinventitias valde ventus torqueo dolorem aestas vinculum. Speciosus celer comes coniuratio tabgo tergeo deleo solvo.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Sono ancilla bos. Audax cogo ciminatio fugit harum. Comburo corroboro velum.

Visit: {{url}}
"""
}
```
