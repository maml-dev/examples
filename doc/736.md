# 736 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "encouragement-web"
    namespace: "default"
    labels: {
      app: "hovel"
      tier: "data"
    }
  }

  # Deployment spec
  spec: {
    replicas: 1
    container: {
      image: "coin:8.11.20"
      port: 8000
      resources: {
        cpu_limit: "1000m"
        memory_limit: "256Mi"
      }
    }

    health_check: {
      path: "/ready"
      interval: 30
      timeout: 3
    }
  }
}
```
