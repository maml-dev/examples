# 3051 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v3"
  routes: [
    {
      method: "GET"
      path: "/products"
      handler: "updateOrders"
      auth: true
    }

    {
      method: "POST"
      path: "/comments"
      handler: "deletePosts"
      auth: false
    }

    {
      method: "POST"
      path: "/users/:id"
      handler: "deleteProducts"
      auth: false
    }
  ]
}
```
