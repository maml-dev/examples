# 1529 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "self-confidence-api"
    namespace: "production"
    labels: {
      app: "petal"
      tier: "frontend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 1
    container: {
      image: "jet:6.5.16"
      port: 3000
      resources: {
        cpu_limit: "500m"
        memory_limit: "512Mi"
      }
    }

    health_check: {
      path: "/health"
      interval: 30
      timeout: 5
    }
  }
}
```
