# 2830 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "rabbitmq"
  connection: {
    host: "mq.internal"
    port: 6379
    vhost: "/staging"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 3
      retry: 5
      timeout: 9244 # ms
    }

    priority: {
      concurrency: 8
      retry: 6
      timeout: 8796 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 9
    queue: "dead-letter"
  }
}
```
