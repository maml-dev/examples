# 2136 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "ai"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 60 # seconds
    labels: {
      env: "staging"
      region: "eu-west"
      team: "games"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 81
    memory_threshold: 85
    latency_p99_ms: 1034
    error_rate: 0.01
    notification: {
      channel: "#ops-alerts"
      severity: "info"
    }
  }
}
```
