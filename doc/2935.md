# 2935 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "diversity-web"
    namespace: "staging"
    labels: {
      app: "battle"
      tier: "frontend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 4
    container: {
      image: "comestible:6.16.4"
      port: 3000
      resources: {
        cpu_limit: "250m"
        memory_limit: "1Gi"
      }
    }

    health_check: {
      path: "/ready"
      interval: 10
      timeout: 3
    }
  }
}
```
