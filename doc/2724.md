# 2724 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.crowded-kick.net"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "1m"
    max_requests: 8587
    by: "api_key"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://nutritious-colon.com"
      "https://app.outrageous-colon.name"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 3600
  }

  cache: {
    enabled: false
    ttl: 443
    store: "redis"
  }
}
```
