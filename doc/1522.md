# 1522 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "ai"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 30 # seconds
    labels: {
      env: "prod"
      region: "eu-west"
      team: "games"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 74
    memory_threshold: 82
    latency_p99_ms: 1685
    error_rate: 0.01
    notification: {
      channel: "#incidents"
      severity: "warning"
    }
  }
}
```
