# 3958 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "vscode"
  shortcuts: [
    {
      key: "Cmd+K"
      action: "save"
      when: "editor"
    }

    {
      key: "Alt+Up"
      action: "debug_run"
      when: "editor"
    }

    {
      key: "F5"
      action: "toggle_sidebar"
      when: "debug"
    }
  ]
}
```
