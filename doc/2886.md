# 2886 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@gorgeous-subsidy.biz"
  subject: "Ultra est viridis."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Caries arma speculum cuius aeger thermae aggero acsi. Calco aspernatur tenuis. Expedita tibi verumtamen demens alienus tricesimus.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Repellendus adipiscor tergiversatio volutabrum facilis vehemens angustus compono. Impedit spes conor currus celebrer. Ter curiositas tredecim.

Visit: {{url}}
"""
}
```
