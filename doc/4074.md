# 4074 - Dashboard Layout

Dashboard widget layout configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  title: "Bosco, Metz and Robel Dashboard"
  refresh_interval: 30 # seconds
  widgets: [
    {
      type: "chart"
      title: "Traffic"
      source: "/api/metrics/revenue"
      chart_type: "line"
      period: "90d"
    }

    {
      type: "stat"
      title: "MRR"
      source: "/api/stats/users"
      format: "percent"
    }

    {
      type: "table"
      title: "Latest Signups"
      source: "/api/recent/signups"
      limit: 14
    }
  ]

  theme: "light"
}
```
