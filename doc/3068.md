# 3068 - Dashboard Layout

Dashboard widget layout configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  title: "Herman, Wyman and Gulgowski Dashboard"
  refresh_interval: 60 # seconds
  widgets: [
    {
      type: "chart"
      title: "Revenue"
      source: "/api/metrics/traffic"
      chart_type: "line"
      period: "90d"
    }

    {
      type: "stat"
      title: "Uptime"
      source: "/api/stats/users"
      format: "currency"
    }

    {
      type: "table"
      title: "Recent Orders"
      source: "/api/recent/signups"
      limit: 16
    }
  ]

  theme: "auto"
}
```
