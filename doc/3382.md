# 3382 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@best-roadway.biz"
  subject: "Convoco cometes ubi."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Decimus quidem tempus iusto tenax confugo cui adulatio. Exercitationem aegrotatio demonstro calamitas similique. Enim ullus aperiam tergo.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Numquam tergo centum bene tempora facilis at. Tero currus thalassinus via ocer. Demonstro tracto aliquid auctor aureus dignissimos anser dolorum abscido volutabrum.

Visit: {{url}}
"""
}
```
