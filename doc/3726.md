# 3726 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.super-pecan.biz"
    port: 8443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "15m"
    max_requests: 7695
    by: "api_key"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://trustworthy-sideboard.name"
      "https://app.spotless-character.name"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 7200
  }

  cache: {
    enabled: false
    ttl: 3316
    store: "memory"
  }
}
```
