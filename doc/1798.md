# 1798 - Quoted Keys

Configuration using special characters in keys. This is an example of a [MAML](https://maml.dev) document.

```maml
# HTTP Headers
{
  Content-Type: "text/html"
  X-Request-ID: "0376a73d-214b-4b15-a298-b3ad59f39457"
  Accept-Language: "de-DE"
  Cache-Control: "no-cache"
  X-Powered-By: "Koa"

  # Keys with spaces get quoted
  "headers with spaces": {
    "custom header": "odit"
    "another one": "autem"
  }
}
```
