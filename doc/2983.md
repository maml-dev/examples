# 2983 - Quoted Keys

Configuration using special characters in keys. This is an example of a [MAML](https://maml.dev) document.

```maml
# HTTP Headers
{
  Content-Type: "application/xml"
  X-Request-ID: "6a304288-5348-4cbe-bfac-c4a43c516d49"
  Accept-Language: "fr-FR"
  Cache-Control: "max-age=3600"
  X-Powered-By: "Koa"

  # Keys with spaces get quoted
  "headers with spaces": {
    "custom header": "desidero"
    "another one": "arcus"
  }
}
```
