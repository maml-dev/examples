# 944 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v3"
  routes: [
    {
      method: "DELETE"
      path: "/orders/:id"
      handler: "deletePosts"
      auth: true
    }

    {
      method: "GET"
      path: "/comments/:id"
      handler: "deletePosts"
      auth: true
    }

    {
      method: "POST"
      path: "/comments"
      handler: "updateProducts"
      auth: true
    }
  ]
}
```
