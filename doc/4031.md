# 4031 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "department-api"
    namespace: "staging"
    labels: {
      app: "majority"
      tier: "backend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 1
    container: {
      image: "swing:5.10.7"
      port: 8080
      resources: {
        cpu_limit: "500m"
        memory_limit: "256Mi"
      }
    }

    health_check: {
      path: "/ready"
      interval: 10
      timeout: 5
    }
  }
}
```
