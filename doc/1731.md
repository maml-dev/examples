# 1731 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "excellent-vicinity.com"
  inbound: {
    mx_records: [
      "mx1.excellent-vicinity.com"
      "mx2.excellent-vicinity.com"
    ]
    spam_filter: {
      enabled: true
      threshold: 5
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.excellent-vicinity.com"
    port: 25
    tls: true
    rate_limit: {
      per_minute: 477
      per_hour: 5716
    }
  }
}
```
