# 633 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "essential-bandwidth.com"
  inbound: {
    mx_records: [
      "mx1.essential-bandwidth.com"
      "mx2.essential-bandwidth.com"
    ]
    spam_filter: {
      enabled: true
      threshold: 7
      action: "quarantine"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.essential-bandwidth.com"
    port: 465
    tls: true
    rate_limit: {
      per_minute: 467
      per_hour: 6582
    }
  }
}
```
