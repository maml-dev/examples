# 159 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "vscode"
  shortcuts: [
    {
      key: "Ctrl+/"
      action: "save"
      when: "global"
    }

    {
      key: "Ctrl+`"
      action: "find_references"
      when: "terminal"
    }

    {
      key: "Ctrl+Shift+P"
      action: "find_references"
      when: "editor"
    }

    {
      key: "F5"
      action: "toggle_sidebar"
      when: "debug"
    }
  ]
}
```
