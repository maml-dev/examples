# 434 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "relationships"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 15 # seconds
    labels: {
      env: "prod"
      region: "us-east"
      team: "grocery"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 90
    memory_threshold: 78
    latency_p99_ms: 1771
    error_rate: 0.01
    notification: {
      channel: "#ops-alerts"
      severity: "info"
    }
  }
}
```
