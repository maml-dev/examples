# 670 - Dashboard Layout

Dashboard widget layout configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  title: "Blanda and Sons Dashboard"
  refresh_interval: 30 # seconds
  widgets: [
    {
      type: "chart"
      title: "Revenue"
      source: "/api/metrics/traffic"
      chart_type: "bar"
      period: "90d"
    }

    {
      type: "stat"
      title: "Uptime"
      source: "/api/stats/revenue"
      format: "number"
    }

    {
      type: "table"
      title: "Latest Signups"
      source: "/api/recent/signups"
      limit: 25
    }
  ]

  theme: "auto"
}
```
