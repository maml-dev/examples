# 2025 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.parched-handover.org"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "1m"
    max_requests: 8321
    by: "ip"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://hard-to-find-valuable.com"
      "https://app.primary-bakeware.name"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 86400
  }

  cache: {
    enabled: false
    ttl: 3279
    store: "memory"
  }
}
```
