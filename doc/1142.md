# 1142 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "that-seagull.com"
  records: [
    {
      type: "CNAME"
      name: "api"
      value: "beloved-monster.name"
      ttl: 300
    }

    {
      type: "AAAA"
      name: "*"
      value: "64.177.200.53"
      ttl: 86400
    }

    {
      type: "CNAME"
      name: "*"
      value: "165.40.249.11"
      ttl: 300
    }

    {
      type: "AAAA"
      name: "mail"
      value: "primary-injunction.name"
      ttl: 300
    }
  ]
}
```
