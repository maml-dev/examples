# 2926 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "rabbitmq"
  connection: {
    host: "mq.internal"
    port: 9092
    vhost: "/staging"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 5
      retry: 2
      timeout: 53809 # ms
    }

    priority: {
      concurrency: 11
      retry: 9
      timeout: 1232 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 7
    queue: "dead-letter"
  }
}
```
