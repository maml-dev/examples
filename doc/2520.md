# 2520 - Proxy Config

Reverse proxy settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  listen: 443
  server_name: "subtle-intervention.net"
  upstream: "http://localhost:4738"

  # TLS settings
  ssl: false
  http2: true

  gzip: true
  gzip_min_length: 256
  client_max_body: "4mb"
}
```
