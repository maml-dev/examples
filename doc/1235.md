# 1235 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "redis"
  connection: {
    host: "mq.internal"
    port: 6379
    vhost: "/"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 1
      retry: 3
      timeout: 42756 # ms
    }

    priority: {
      concurrency: 9
      retry: 5
      timeout: 6157 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 6
    queue: "dead-letter"
  }
}
```
