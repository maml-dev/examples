# 2224 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "fuzzy-dredger.name"
  inbound: {
    mx_records: [
      "mx1.fuzzy-dredger.name"
      "mx2.fuzzy-dredger.name"
    ]
    spam_filter: {
      enabled: true
      threshold: 5
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.fuzzy-dredger.name"
    port: 587
    tls: true
    rate_limit: {
      per_minute: 439
      per_hour: 6642
    }
  }
}
```
