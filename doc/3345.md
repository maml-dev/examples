# 3345 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "sublime"
  shortcuts: [
    {
      key: "Ctrl+Shift+P"
      action: "format"
      when: "debug"
    }

    {
      key: "Ctrl+`"
      action: "toggle_comment"
      when: "editor"
    }

    {
      key: "Ctrl+`"
      action: "select_all_occurrences"
      when: "editor"
    }

    {
      key: "Cmd+K"
      action: "toggle_sidebar"
      when: "editor"
    }

    {
      key: "Ctrl+`"
      action: "search_project"
      when: "terminal"
    }

    {
      key: "Ctrl+D"
      action: "toggle_sidebar"
      when: "terminal"
    }
  ]
}
```
