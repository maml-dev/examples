# 4149 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "vim"
  shortcuts: [
    {
      key: "Alt+Up"
      action: "debug_run"
      when: "terminal"
    }

    {
      key: "Ctrl+B"
      action: "format"
      when: "debug"
    }

    {
      key: "Ctrl+`"
      action: "select_all_occurrences"
      when: "global"
    }

    {
      key: "Ctrl+S"
      action: "toggle_sidebar"
      when: "editor"
    }

    {
      key: "Alt+Up"
      action: "toggle_sidebar"
      when: "terminal"
    }

    {
      key: "Cmd+K"
      action: "search_project"
      when: "global"
    }

    {
      key: "Alt+Up"
      action: "toggle_comment"
      when: "terminal"
    }
  ]
}
```
