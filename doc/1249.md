# 1249 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "tangible-bug.org"
  records: [
    {
      type: "CNAME"
      name: "www"
      value: "v=spf1 include:_spf.parched-privilege.com ~all"
      ttl: 86400
    }

    {
      type: "AAAA"
      name: "mail"
      value: "57.132.177.126"
      ttl: 86400
    }

    {
      type: "CNAME"
      name: "cdn"
      value: "v=spf1 include:_spf.square-ruin.info ~all"
      ttl: 86400
    }

    {
      type: "A"
      name: "staging"
      value: "married-inspection.name"
      ttl: 86400
    }
  ]
}
```
