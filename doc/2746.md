# 2746 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v2"
  routes: [
    {
      method: "DELETE"
      path: "/posts"
      handler: "updateUsers"
      auth: true
    }

    {
      method: "PUT"
      path: "/comments/:id"
      handler: "updateOrders"
      auth: false
    }

    {
      method: "PUT"
      path: "/orders/:id"
      handler: "updateUsers"
      auth: false
    }

    {
      method: "PATCH"
      path: "/orders/:id"
      handler: "updateProducts"
      auth: false
    }
  ]
}
```
