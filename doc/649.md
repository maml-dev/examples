# 649 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "emacs"
  shortcuts: [
    {
      key: "Ctrl+B"
      action: "search_project"
      when: "terminal"
    }

    {
      key: "Ctrl+D"
      action: "debug_run"
      when: "editor"
    }

    {
      key: "Alt+F7"
      action: "toggle_sidebar"
      when: "editor"
    }

    {
      key: "Alt+Up"
      action: "format"
      when: "editor"
    }

    {
      key: "Ctrl+/"
      action: "command_palette"
      when: "global"
    }

    {
      key: "Ctrl+D"
      action: "command_palette"
      when: "editor"
    }
  ]
}
```
