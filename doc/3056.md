# 3056 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "livid-receptor.info"
  records: [
    {
      type: "CNAME"
      name: "mail"
      value: "v=spf1 include:_spf.entire-flood.com ~all"
      ttl: 3600
    }

    {
      type: "MX"
      name: "@"
      value: "93.130.24.204"
      ttl: 3600
    }

    {
      type: "MX"
      name: "*"
      value: "vain-pinstripe.org"
      ttl: 86400
    }

    {
      type: "NS"
      name: "cdn"
      value: "delectable-mallard.info"
      ttl: 86400
    }

    {
      type: "A"
      name: "cdn"
      value: "focused-bowling.name"
      ttl: 300
    }
  ]
}
```
