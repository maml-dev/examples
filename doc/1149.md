# 1149 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "vim"
  shortcuts: [
    {
      key: "Cmd+K"
      action: "save"
      when: "editor"
    }

    {
      key: "F5"
      action: "format"
      when: "debug"
    }

    {
      key: "Ctrl+S"
      action: "save"
      when: "terminal"
    }

    {
      key: "Alt+F7"
      action: "debug_run"
      when: "editor"
    }

    {
      key: "Ctrl+B"
      action: "debug_run"
      when: "terminal"
    }

    {
      key: "Alt+F7"
      action: "toggle_sidebar"
      when: "terminal"
    }

    {
      key: "Alt+F7"
      action: "save"
      when: "editor"
    }
  ]
}
```
