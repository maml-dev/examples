# 739 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.steep-premeditation.net"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "15m"
    max_requests: 2115
    by: "user"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://impure-swath.org"
      "https://app.needy-scorpion.name"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 86400
  }

  cache: {
    enabled: false
    ttl: 333
    store: "memory"
  }
}
```
