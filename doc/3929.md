# 3929 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "front-monster.info"
  inbound: {
    mx_records: [
      "mx1.front-monster.info"
      "mx2.front-monster.info"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "reject"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.front-monster.info"
    port: 465
    tls: true
    rate_limit: {
      per_minute: 481
      per_hour: 4258
    }
  }
}
```
