# 3443 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v1"
  routes: [
    {
      method: "PUT"
      path: "/comments/:id"
      handler: "listPosts"
      auth: false
    }

    {
      method: "GET"
      path: "/posts"
      handler: "listUsers"
      auth: true
    }

    {
      method: "POST"
      path: "/orders"
      handler: "deleteUsers"
      auth: true
    }

    {
      method: "PATCH"
      path: "/products"
      handler: "getOrders"
      auth: false
    }
  ]
}
```
