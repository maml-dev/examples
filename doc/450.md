# 450 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "wicked-teriyaki.net"
  records: [
    {
      type: "TXT"
      name: "mail"
      value: "grubby-coliseum.org"
      ttl: 300
    }

    {
      type: "CNAME"
      name: "www"
      value: "v=spf1 include:_spf.petty-chairperson.com ~all"
      ttl: 300
    }

    {
      type: "A"
      name: "cdn"
      value: "unhealthy-fraudster.name"
      ttl: 3600
    }

    {
      type: "MX"
      name: "api"
      value: "trustworthy-wriggler.com"
      ttl: 3600
    }

    {
      type: "TXT"
      name: "@"
      value: "v=spf1 include:_spf.lanky-tuber.net ~all"
      ttl: 300
    }
  ]
}
```
