# 954 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "focused-formation.biz"
  records: [
    {
      type: "MX"
      name: "api"
      value: "31.74.134.139"
      ttl: 300
    }

    {
      type: "TXT"
      name: "cdn"
      value: "tricky-window.net"
      ttl: 3600
    }

    {
      type: "TXT"
      name: "staging"
      value: "surprised-knuckle.org"
      ttl: 86400
    }

    {
      type: "TXT"
      name: "mail"
      value: "v=spf1 include:_spf.lovely-violin.biz ~all"
      ttl: 3600
    }
  ]
}
```
