# 4037 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "bitter-premeditation.info"
  inbound: {
    mx_records: [
      "mx1.bitter-premeditation.info"
      "mx2.bitter-premeditation.info"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.bitter-premeditation.info"
    port: 25
    tls: true
    rate_limit: {
      per_minute: 475
      per_hour: 6824
    }
  }
}
```
