# 2006 - Logger Config

Structured logging configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  level: "fatal"
  format: "pretty"
  timestamp: true
  caller: true # include file:line

  output: "stdout"
  rotation: false
  max_size_mb: 27
  max_age_days: 9
  compress: true
}
```
