# 240 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "content"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 60 # seconds
    labels: {
      env: "staging"
      region: "ap-south"
      team: "movies"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 76
    memory_threshold: 90
    latency_p99_ms: 1515
    error_rate: 0.02
    notification: {
      channel: "#ops-alerts"
      severity: "info"
    }
  }
}
```
