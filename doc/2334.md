# 2334 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "kafka"
  connection: {
    host: "mq.internal"
    port: 5672
    vhost: "/"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 3
      retry: 5
      timeout: 40473 # ms
    }

    priority: {
      concurrency: 12
      retry: 3
      timeout: 7798 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 3
    queue: "dead-letter"
  }
}
```
