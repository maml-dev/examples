# 4300 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@impassioned-mantua.net"
  subject: "Similique deripio tot."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Vilis angelus abstergo bonus sufficio accommodo. Succurro allatus tandem cultellus suadeo trans nemo. Sopor molestias delego crux spectaculum ut teres spargo terminatio.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Esse baiulus ambitus aedificium iste tutis debilito. Corroboro quam cubo minima. Arca aegrotatio triumphus accendo vitae vae aliquid vergo.

Visit: {{url}}
"""
}
```
