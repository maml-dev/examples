# 629 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "seagull-api"
    namespace: "default"
    labels: {
      app: "cap"
      tier: "data"
    }
  }

  # Deployment spec
  spec: {
    replicas: 4
    container: {
      image: "shoulder:7.4.12"
      port: 8000
      resources: {
        cpu_limit: "500m"
        memory_limit: "512Mi"
      }
    }

    health_check: {
      path: "/health"
      interval: 10
      timeout: 3
    }
  }
}
```
