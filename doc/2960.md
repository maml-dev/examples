# 2960 - Cron Jobs

Scheduled task definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  timezone: "Europe/London"
  jobs: [
    {
      name: "sync-data"
      schedule: "0 * * * *"
      command: "python manage.py report"
      enabled: false
    }

    {
      name: "backup-db"
      schedule: "0 0 * * *"
      command: "node scripts/cleanup.js"
      enabled: true
    }

    {
      name: "cleanup-temp"
      schedule: "0 0 * * *"
      command: "pg_dump $DATABASE_URL"
      enabled: true
    }

    {
      name: "backup-db"
      schedule: "0 2 * * 0"
      command: "pg_dump $DATABASE_URL"
      enabled: true
    }

    {
      name: "rotate-logs"
      schedule: "0 2 * * 0"
      command: "pg_dump $DATABASE_URL"
      enabled: false
    }
  ]
}
```
