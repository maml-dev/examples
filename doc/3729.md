# 3729 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "kafka"
  connection: {
    host: "mq.internal"
    port: 5672
    vhost: "/staging"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 9
      retry: 5
      timeout: 26294 # ms
    }

    priority: {
      concurrency: 13
      retry: 10
      timeout: 9178 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 5
    queue: "dead-letter"
  }
}
```
