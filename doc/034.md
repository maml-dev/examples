# 034 - Theme Config

Design system theme configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  name: "silver Theme"
  # Color palette
  colors: {
    primary: "#9d51aa"
    secondary: "#2f82ae"
    accent: "#cc50a7"
    background: "#f8f9fa"

    warning: "#ffc107"
    success: "#28a745"

  # Typography

    font_family: "\"JetBrains Mono\", monospace"

      font_family: "Inter, sans-serif"

    unit: 8

      0
      4
      8
      16
      24
      32
      48
    ]
  }
  border_radius: 12
}
```
