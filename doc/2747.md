# 2747 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "support"
      permissions: [
        "manage_users"
        "write"
        "export_data"
        "delete"
      ]
      max_seats: 19
    }

    {
      name: "editor"
      permissions: [
        "manage_users"
        "delete"
        "write"
        "read"
        "export_data"
      ]
      max_seats: 43
    }

    {
      name: "viewer"
      permissions: [
        "write"
        "export_data"
        "manage_billing"
        "delete"
      ]
      max_seats: null
    }
  ]
}
```
