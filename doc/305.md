# 305 - Proxy Config

Reverse proxy settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  listen: 443
  server_name: "comfortable-catalyst.biz"
  upstream: "http://127.0.0.1:3302"

  # TLS settings
  ssl: false
  http2: true

  gzip: true
  gzip_min_length: 512
  client_max_body: "45mb"
}
```
