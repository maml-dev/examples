# 3135 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.peppery-space.org"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "15m"
    max_requests: 9069
    by: "ip"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://long-term-republican.name"
      "https://app.cool-lamp.biz"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 7200
  }

  cache: {
    enabled: true
    ttl: 2076
    store: "memory"
  }
}
```
