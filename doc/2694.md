# 2694 - Quoted Keys

Configuration using special characters in keys. This is an example of a [MAML](https://maml.dev) document.

```maml
# HTTP Headers
{
  Content-Type: "application/json"
  X-Request-ID: "9f238ad5-a655-4adf-b819-3a7b8dd51f9d"
  Accept-Language: "fr-FR"
  Cache-Control: "no-cache"
  X-Powered-By: "Express"

  # Keys with spaces get quoted
  "headers with spaces": {
    "custom header": "crur"
    "another one": "credo"
  }
}
```
