# 3331 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "architectures"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 60 # seconds
    labels: {
      env: "staging"
      region: "us-east"
      team: "home"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 84
    memory_threshold: 85
    latency_p99_ms: 1976
    error_rate: 0.01
    notification: {
      channel: "#oncall"
      severity: "info"
    }
  }
}
```
