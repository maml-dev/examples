# 2364 - Dashboard Layout

Dashboard widget layout configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  title: "Daniel, Dietrich and Hyatt Dashboard"
  refresh_interval: 300 # seconds
  widgets: [
    {
      type: "chart"
      title: "Signups"
      source: "/api/metrics/traffic"
      chart_type: "area"
      period: "7d"
    }

    {
      type: "stat"
      title: "Error Rate"
      source: "/api/stats/revenue"
      format: "currency"
    }

    {
      type: "table"
      title: "Recent Orders"
      source: "/api/recent/orders"
      limit: 13
    }
  ]

  theme: "dark"
}
```
