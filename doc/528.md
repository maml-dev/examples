# 528 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "partnerships"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 10 # seconds
    labels: {
      env: "prod"
      region: "us-east"
      team: "books"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 92
    memory_threshold: 95
    latency_p99_ms: 1132
    error_rate: 0.02
    notification: {
      channel: "#incidents"
      severity: "info"
    }
  }
}
```
