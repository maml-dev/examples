# 1430 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "sonar-api"
    namespace: "default"
    labels: {
      app: "trash"
      tier: "frontend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 5
    container: {
      image: "lender:9.4.15"
      port: 9090
      resources: {
        cpu_limit: "500m"
        memory_limit: "256Mi"
      }
    }

    health_check: {
      path: "/healthz"
      interval: 10
      timeout: 3
    }
  }
}
```
