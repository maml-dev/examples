# 4220 - Proxy Config

Reverse proxy settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  listen: 8080
  server_name: "low-translation.net"
  upstream: "http://localhost:5140"

  # TLS settings
  ssl: false
  http2: true

  gzip: true
  gzip_min_length: 256
  client_max_body: "16mb"
}
```
