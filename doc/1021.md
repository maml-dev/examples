# 1021 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "redesign-web"
    namespace: "default"
    labels: {
      app: "mousse"
      tier: "backend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 5
    container: {
      image: "pick:4.0.12"
      port: 8080
      resources: {
        cpu_limit: "1000m"
        memory_limit: "512Mi"
      }
    }

    health_check: {
      path: "/ready"
      interval: 15
      timeout: 10
    }
  }
}
```
