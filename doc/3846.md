# 3846 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "editor"
      permissions: [
        "delete"
        "read"
      ]
      max_seats: 26
    }

    {
      name: "billing"
      permissions: [
        "write"
        "view_analytics"
        "read"
        "delete"
        "export_data"
      ]
      max_seats: 35
    }

    {
      name: "viewer"
      permissions: [
        "export_data"
        "manage_billing"
      ]
      max_seats: 19
    }
  ]
}
```
