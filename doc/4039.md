# 4039 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "torn-markup.info"
  inbound: {
    mx_records: [
      "mx1.torn-markup.info"
      "mx2.torn-markup.info"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "quarantine"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.torn-markup.info"
    port: 25
    tls: true
    rate_limit: {
      per_minute: 321
      per_hour: 4880
    }
  }
}
```
