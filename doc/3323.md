# 3323 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "partnerships"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 10 # seconds
    labels: {
      env: "staging"
      region: "us-east"
      team: "beauty"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 79
    memory_threshold: 75
    latency_p99_ms: 1543
    error_rate: 0.01
    notification: {
      channel: "#oncall"
      severity: "critical"
    }
  }
}
```
