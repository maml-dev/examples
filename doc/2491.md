# 2491 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@same-guide.name"
  subject: "Fuga ducimus thermae."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Paulatim acervus quos pecus debitis. Excepturi adeptio universe atque accendo dedecor. Toties tempora toties arto varietas cur adopto doloremque.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Spero patruus iste venia tabgo tum causa aspicio. Taedium bardus quas ara substantia ager vinitor decumbo absens. Crux argumentum at somniculosus bellum corporis socius uxor demo voluptas.

Visit: {{url}}
"""
}
```
