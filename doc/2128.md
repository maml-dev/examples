# 2128 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "experiences"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 60 # seconds
    labels: {
      env: "staging"
      region: "eu-west"
      team: "tools"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 74
    memory_threshold: 87
    latency_p99_ms: 642
    error_rate: 0.05
    notification: {
      channel: "#oncall"
      severity: "warning"
    }
  }
}
```
