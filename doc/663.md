# 663 - Dashboard Layout

Dashboard widget layout configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  title: "Johnson-Schimmel, Hartmann-Glover and Rath Dashboard"
  refresh_interval: 60 # seconds
  widgets: [
    {
      type: "chart"
      title: "Signups"
      source: "/api/metrics/traffic"
      chart_type: "area"
      period: "90d"
    }

    {
      type: "stat"
      title: "Uptime"
      source: "/api/stats/revenue"
      format: "percent"
    }

    {
      type: "table"
      title: "Latest Signups"
      source: "/api/recent/orders"
      limit: 7
    }
  ]

  theme: "light"
}
```
