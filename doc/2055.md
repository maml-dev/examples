# 2055 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "viewer"
      permissions: [
        "manage_billing"
        "delete"
      ]
      max_seats: null
    }

    {
      name: "admin"
      permissions: [
        "read"
        "view_analytics"
      ]
      max_seats: 4
    }

    {
      name: "admin"
      permissions: [
        "delete"
        "manage_billing"
      ]
      max_seats: 14
    }

    {
      name: "billing"
      permissions: [
        "write"
        "view_analytics"
        "manage_billing"
        "delete"
        "read"
      ]
      max_seats: null
    }

    {
      name: "moderator"
      permissions: [
        "delete"
        "manage_billing"
      ]
      max_seats: null
    }
  ]
}
```
