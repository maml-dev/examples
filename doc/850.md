# 850 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v3"
  routes: [
    {
      method: "GET"
      path: "/posts/:id"
      handler: "createUsers"
      auth: false
    }

    {
      method: "DELETE"
      path: "/posts/:id"
      handler: "getPosts"
      auth: false
    }

    {
      method: "DELETE"
      path: "/comments/:id"
      handler: "listUsers"
      auth: false
    }
  ]
}
```
