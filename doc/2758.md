# 2758 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "vscode"
  shortcuts: [
    {
      key: "Alt+F7"
      action: "search_project"
      when: "terminal"
    }

    {
      key: "Ctrl+/"
      action: "search_project"
      when: "debug"
    }

    {
      key: "Ctrl+/"
      action: "toggle_comment"
      when: "debug"
    }

    {
      key: "Alt+F7"
      action: "save"
      when: "global"
    }
  ]
}
```
