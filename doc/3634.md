# 3634 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.narrow-graffiti.org"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "5m"
    max_requests: 834
    by: "api_key"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://trustworthy-maintainer.name"
      "https://app.silent-signature.org"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 3600
  }

  cache: {
    enabled: true
    ttl: 3517
    store: "redis"
  }
}
```
