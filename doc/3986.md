# 3986 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@spotless-league.info"
  subject: "Deprecator sol commodi."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Eveniet velit vilitas caste nisi bardus decipio tamdiu tollo. Aestas tepidus debilito statua spiritus. In chirographum territo carus.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Aequus coaegresco tardus textus cognomen debilito balbus voluptatem thermae vitiosus. Somnus autus aurum absum ait. Turpis pecus claudeo caput demum ustilo artificiose acsi crinis.

Visit: {{url}}
"""
}
```
