# 822 - Backup Strategy

Backup schedule and retention policy. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  source: {
    type: "mongodb"
    host: "db.internal"
    databases: [
      "community"
      "skean"
    ]
  }

  # Backup schedule
  schedule: {
    full: "0 2 * * 0" # weekly Sunday 2am
    incremental: "0 2 * * 1-6" # daily except Sunday
    snapshot: "0 */4 * * *" # every 4 hours
  }

  retention: {
    daily: 13
    weekly: 5
    monthly: 20
  }

  destination: {
    type: "local"
    path: "s3://backups-schnitzel"
    encryption: true
    compression: "zstd"
  }
}
```
