# 1743 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "billing"
      permissions: [
        "export_data"
        "manage_users"
      ]
      max_seats: 18
    }

    {
      name: "billing"
      permissions: [
        "manage_billing"
        "export_data"
        "delete"
      ]
      max_seats: 5
    }

    {
      name: "billing"
      permissions: [
        "delete"
        "view_analytics"
        "manage_billing"
      ]
      max_seats: 21
    }
  ]
}
```
