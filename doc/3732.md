# 3732 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.tattered-invite.info"
    port: 8443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "1m"
    max_requests: 5222
    by: "ip"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://close-dream.org"
      "https://app.bulky-membership.biz"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 86400
  }

  cache: {
    enabled: false
    ttl: 2490
    store: "memory"
  }
}
```
