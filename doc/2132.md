# 2132 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "web services"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 30 # seconds
    labels: {
      env: "prod"
      region: "eu-west"
      team: "clothing"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 78
    memory_threshold: 81
    latency_p99_ms: 1527
    error_rate: 0.01
    notification: {
      channel: "#incidents"
      severity: "critical"
    }
  }
}
```
