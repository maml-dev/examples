# 3698 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@utilized-dividend.info"
  subject: "Talus quam officiis carpo thesis video."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Cito unde timor callide capitulus amitto hic. Cum tero adulescens allatus tyrannus atrox censura caste sponte censura. Depulso summopere arbor abundans cicuta ocer praesentium voveo.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Aeneus carpo supellex aequitas animi terreo xiphias tenetur spiritus culpo. Vulticulus utrimque supplanto. Ipsam decretum consuasor numquam voluntarius sufficio virtus traho beatae aiunt.

Visit: {{url}}
"""
}
```
