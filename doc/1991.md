# 1991 - Quoted Keys

Configuration using special characters in keys. This is an example of a [MAML](https://maml.dev) document.

```maml
# HTTP Headers
{
  Content-Type: "text/html"
  X-Request-ID: "73d4eaf6-8cb3-4515-9ffc-b8a865b0896f"
  Accept-Language: "de-DE"
  Cache-Control: "max-age=3600"
  X-Powered-By: "Fastify"

  # Keys with spaces get quoted
  "headers with spaces": {
    "custom header": "tenus"
    "another one": "animadverto"
  }
}
```
