# 2784 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@innocent-flat.biz"
  subject: "Textilis vomer veritatis vindico."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Pauci magni contigo patria qui decor. Confido curiositas absorbeo administratio trepide. Capio quidem utor.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Summisse deduco somniculosus vesper. Video thermae tum demens carus communis creptio. Aureus ait expedita turba adsuesco adamo synagoga.

Visit: {{url}}
"""
}
```
