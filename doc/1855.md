# 1855 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "vscode"
  shortcuts: [
    {
      key: "Alt+F7"
      action: "find_references"
      when: "terminal"
    }

    {
      key: "F5"
      action: "save"
      when: "editor"
    }

    {
      key: "Cmd+Shift+L"
      action: "toggle_sidebar"
      when: "editor"
    }

    {
      key: "Ctrl+/"
      action: "debug_run"
      when: "debug"
    }

    {
      key: "F5"
      action: "find_references"
      when: "debug"
    }
  ]
}
```
