# 3348 - Cron Jobs

Scheduled task definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  timezone: "Europe/London"
  jobs: [
    {
      name: "sync-data"
      schedule: "0 2 * * 0"
      command: "node scripts/cleanup.js"
      enabled: true
    }

    {
      name: "refresh-cache"
      schedule: "0 2 * * 0"
      command: "node scripts/cleanup.js"
      enabled: true
    }

    {
      name: "refresh-cache"
      schedule: "30 6 * * 1-5"
      command: "node scripts/cleanup.js"
      enabled: true
    }

    {
      name: "sync-data"
      schedule: "*/15 * * * *"
      command: "./bin/sync --all"
      enabled: false
    }

    {
      name: "backup-db"
      schedule: "0 0 * * *"
      command: "pg_dump $DATABASE_URL"
      enabled: true
    }
  ]
}
```
