# 1821 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "rabbitmq"
  connection: {
    host: "localhost"
    port: 6379
    vhost: "/"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 7
      retry: 1
      timeout: 58006 # ms
    }

    priority: {
      concurrency: 7
      retry: 7
      timeout: 3702 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 5
    queue: "dead-letter"
  }
}
```
