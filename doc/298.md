# 298 - Quoted Keys

Configuration using special characters in keys. This is an example of a [MAML](https://maml.dev) document.

```maml
# HTTP Headers
{
  Content-Type: "application/json"
  X-Request-ID: "f9016159-3fe7-4c27-94ff-860ba0dad8e7"
  Accept-Language: "fr-FR"
  Cache-Control: "public, max-age=86400"
  X-Powered-By: "Hono"

  # Keys with spaces get quoted
  "headers with spaces": {
    "custom header": "tui"
    "another one": "doloribus"
  }
}
```
