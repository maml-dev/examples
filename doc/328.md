# 328 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "scientific-deed.name"
  inbound: {
    mx_records: [
      "mx1.scientific-deed.name"
      "mx2.scientific-deed.name"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.scientific-deed.name"
    port: 587
    tls: true
    rate_limit: {
      per_minute: 392
      per_hour: 7446
    }
  }
}
```
