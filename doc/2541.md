# 2541 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "vscode"
  shortcuts: [
    {
      key: "Ctrl+`"
      action: "save"
      when: "global"
    }

    {
      key: "Alt+F7"
      action: "toggle_comment"
      when: "global"
    }

    {
      key: "F5"
      action: "search_project"
      when: "editor"
    }

    {
      key: "Alt+Up"
      action: "debug_run"
      when: "editor"
    }
  ]
}
```
