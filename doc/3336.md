# 3336 - Backup Strategy

Backup schedule and retention policy. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  source: {
    type: "filesystem"
    host: "db.internal"
    databases: [
      "hippodrome"
      "printer"
    ]
  }

  # Backup schedule
  schedule: {
    full: "0 2 * * 0" # weekly Sunday 2am
    incremental: "0 2 * * 1-6" # daily except Sunday
    snapshot: "0 */4 * * *" # every 4 hours
  }

  retention: {
    daily: 10
    weekly: 12
    monthly: 12
  }

  destination: {
    type: "sftp"
    path: "/mnt/backups/march"
    encryption: true
    compression: "lz4"
  }
}
```
