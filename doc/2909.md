# 2909 - Proxy Config

Reverse proxy settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  listen: 443
  server_name: "shiny-markup.info"
  upstream: "http://localhost:3165"

  # TLS settings
  ssl: false
  http2: true

  gzip: true
  gzip_min_length: 512
  client_max_body: "19mb"
}
```
