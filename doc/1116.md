# 1116 - DNS Config

DNS zone configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "pushy-merit.biz"
  ttl: 86400 # seconds
  nameservers: [
    "ns1.acidic-louse.name"
    "ns2.proud-grouper.net"
  ]

  soa_email: "admin@vivacious-scout.org"
  refresh: 7200
  retry: 600
  expire: 604800
}
```
