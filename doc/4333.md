# 4333 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "web services"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 60 # seconds
    labels: {
      env: "prod"
      region: "eu-west"
      team: "health"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 93
    memory_threshold: 86
    latency_p99_ms: 1762
    error_rate: 0.05
    notification: {
      channel: "#ops-alerts"
      severity: "warning"
    }
  }
}
```
