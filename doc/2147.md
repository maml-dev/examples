# 2147 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v1"
  routes: [
    {
      method: "DELETE"
      path: "/comments/:id"
      handler: "updateProducts"
      auth: false
    }

    {
      method: "PATCH"
      path: "/users"
      handler: "deleteUsers"
      auth: false
    }

    {
      method: "PUT"
      path: "/orders"
      handler: "createProducts"
      auth: true
    }

    {
      method: "DELETE"
      path: "/orders/:id"
      handler: "deletePosts"
      auth: true
    }

    {
      method: "GET"
      path: "/products"
      handler: "listOrders"
      auth: true
    }
  ]
}
```
