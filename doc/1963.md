# 1963 - Dashboard Layout

Dashboard widget layout configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  title: "Jerde, Homenick and Brekke Dashboard"
  refresh_interval: 300 # seconds
  widgets: [
    {
      type: "chart"
      title: "Signups"
      source: "/api/metrics/traffic"
      chart_type: "area"
      period: "7d"
    }

    {
      type: "stat"
      title: "MRR"
      source: "/api/stats/users"
      format: "number"
    }

    {
      type: "table"
      title: "Recent Orders"
      source: "/api/recent/signups"
      limit: 25
    }
  ]

  theme: "dark"
}
```
