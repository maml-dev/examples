# 1809 - Logger Config

Structured logging configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  level: "trace"
  format: "logfmt"
  timestamp: true
  caller: false # include file:line

  output: "/var/log/app.log"
  rotation: false
  max_size_mb: 42
  max_age_days: 29
  compress: true
}
```
