# 3758 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "descriptive-stock.name"
  records: [
    {
      type: "TXT"
      name: "staging"
      value: "63.40.227.55"
      ttl: 3600
    }

    {
      type: "A"
      name: "@"
      value: "v=spf1 include:_spf.coarse-jogging.com ~all"
      ttl: 3600
    }

    {
      type: "NS"
      name: "@"
      value: "49.55.151.73"
      ttl: 300
    }

    {
      type: "CNAME"
      name: "www"
      value: "87.140.253.148"
      ttl: 86400
    }

    {
      type: "CNAME"
      name: "@"
      value: "wretched-derby.org"
      ttl: 86400
    }
  ]
}
```
