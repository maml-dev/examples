# 1541 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "sublime"
  shortcuts: [
    {
      key: "Ctrl+`"
      action: "search_project"
      when: "editor"
    }

    {
      key: "Ctrl+/"
      action: "search_project"
      when: "terminal"
    }

    {
      key: "Ctrl+Shift+F"
      action: "toggle_terminal"
      when: "terminal"
    }

    {
      key: "Cmd+Shift+L"
      action: "toggle_sidebar"
      when: "global"
    }
  ]
}
```
