# 275 - Game Config

Video game settings and preferences. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  display: {
    resolution: "1920x1080"
    fullscreen: true
    vsync: true
    fps_limit: 0
  }

  graphics: {
    quality: "ultra"
    shadows: "high"
    anti_aliasing: "TAA"
    render_distance: 12
    ray_tracing: true
  }

  audio: {
    master: 92
    music: 95
    sfx: 69
    voice: 63
  }

  controls: {
    sensitivity: 2.1
    invert_y: false
    vibration: false
  }
}
```
