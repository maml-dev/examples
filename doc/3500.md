# 3500 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@windy-resource.com"
  subject: "Audio tametsi vaco umquam."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Alo nihil vitiosus patria delego. Torrens cometes concido adopto iure delectatio denique vorago ustilo. Abscido cariosus architecto assentator termes solum nesciunt cinis statua.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Sum tricesimus concido catena cultellus calculus. Pauper dignissimos abbas uberrime talio bene autus desino spargo tempore. Sublime damnatio quas tenetur caecus vulgivagus patrocinor tamdiu occaecati.

Visit: {{url}}
"""
}
```
