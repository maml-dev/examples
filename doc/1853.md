# 1853 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "emacs"
  shortcuts: [
    {
      key: "Ctrl+`"
      action: "find_references"
      when: "debug"
    }

    {
      key: "Ctrl+Shift+F"
      action: "command_palette"
      when: "debug"
    }

    {
      key: "Ctrl+/"
      action: "search_project"
      when: "editor"
    }

    {
      key: "Ctrl+Shift+F"
      action: "select_all_occurrences"
      when: "terminal"
    }

    {
      key: "Alt+F7"
      action: "save"
      when: "global"
    }

    {
      key: "Alt+F7"
      action: "command_palette"
      when: "editor"
    }
  ]
}
```
