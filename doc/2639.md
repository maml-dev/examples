# 2639 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "stranger-worker"
    namespace: "production"
    labels: {
      app: "armoire"
      tier: "data"
    }
  }

  # Deployment spec
  spec: {
    replicas: 1
    container: {
      image: "steeple:9.0.16"
      port: 8000
      resources: {
        cpu_limit: "250m"
        memory_limit: "512Mi"
      }
    }

    health_check: {
      path: "/healthz"
      interval: 15
      timeout: 10
    }
  }
}
```
