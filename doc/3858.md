# 3858 - Cron Jobs

Scheduled task definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  timezone: "Europe/London"
  jobs: [
    {
      name: "cleanup-temp"
      schedule: "30 6 * * 1-5"
      command: "node scripts/cleanup.js"
      enabled: false
    }

    {
      name: "refresh-cache"
      schedule: "0 * * * *"
      command: "./bin/sync --all"
      enabled: true
    }

    {
      name: "send-reports"
      schedule: "0 0 * * *"
      command: "python manage.py report"
      enabled: true
    }

    {
      name: "send-reports"
      schedule: "*/15 * * * *"
      command: "python manage.py report"
      enabled: false
    }

    {
      name: "sync-data"
      schedule: "*/15 * * * *"
      command: "node scripts/cleanup.js"
      enabled: false
    }
  ]
}
```
