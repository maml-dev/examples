# 1768 - Dashboard Layout

Dashboard widget layout configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  title: "Littel - Johnson Dashboard"
  refresh_interval: 300 # seconds
  widgets: [
    {
      type: "chart"
      title: "Revenue"
      source: "/api/metrics/traffic"
      chart_type: "area"
      period: "90d"
    }

    {
      type: "stat"
      title: "Uptime"
      source: "/api/stats/revenue"
      format: "currency"
    }

    {
      type: "table"
      title: "Top Pages"
      source: "/api/recent/orders"
      limit: 11
    }
  ]

  theme: "dark"
}
```
