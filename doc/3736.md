# 3736 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "forceful-guacamole.net"
  inbound: {
    mx_records: [
      "mx1.forceful-guacamole.net"
      "mx2.forceful-guacamole.net"
    ]
    spam_filter: {
      enabled: true
      threshold: 5
      action: "quarantine"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.forceful-guacamole.net"
    port: 25
    tls: true
    rate_limit: {
      per_minute: 483
      per_hour: 3808
    }
  }
}
```
