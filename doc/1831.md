# 1831 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "cruel-sightseeing.com"
  inbound: {
    mx_records: [
      "mx1.cruel-sightseeing.com"
      "mx2.cruel-sightseeing.com"
    ]
    spam_filter: {
      enabled: true
      threshold: 5
      action: "reject"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.cruel-sightseeing.com"
    port: 587
    tls: true
    rate_limit: {
      per_minute: 220
      per_hour: 3568
    }
  }
}
```
