# 658 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "monstrous-horde.com"
  records: [
    {
      type: "A"
      name: "staging"
      value: "32.171.71.231"
      ttl: 300
    }

    {
      type: "MX"
      name: "api"
      value: "agile-avalanche.com"
      ttl: 300
    }

    {
      type: "AAAA"
      name: "www"
      value: "v=spf1 include:_spf.trusty-kit.biz ~all"
      ttl: 3600
    }

    {
      type: "A"
      name: "@"
      value: "5.214.164.63"
      ttl: 300
    }

    {
      type: "TXT"
      name: "mail"
      value: "254.154.88.172"
      ttl: 3600
    }

    {
      type: "MX"
      name: "@"
      value: "21.141.54.51"
      ttl: 300
    }

    {
      type: "MX"
      name: "www"
      value: "v=spf1 include:_spf.wide-eyed-creator.net ~all"
      ttl: 3600
    }
  ]
}
```
