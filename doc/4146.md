# 4146 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "editor"
      permissions: [
        "view_analytics"
        "delete"
        "export_data"
      ]
      max_seats: 16
    }

    {
      name: "moderator"
      permissions: [
        "export_data"
        "read"
      ]
      max_seats: 29
    }

    {
      name: "support"
      permissions: [
        "delete"
        "manage_users"
      ]
      max_seats: 2
    }

    {
      name: "support"
      permissions: [
        "export_data"
        "read"
      ]
      max_seats: null
    }

    {
      name: "viewer"
      permissions: [
        "view_analytics"
        "write"
        "read"
        "manage_users"
      ]
      max_seats: null
    }
  ]
}
```
