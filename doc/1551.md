# 1551 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "vscode"
  shortcuts: [
    {
      key: "Ctrl+/"
      action: "command_palette"
      when: "debug"
    }

    {
      key: "Ctrl+`"
      action: "select_all_occurrences"
      when: "editor"
    }

    {
      key: "Cmd+K"
      action: "debug_run"
      when: "terminal"
    }

    {
      key: "Alt+F7"
      action: "format"
      when: "terminal"
    }
  ]
}
```
