# 1746 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "emacs"
  shortcuts: [
    {
      key: "Cmd+Shift+L"
      action: "format"
      when: "debug"
    }

    {
      key: "Ctrl+D"
      action: "search_project"
      when: "debug"
    }

    {
      key: "Alt+F7"
      action: "format"
      when: "global"
    }

    {
      key: "Ctrl+Shift+P"
      action: "toggle_terminal"
      when: "global"
    }

    {
      key: "Ctrl+S"
      action: "format"
      when: "editor"
    }
  ]
}
```
