# 1223 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "redis"
  connection: {
    host: "localhost"
    port: 6379
    vhost: "/"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 5
      retry: 4
      timeout: 23513 # ms
    }

    priority: {
      concurrency: 19
      retry: 7
      timeout: 2722 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 7
    queue: "dead-letter"
  }
}
```
