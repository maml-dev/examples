# 2836 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "mindshare"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 15 # seconds
    labels: {
      env: "prod"
      region: "eu-west"
      team: "books"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 80
    memory_threshold: 85
    latency_p99_ms: 1667
    error_rate: 0.02
    notification: {
      channel: "#incidents"
      severity: "critical"
    }
  }
}
```
