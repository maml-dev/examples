# 4239 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "architectures"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 60 # seconds
    labels: {
      env: "staging"
      region: "eu-west"
      team: "toys"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 84
    memory_threshold: 90
    latency_p99_ms: 982
    error_rate: 0.01
    notification: {
      channel: "#oncall"
      severity: "warning"
    }
  }
}
```
