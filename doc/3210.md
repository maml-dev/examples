# 3210 - DNS Config

DNS zone configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "sad-lift.net"
  ttl: 300 # seconds
  nameservers: [
    "ns1.glum-pine.com"
    "ns2.frugal-cheetah.net"
  ]

  soa_email: "admin@these-lieu.org"
  refresh: 7200
  retry: 900
  expire: 604800
}
```
