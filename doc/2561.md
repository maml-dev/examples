# 2561 - Dashboard Layout

Dashboard widget layout configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  title: "Effertz-Nienow, Bode and Jacobi Dashboard"
  refresh_interval: 300 # seconds
  widgets: [
    {
      type: "chart"
      title: "Active Users"
      source: "/api/metrics/revenue"
      chart_type: "area"
      period: "90d"
    }

    {
      type: "stat"
      title: "MRR"
      source: "/api/stats/users"
      format: "currency"
    }

    {
      type: "table"
      title: "Top Pages"
      source: "/api/recent/signups"
      limit: 9
    }
  ]

  theme: "auto"
}
```
