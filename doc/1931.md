# 1931 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "gown-api"
    namespace: "production"
    labels: {
      app: "fork"
      tier: "data"
    }
  }

  # Deployment spec
  spec: {
    replicas: 1
    container: {
      image: "giant:6.12.1"
      port: 9090
      resources: {
        cpu_limit: "500m"
        memory_limit: "256Mi"
      }
    }

    health_check: {
      path: "/ready"
      interval: 10
      timeout: 5
    }
  }
}
```
