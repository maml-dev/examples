# 3937 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "unripe-blowgun.name"
  inbound: {
    mx_records: [
      "mx1.unripe-blowgun.name"
      "mx2.unripe-blowgun.name"
    ]
    spam_filter: {
      enabled: true
      threshold: 5
      action: "quarantine"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.unripe-blowgun.name"
    port: 465
    tls: true
    rate_limit: {
      per_minute: 117
      per_hour: 7245
    }
  }
}
```
