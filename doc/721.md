# 721 - Backup Strategy

Backup schedule and retention policy. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  source: {
    type: "mongodb"
    host: "db.internal"
    databases: [
      "hose"
      "tenant"
    ]
  }

  # Backup schedule
  schedule: {
    full: "0 2 * * 0" # weekly Sunday 2am
    incremental: "0 2 * * 1-6" # daily except Sunday
    snapshot: "0 */4 * * *" # every 4 hours
  }

  retention: {
    daily: 13
    weekly: 7
    monthly: 7
  }

  destination: {
    type: "gcs"
    path: "/mnt/backups/bookcase"
    encryption: true
    compression: "lz4"
  }
}
```
