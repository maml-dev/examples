# 3281 - Quoted Keys

Configuration using special characters in keys. This is an example of a [MAML](https://maml.dev) document.

```maml
# HTTP Headers
{
  Content-Type: "application/json"
  X-Request-ID: "4fd8c2e1-6d62-414f-a6a7-bc09608ef0ca"
  Accept-Language: "fr-FR"
  Cache-Control: "no-cache"
  X-Powered-By: "Koa"

  # Keys with spaces get quoted
  "headers with spaces": {
    "custom header": "demo"
    "another one": "avarus"
  }
}
```
