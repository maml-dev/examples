# 4321 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "e-commerce"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 10 # seconds
    labels: {
      env: "prod"
      region: "eu-west"
      team: "electronics"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 87
    memory_threshold: 94
    latency_p99_ms: 1851
    error_rate: 0.05
    notification: {
      channel: "#incidents"
      severity: "warning"
    }
  }
}
```
