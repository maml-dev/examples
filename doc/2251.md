# 2251 - Cron Jobs

Scheduled task definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  timezone: "America/New_York"
  jobs: [
    {
      name: "rotate-logs"
      schedule: "0 2 * * 0"
      command: "./bin/sync --all"
      enabled: true
    }

    {
      name: "rotate-logs"
      schedule: "0 * * * *"
      command: "python manage.py report"
      enabled: false
    }

    {
      name: "cleanup-temp"
      schedule: "0 * * * *"
      command: "pg_dump $DATABASE_URL"
      enabled: false
    }

    {
      name: "cleanup-temp"
      schedule: "0 2 * * 0"
      command: "python manage.py report"
      enabled: true
    }
  ]
}
```
