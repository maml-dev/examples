# 3516 - Proxy Config

Reverse proxy settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  listen: 443
  server_name: "aggravating-blossom.biz"
  upstream: "http://localhost:7384"

  # TLS settings
  ssl: true
  http2: true

  gzip: true
  gzip_min_length: 1024
  client_max_body: "13mb"
}
```
