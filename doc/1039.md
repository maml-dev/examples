# 1039 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "technician-worker"
    namespace: "staging"
    labels: {
      app: "replacement"
      tier: "data"
    }
  }

  # Deployment spec
  spec: {
    replicas: 5
    container: {
      image: "thorn:3.9.9"
      port: 9090
      resources: {
        cpu_limit: "250m"
        memory_limit: "512Mi"
      }
    }

    health_check: {
      path: "/ready"
      interval: 30
      timeout: 10
    }
  }
}
```
