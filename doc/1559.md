# 1559 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "viewer"
      permissions: [
        "manage_billing"
        "view_analytics"
        "manage_users"
        "read"
      ]
      max_seats: null
    }

    {
      name: "editor"
      permissions: [
        "manage_users"
        "view_analytics"
        "manage_billing"
      ]
      max_seats: 40
    }

    {
      name: "viewer"
      permissions: [
        "view_analytics"
        "write"
        "export_data"
        "read"
      ]
      max_seats: null
    }

    {
      name: "editor"
      permissions: [
        "delete"
        "manage_billing"
        "read"
        "view_analytics"
      ]
      max_seats: null
    }
  ]
}
```
