# 1046 - Cron Jobs

Scheduled task definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  timezone: "Europe/London"
  jobs: [
    {
      name: "sync-data"
      schedule: "0 * * * *"
      command: "node scripts/cleanup.js"
      enabled: true
    }

    {
      name: "cleanup-temp"
      schedule: "*/15 * * * *"
      command: "python manage.py report"
      enabled: true
    }

    {
      name: "sync-data"
      schedule: "0 * * * *"
      command: "./bin/sync --all"
      enabled: false
    }

    {
      name: "backup-db"
      schedule: "0 2 * * 0"
      command: "node scripts/cleanup.js"
      enabled: false
    }

    {
      name: "cleanup-temp"
      schedule: "*/15 * * * *"
      command: "node scripts/cleanup.js"
      enabled: false
    }

    {
      name: "cleanup-temp"
      schedule: "0 * * * *"
      command: "python manage.py report"
      enabled: false
    }
  ]
}
```
