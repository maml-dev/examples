# 3990 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@spotless-management.com"
  subject: "Vinculum nostrum aperiam."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Stillicidium amiculum tremo thema stips. Ullam conventus eaque. Antea dolore tubineus.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Possimus nemo suffoco sustineo uter beatae aveho tactus. Synagoga vapulus repellendus conservo terminatio victus. Suffragium coniuratio tempora conculco dolores clementia.

Visit: {{url}}
"""
}
```
