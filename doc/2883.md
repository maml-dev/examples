# 2883 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@unfortunate-guide.com"
  subject: "Caute thesis adopto numquam."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Stips tantum cunae. Vulgus concedo denique accusator ullus cultellus carcer crur capto. Deserunt tres sustineo dapifer strues omnis quam uredo quisquam supellex.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Apud maxime ultra esse cura ciminatio umbra arca. Arto capio decumbo communis tamen. Reprehenderit supra illo demergo amissio bos cavus terra.

Visit: {{url}}
"""
}
```
