# 1424 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "platforms"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 15 # seconds
    labels: {
      env: "prod"
      region: "ap-south"
      team: "baby"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 78
    memory_threshold: 94
    latency_p99_ms: 783
    error_rate: 0.02
    notification: {
      channel: "#ops-alerts"
      severity: "info"
    }
  }
}
```
