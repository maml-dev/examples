# 040 - Theme Config

Design system theme configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  name: "short-term Theme"
  # Color palette
  colors: {
    primary: "#a80bbe"
    secondary: "#db639e"
    accent: "#82bebe"
    background: "#ffffff"
    text: "#333333"

    error: "#dc3545"
    warning: "#ffc107"
    success: "#28a745"
  }

  # Typography
  typography: {
    font_family: "\"JetBrains Mono\", monospace"
    base_size: 16
    line_height: 1.6
    headings: {
      font_family: "\"Playfair Display\", serif"
      weight: 700
    }
  }

  spacing: {
    unit: 4
    scale: [
      0
      4
      8
      16
      24
      32
      48
    ]
  }
  border_radius: 8
}
```
