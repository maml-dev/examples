# 1325 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.next-elevation.com"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "5m"
    max_requests: 1923
    by: "user"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://bouncy-retrospectivity.net"
      "https://app.pointless-habit.info"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 3600
  }

  cache: {
    enabled: true
    ttl: 3402
    store: "redis"
  }
}
```
