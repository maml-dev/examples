# 1425 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.competent-remark.com"
    port: 8443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "1m"
    max_requests: 1592
    by: "user"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://blind-alb.biz"
      "https://app.grumpy-chasuble.net"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 7200
  }

  cache: {
    enabled: false
    ttl: 344
    store: "memory"
  }
}
```
