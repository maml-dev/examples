# 1060 - Environment Vars

Environment variable definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  app: "apparatus"
  variables: [
    {
      name: "AWS_ACCESS_KEY"
      required: true
      secret: true
      default: null
    }

    {
      name: "SENTRY_DSN"
      required: true
      secret: false
      default: "ambulo"
    }

    {
      name: "API_KEY"
      required: false
      secret: true
      default: "abbas"
    }

    {
      name: "SMTP_PASSWORD"
      required: false
      secret: false
      default: "minima"
    }
  ]
}
```
