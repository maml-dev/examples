# 1733 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.dirty-schnitzel.org"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "1m"
    max_requests: 3035
    by: "user"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://chubby-pilot.com"
      "https://app.concerned-poppy.biz"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 86400
  }

  cache: {
    enabled: false
    ttl: 2394
    store: "redis"
  }
}
```
