# 194 - Quoted Keys

Configuration using special characters in keys. This is an example of a [MAML](https://maml.dev) document.

```maml
# HTTP Headers - keys need quoting
{
  Content-Type: "text/html"
  X-Request-ID: "5c09be0a-ee8b-44ee-9845-6a52beac2bd9"
  Accept-Language: "de-DE"
  Cache-Control: "public, max-age=86400"
  X-Powered-By: "Koa"

  # Keys with spaces get quoted
  "headers with spaces": {
    "custom header": "curis"
    "another one": "tredecim"
  }
}
```
