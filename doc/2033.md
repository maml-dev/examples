# 2033 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.glass-statue.org"
    port: 8443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "15m"
    max_requests: 4212
    by: "user"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://tasty-nudge.com"
      "https://app.fair-antelope.info"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 7200
  }

  cache: {
    enabled: false
    ttl: 2347
    store: "memory"
  }
}
```
