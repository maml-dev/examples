# 1938 - Backup Strategy

Backup schedule and retention policy. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  source: {
    type: "mongodb"
    host: "db.internal"
    databases: [
      "ectoderm"
      "bourgeoisie"
    ]
  }

  # Backup schedule
  schedule: {
    full: "0 2 * * 0" # weekly Sunday 2am
    incremental: "0 2 * * 1-6" # daily except Sunday
    snapshot: "0 */4 * * *" # every 4 hours
  }

  retention: {
    daily: 11
    weekly: 10
    monthly: 23
  }

  destination: {
    type: "sftp"
    path: "/mnt/backups/tuber"
    encryption: true
    compression: "zstd"
  }
}
```
