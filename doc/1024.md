# 1024 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "turbulent-apricot.net"
  inbound: {
    mx_records: [
      "mx1.turbulent-apricot.net"
      "mx2.turbulent-apricot.net"
    ]
    spam_filter: {
      enabled: true
      threshold: 5
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.turbulent-apricot.net"
    port: 465
    tls: true
    rate_limit: {
      per_minute: 318
      per_hour: 5789
    }
  }
}
```
