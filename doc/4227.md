# 4227 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "partnerships"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 30 # seconds
    labels: {
      env: "staging"
      region: "eu-west"
      team: "kids"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 83
    memory_threshold: 80
    latency_p99_ms: 1151
    error_rate: 0.05
    notification: {
      channel: "#oncall"
      severity: "warning"
    }
  }
}
```
