# 2757 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "vim"
  shortcuts: [
    {
      key: "Ctrl+/"
      action: "find_references"
      when: "terminal"
    }

    {
      key: "Alt+F7"
      action: "toggle_comment"
      when: "debug"
    }

    {
      key: "Alt+F7"
      action: "format"
      when: "global"
    }

    {
      key: "Ctrl+`"
      action: "find_references"
      when: "terminal"
    }

    {
      key: "Alt+Up"
      action: "find_references"
      when: "debug"
    }
  ]
}
```
