# 428 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "translation-web"
    namespace: "staging"
    labels: {
      app: "technologist"
      tier: "data"
    }
  }

  # Deployment spec
  spec: {
    replicas: 2
    container: {
      image: "citizen:6.7.4"
      port: 9090
      resources: {
        cpu_limit: "250m"
        memory_limit: "512Mi"
      }
    }

    health_check: {
      path: "/healthz"
      interval: 10
      timeout: 3
    }
  }
}
```
