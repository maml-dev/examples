# 2098 - Quoted Keys

Configuration using special characters in keys. This is an example of a [MAML](https://maml.dev) document.

```maml
# HTTP Headers
{
  Content-Type: "application/json"
  X-Request-ID: "74165066-5d6d-4ac5-8616-e2973aa7bc89"
  Accept-Language: "fr-FR"
  Cache-Control: "public, max-age=86400"
  X-Powered-By: "Koa"

  # Keys with spaces get quoted
  "headers with spaces": {
    "custom header": "aptus"
    "another one": "laborum"
  }
}
```
