# 2725 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "redis"
  connection: {
    host: "localhost"
    port: 9092
    vhost: "/production"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 10
      retry: 2
      timeout: 55381 # ms
    }

    priority: {
      concurrency: 14
      retry: 9
      timeout: 10000 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 7
    queue: "dead-letter"
  }
}
```
