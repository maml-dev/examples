# 4040 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.superb-plugin.biz"
    port: 8443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "1m"
    max_requests: 2442
    by: "user"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://darling-clamp.com"
      "https://app.dazzling-thread.name"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 86400
  }

  cache: {
    enabled: true
    ttl: 2316
    store: "redis"
  }
}
```
