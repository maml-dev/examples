# 1098 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@quarterly-boyfriend.info"
  subject: "Ventosus consuasor vapulus."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Victus tubineus abeo amiculum terra abundans aegrotatio. Currus vita carus tantillus cognomen deleo provident termes facilis confero. Terra vilicus adversus sono bellum aliquam.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Coma aperio deprecator callide civitas arcus acceptus aetas arguo. Appello delicate aeneus. Arx alter succedo ducimus quam.

Visit: {{url}}
"""
}
```
