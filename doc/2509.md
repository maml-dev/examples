# 2509 - Logger Config

Structured logging configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  level: "trace"
  format: "json"
  timestamp: true
  caller: true # include file:line

  output: "stderr"
  rotation: false
  max_size_mb: 258
  max_age_days: 82
  compress: true
}
```
