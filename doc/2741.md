# 2741 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "vscode"
  shortcuts: [
    {
      key: "Ctrl+Shift+F"
      action: "select_all_occurrences"
      when: "global"
    }

    {
      key: "Ctrl+`"
      action: "save"
      when: "terminal"
    }

    {
      key: "Cmd+Shift+L"
      action: "toggle_comment"
      when: "terminal"
    }

    {
      key: "Ctrl+/"
      action: "debug_run"
      when: "debug"
    }

    {
      key: "Alt+F7"
      action: "toggle_terminal"
      when: "debug"
    }

    {
      key: "Ctrl+Shift+F"
      action: "toggle_terminal"
      when: "editor"
    }

    {
      key: "Alt+Up"
      action: "find_references"
      when: "global"
    }
  ]
}
```
