# 1925 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "fen-web"
    namespace: "staging"
    labels: {
      app: "morbidity"
      tier: "backend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 5
    container: {
      image: "platter:6.6.6"
      port: 8000
      resources: {
        cpu_limit: "250m"
        memory_limit: "256Mi"
      }
    }

    health_check: {
      path: "/ready"
      interval: 15
      timeout: 5
    }
  }
}
```
