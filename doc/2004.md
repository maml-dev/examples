# 2004 - Proxy Config

Reverse proxy settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  listen: 443
  server_name: "jittery-bump.biz"
  upstream: "http://127.0.0.1:3906"

  # TLS settings
  ssl: false
  http2: true

  gzip: true
  gzip_min_length: 256
  client_max_body: "2mb"
}
```
