# 1290 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@coarse-dredger.biz"
  subject: "Velociter tripudio conspergo suasoria."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Adopto accommodo caelum alienus est crepusculum cibus toties infit cura. Beneficium sodalitas villa sto auctus nemo id volup harum. Valetudo dicta adhuc.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Demens cetera cito tendo vere. Infit cui sto amor balbus apto. Bene hic abstergo.

Visit: {{url}}
"""
}
```
