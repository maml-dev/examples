# 1144 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "vscode"
  shortcuts: [
    {
      key: "Ctrl+D"
      action: "toggle_terminal"
      when: "global"
    }

    {
      key: "Ctrl+D"
      action: "find_references"
      when: "global"
    }

    {
      key: "F5"
      action: "find_references"
      when: "editor"
    }

    {
      key: "Alt+F7"
      action: "command_palette"
      when: "editor"
    }

    {
      key: "Ctrl+D"
      action: "command_palette"
      when: "terminal"
    }

    {
      key: "Ctrl+Shift+F"
      action: "debug_run"
      when: "terminal"
    }
  ]
}
```
