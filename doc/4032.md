# 4032 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "exotic-effector.org"
  inbound: {
    mx_records: [
      "mx1.exotic-effector.org"
      "mx2.exotic-effector.org"
    ]
    spam_filter: {
      enabled: true
      threshold: 5
      action: "quarantine"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.exotic-effector.org"
    port: 465
    tls: true
    rate_limit: {
      per_minute: 293
      per_hour: 1333
    }
  }
}
```
