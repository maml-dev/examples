# 2228 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "technologies"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 30 # seconds
    labels: {
      env: "staging"
      region: "eu-west"
      team: "clothing"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 81
    memory_threshold: 89
    latency_p99_ms: 230
    error_rate: 0.01
    notification: {
      channel: "#oncall"
      severity: "warning"
    }
  }
}
```
