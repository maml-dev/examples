# 2331 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "digital-metabolite.name"
  inbound: {
    mx_records: [
      "mx1.digital-metabolite.name"
      "mx2.digital-metabolite.name"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "reject"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.digital-metabolite.name"
    port: 587
    tls: true
    rate_limit: {
      per_minute: 277
      per_hour: 7881
    }
  }
}
```
