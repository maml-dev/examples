# 2329 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.smooth-curl.info"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "15m"
    max_requests: 6274
    by: "ip"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://showy-gazebo.info"
      "https://app.buzzing-sanity.info"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 7200
  }

  cache: {
    enabled: false
    ttl: 3050
    store: "redis"
  }
}
```
