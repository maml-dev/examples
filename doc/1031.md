# 1031 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "swift-aftermath.org"
  inbound: {
    mx_records: [
      "mx1.swift-aftermath.org"
      "mx2.swift-aftermath.org"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "reject"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.swift-aftermath.org"
    port: 25
    tls: true
    rate_limit: {
      per_minute: 186
      per_hour: 9190
    }
  }
}
```
