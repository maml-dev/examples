# 3924 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "markets"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 10 # seconds
    labels: {
      env: "staging"
      region: "ap-south"
      team: "clothing"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 81
    memory_threshold: 90
    latency_p99_ms: 1169
    error_rate: 0.02
    notification: {
      channel: "#ops-alerts"
      severity: "warning"
    }
  }
}
```
