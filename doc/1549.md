# 1549 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "worthwhile-fireplace.biz"
  records: [
    {
      type: "NS"
      name: "api"
      value: "rewarding-vein.info"
      ttl: 3600
    }

    {
      type: "NS"
      name: "www"
      value: "growing-verve.net"
      ttl: 3600
    }

    {
      type: "MX"
      name: "@"
      value: "244.115.145.244"
      ttl: 3600
    }

    {
      type: "CNAME"
      name: "www"
      value: "wide-cannon.info"
      ttl: 86400
    }
  ]
}
```
