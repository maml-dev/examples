# 2760 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "terrible-chasuble.info"
  records: [
    {
      type: "MX"
      name: "staging"
      value: "shy-publicity.biz"
      ttl: 300
    }

    {
      type: "NS"
      name: "api"
      value: "121.70.57.205"
      ttl: 3600
    }

    {
      type: "A"
      name: "*"
      value: "103.143.221.75"
      ttl: 86400
    }

    {
      type: "CNAME"
      name: "@"
      value: "spirited-relative.com"
      ttl: 3600
    }
  ]
}
```
