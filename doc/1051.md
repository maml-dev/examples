# 1051 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v1"
  routes: [
    {
      method: "DELETE"
      path: "/orders"
      handler: "updatePosts"
      auth: true
    }

    {
      method: "GET"
      path: "/comments"
      handler: "listProducts"
      auth: false
    }

    {
      method: "GET"
      path: "/products"
      handler: "createProducts"
      auth: true
    }

    {
      method: "DELETE"
      path: "/products"
      handler: "getPosts"
      auth: true
    }
  ]
}
```
