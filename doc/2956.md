# 2956 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "outrageous-elevator.net"
  records: [
    {
      type: "A"
      name: "*"
      value: "torn-appliance.biz"
      ttl: 86400
    }

    {
      type: "TXT"
      name: "mail"
      value: "137.124.62.59"
      ttl: 300
    }

    {
      type: "MX"
      name: "www"
      value: "223.132.181.167"
      ttl: 300
    }

    {
      type: "CNAME"
      name: "www"
      value: "166.34.219.135"
      ttl: 86400
    }
  ]
}
```
