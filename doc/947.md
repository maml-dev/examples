# 947 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v3"
  routes: [
    {
      method: "DELETE"
      path: "/posts/:id"
      handler: "createPosts"
      auth: false
    }

    {
      method: "PUT"
      path: "/products"
      handler: "getPosts"
      auth: false
    }

    {
      method: "PUT"
      path: "/users/:id"
      handler: "createUsers"
      auth: false
    }
  ]
}
```
