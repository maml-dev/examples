# 2831 - Backup Strategy

Backup schedule and retention policy. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  source: {
    type: "postgresql"
    host: "db.internal"
    databases: [
      "decongestant"
      "injunction"
    ]
  }

  # Backup schedule
  schedule: {
    full: "0 2 * * 0" # weekly Sunday 2am
    incremental: "0 2 * * 1-6" # daily except Sunday
    snapshot: "0 */4 * * *" # every 4 hours
  }

  retention: {
    daily: 14
    weekly: 5
    monthly: 24
  }

  destination: {
    type: "sftp"
    path: "s3://backups-crest"
    encryption: true
    compression: "gzip"
  }
}
```
