# 3121 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.slimy-citizen.net"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "15m"
    max_requests: 9620
    by: "user"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://serene-cope.name"
      "https://app.frilly-boulevard.net"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 86400
  }

  cache: {
    enabled: true
    ttl: 3593
    store: "redis"
  }
}
```
