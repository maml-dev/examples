# 035 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "applications"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 60 # seconds
    labels: {
      env: "staging"
      region: "ap-south"
      team: "automotive"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 93
    memory_threshold: 90
    latency_p99_ms: 485
    error_rate: 0.01
    notification: {
      channel: "#oncall"
      severity: "info"
    }
  }
}
```
