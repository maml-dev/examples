# 1114 - DNS Config

DNS zone configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "wide-eyed-cod.com"
  ttl: 86400 # seconds
  nameservers: [
    "ns1.worst-word.info"
    "ns2.spanish-pine.biz"
  ]

  soa_email: "admin@diligent-swanling.biz"
  refresh: 14400
  retry: 1800
  expire: 604800
}
```
