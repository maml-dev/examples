# 2106 - Logger Config

Structured logging configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  level: "trace"
  format: "logfmt"
  timestamp: true
  caller: false # include file:line

  output: "stdout"
  rotation: false
  max_size_mb: 46
  max_age_days: 53
  compress: true
}
```
