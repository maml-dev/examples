# 948 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "admin"
      permissions: [
        "export_data"
        "view_analytics"
        "manage_billing"
      ]
      max_seats: 47
    }

    {
      name: "editor"
      permissions: [
        "view_analytics"
        "export_data"
        "write"
        "manage_billing"
      ]
      max_seats: 44
    }

    {
      name: "moderator"
      permissions: [
        "delete"
        "write"
        "manage_users"
      ]
      max_seats: null
    }
  ]
}
```
