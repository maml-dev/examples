# 610 - Linter Config

Code linter rule configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  parser: "typescript"
  env: "es2024"
  indent: 2
  semicolons: true
  quotes: "single"

  # Formatting
  max_line_length: 100
  trailing_comma: "es5"
  no_unused_vars: true # error
  no_console: true # warn
}
```
