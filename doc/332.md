# 332 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "cavernous-pneumonia.info"
  inbound: {
    mx_records: [
      "mx1.cavernous-pneumonia.info"
      "mx2.cavernous-pneumonia.info"
    ]
    spam_filter: {
      enabled: true
      threshold: 5
      action: "quarantine"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.cavernous-pneumonia.info"
    port: 25
    tls: true
    rate_limit: {
      per_minute: 481
      per_hour: 4897
    }
  }
}
```
