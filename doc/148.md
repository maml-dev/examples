# 148 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "diligent-yeast.name"
  records: [
    {
      type: "MX"
      name: "*"
      value: "wordy-tarragon.biz"
      ttl: 300
    }

    {
      type: "A"
      name: "mail"
      value: "mature-wombat.org"
      ttl: 86400
    }

    {
      type: "AAAA"
      name: "www"
      value: "152.95.53.232"
      ttl: 300
    }

    {
      type: "AAAA"
      name: "api"
      value: "164.221.218.107"
      ttl: 3600
    }
  ]
}
```
