# 239 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.talkative-amnesty.org"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "5m"
    max_requests: 6825
    by: "user"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://wavy-lashes.com"
      "https://app.lanky-range.org"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 7200
  }

  cache: {
    enabled: false
    ttl: 1454
    store: "redis"
  }
}
```
