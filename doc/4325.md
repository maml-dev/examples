# 4325 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "hubris-api"
    namespace: "production"
    labels: {
      app: "sarong"
      tier: "frontend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 5
    container: {
      image: "alligator:2.16.7"
      port: 3000
      resources: {
        cpu_limit: "250m"
        memory_limit: "256Mi"
      }
    }

    health_check: {
      path: "/health"
      interval: 10
      timeout: 5
    }
  }
}
```
