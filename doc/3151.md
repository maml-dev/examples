# 3151 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "sublime"
  shortcuts: [
    {
      key: "Ctrl+B"
      action: "debug_run"
      when: "debug"
    }

    {
      key: "Ctrl+/"
      action: "format"
      when: "global"
    }

    {
      key: "Ctrl+S"
      action: "save"
      when: "global"
    }

    {
      key: "Ctrl+`"
      action: "command_palette"
      when: "terminal"
    }
  ]
}
```
