# 2148 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v1"
  routes: [
    {
      method: "GET"
      path: "/posts"
      handler: "deleteUsers"
      auth: false
    }

    {
      method: "PUT"
      path: "/comments/:id"
      handler: "updateUsers"
      auth: true
    }

    {
      method: "POST"
      path: "/orders"
      handler: "createOrders"
      auth: false
    }
  ]
}
```
