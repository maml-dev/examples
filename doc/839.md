# 839 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "systems"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 30 # seconds
    labels: {
      env: "staging"
      region: "us-east"
      team: "industrial"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 80
    memory_threshold: 89
    latency_p99_ms: 683
    error_rate: 0.01
    notification: {
      channel: "#oncall"
      severity: "info"
    }
  }
}
```
