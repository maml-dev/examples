# 178 - Game Config

Video game settings and preferences. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  display: {
    resolution: "2560x1440"
    fullscreen: false
    vsync: true
    fps_limit: 120
  }

  graphics: {
    quality: "high"
    shadows: "off"
    anti_aliasing: "MSAA"
    render_distance: 29
    ray_tracing: false
  }

  audio: {
    master: 95
    music: 68
    sfx: 65
    voice: 69
  }

  controls: {
    sensitivity: 2.4
    invert_y: false
    vibration: false
  }
}
```
