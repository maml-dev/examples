# 390 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@elastic-stir-fry.biz"
  subject: "Vinum urbanus conforto summisse."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Constans cedo ter cruciamentum turba corrumpo corrigo. Cilicium quis veritatis ante aeger cado studio subiungo acsi tabella. Doloribus blandior triduana video.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Vapulus aggero accommodo blanditiis sortitus perferendis. Sono curso civitas capitulus claudeo corona sum deserunt desipio adhuc. Atrox ventosus confugo cupio bellicus cauda infit deprimo.

Visit: {{url}}
"""
}
```
