# 2358 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v3"
  routes: [
    {
      method: "POST"
      path: "/comments"
      handler: "updatePosts"
      auth: true
    }

    {
      method: "PATCH"
      path: "/products/:id"
      handler: "getUsers"
      auth: false
    }

    {
      method: "PUT"
      path: "/products/:id"
      handler: "getPosts"
      auth: false
    }

    {
      method: "DELETE"
      path: "/products"
      handler: "updateProducts"
      auth: false
    }

    {
      method: "GET"
      path: "/orders"
      handler: "deletePosts"
      auth: false
    }

    {
      method: "GET"
      path: "/users"
      handler: "deleteProducts"
      auth: true
    }
  ]
}
```
