# 2733 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "biodegradable-tuxedo.net"
  inbound: {
    mx_records: [
      "mx1.biodegradable-tuxedo.net"
      "mx2.biodegradable-tuxedo.net"
    ]
    spam_filter: {
      enabled: true
      threshold: 5
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.biodegradable-tuxedo.net"
    port: 587
    tls: true
    rate_limit: {
      per_minute: 388
      per_hour: 5133
    }
  }
}
```
