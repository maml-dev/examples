# 1729 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "niches"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 30 # seconds
    labels: {
      env: "prod"
      region: "us-east"
      team: "sports"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 74
    memory_threshold: 93
    latency_p99_ms: 1569
    error_rate: 0.02
    notification: {
      channel: "#oncall"
      severity: "critical"
    }
  }
}
```
