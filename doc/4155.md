# 4155 - Font Stack

Typography font family definitions. This is an example of a [MAML](https://maml.dev) document.

{
  fonts: [
    {
      family: "JetBrains Mono"
      category: "display"
      weights: [
        100
        200
        400
        600
        900

      variable: true

      family: "Roboto"

        100

        800

    {
      family: "Space Grotesk"
      category: "serif"
      weights: [
        100
        200
        900
      ]
      variable: false
    }
  ]
}
