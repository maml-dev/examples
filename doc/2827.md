# 2827 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.hungry-council.com"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "15m"
    max_requests: 3198
    by: "ip"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://tight-pigpen.org"
      "https://app.appropriate-mechanic.net"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 3600
  }

  cache: {
    enabled: true
    ttl: 1545
    store: "redis"
  }
}
```
