# 571 - Dashboard Layout

Dashboard widget layout configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  title: "Sauer Group Dashboard"
  refresh_interval: 60 # seconds
  widgets: [
    {
      type: "chart"
      title: "Signups"
      source: "/api/metrics/revenue"
      chart_type: "area"
      period: "30d"
    }

    {
      type: "stat"
      title: "Total Users"
      source: "/api/stats/revenue"
      format: "percent"
    }

    {
      type: "table"
      title: "Latest Signups"
      source: "/api/recent/orders"
      limit: 8
    }
  ]

  theme: "light"
}
```
