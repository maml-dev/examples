# 3757 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "emacs"
  shortcuts: [
    {
      key: "Ctrl+B"
      action: "search_project"
      when: "debug"
    }

    {
      key: "Alt+Up"
      action: "debug_run"
      when: "editor"
    }

    {
      key: "Ctrl+S"
      action: "find_references"
      when: "terminal"
    }

    {
      key: "Ctrl+Shift+F"
      action: "command_palette"
      when: "editor"
    }
  ]
}
```
