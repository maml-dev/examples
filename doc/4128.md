# 4128 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "intent-worker"
    namespace: "default"
    labels: {
      app: "completion"
      tier: "data"
    }
  }

  # Deployment spec
  spec: {
    replicas: 1
    container: {
      image: "intervention:5.13.4"
      port: 9090
      resources: {
        cpu_limit: "500m"
        memory_limit: "1Gi"
      }
    }

    health_check: {
      path: "/ready"
      interval: 30
      timeout: 10
    }
  }
}
```
