# 938 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "technologies"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 60 # seconds
    labels: {
      env: "staging"
      region: "ap-south"
      team: "industrial"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 93
    memory_threshold: 94
    latency_p99_ms: 1493
    error_rate: 0.05
    notification: {
      channel: "#incidents"
      severity: "warning"
    }
  }
}
```
