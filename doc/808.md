# 808 - Proxy Config

Reverse proxy settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  listen: 8443
  server_name: "known-contractor.name"
  upstream: "http://localhost:5856"

  # TLS settings
  ssl: false
  http2: true

  gzip: true
  gzip_min_length: 1024
  client_max_body: "46mb"
}
```
