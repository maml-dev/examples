# 3240 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "icebreaker-api"
    namespace: "default"
    labels: {
      app: "cod"
      tier: "frontend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 5
    container: {
      image: "goat:2.5.6"
      port: 9090
      resources: {
        cpu_limit: "250m"
        memory_limit: "512Mi"
      }
    }

    health_check: {
      path: "/healthz"
      interval: 30
      timeout: 5
    }
  }
}
```
