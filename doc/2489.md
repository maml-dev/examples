# 2489 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@rubbery-makeover.name"
  subject: "Absens voluptatum concido condico ad testimonium."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Trans cupiditas strenuus aveho corrigo utor deleo contego assumenda studio. Cogito contra deludo charisma modi adfero consequatur abscido venia crepusculum. Admoneo accusamus studio excepturi sui aduro.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Tumultus cui vix tersus cui desino. Bardus cultura despecto occaecati ater aeneus. Tamen est victoria depraedor decet maxime cohors caput vulgaris.

Visit: {{url}}
"""
}
```
