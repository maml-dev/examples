# 2239 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.minor-hydrolyse.net"
    port: 8443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "15m"
    max_requests: 2065
    by: "api_key"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://accurate-hammock.biz"
      "https://app.inexperienced-porter.net"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 86400
  }

  cache: {
    enabled: true
    ttl: 850
    store: "memory"
  }
}
```
