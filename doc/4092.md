# 4092 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@lawful-sideboard.net"
  subject: "Tabula degenero asporto volup civis."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Paulatim volutabrum suscipio speciosus occaecati vester. Paens at suggero hic terminatio accendo utrum. Calculus apostolus cilicium carbo adicio bibo.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Benigne benigne uter turbo tergum. Tabgo vorax copia abutor. Vitiosus corona cognomen vos adopto sum aspernatur corrumpo cogito xiphias.

Visit: {{url}}
"""
}
```
