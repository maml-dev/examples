# 1456 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "emacs"
  shortcuts: [
    {
      key: "Alt+F7"
      action: "toggle_terminal"
      when: "global"
    }

    {
      key: "Ctrl+Shift+F"
      action: "toggle_comment"
      when: "editor"
    }

    {
      key: "Alt+Up"
      action: "save"
      when: "global"
    }

    {
      key: "Ctrl+Shift+F"
      action: "save"
      when: "editor"
    }

    {
      key: "Alt+F7"
      action: "save"
      when: "editor"
    }

    {
      key: "Cmd+Shift+L"
      action: "toggle_sidebar"
      when: "terminal"
    }
  ]
}
```
