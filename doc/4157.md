# 4157 - Cron Jobs

Scheduled task definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  timezone: "Europe/London"
  jobs: [
    {
      name: "cleanup-temp"
      schedule: "0 * * * *"
      command: "node scripts/cleanup.js"
      enabled: false
    }

    {
      name: "cleanup-temp"
      schedule: "30 6 * * 1-5"
      command: "./bin/sync --all"
      enabled: false
    }

    {
      name: "sync-data"
      schedule: "0 2 * * 0"
      command: "python manage.py report"
      enabled: true
    }

    {
      name: "send-reports"
      schedule: "30 6 * * 1-5"
      command: "node scripts/cleanup.js"
      enabled: false
    }

    {
      name: "backup-db"
      schedule: "0 * * * *"
      command: "python manage.py report"
      enabled: false
    }

    {
      name: "sync-data"
      schedule: "0 0 * * *"
      command: "pg_dump $DATABASE_URL"
      enabled: true
    }
  ]
}
```
