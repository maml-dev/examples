# 2934 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "glorious-confusion.biz"
  inbound: {
    mx_records: [
      "mx1.glorious-confusion.biz"
      "mx2.glorious-confusion.biz"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.glorious-confusion.biz"
    port: 587
    tls: true
    rate_limit: {
      per_minute: 211
      per_hour: 6194
    }
  }
}
```
