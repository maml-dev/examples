# 128 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.fat-quart.com"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "15m"
    max_requests: 4515
    by: "ip"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://skeletal-dime.org"
      "https://app.glorious-joy.info"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 3600
  }

  cache: {
    enabled: true
    ttl: 2983
    store: "redis"
  }
}
```
