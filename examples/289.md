# 289 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@cute-commercial.biz"
  subject: "Articulus tener aggero vorax earum via."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Utpote denique architecto taceo audentia id distinctio. Deripio adeptio velit apto debitis summa aliquam volup. Vigilo cognatus adsidue.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Copiose esse quis fugiat cubo magni eveniet constans aggero aetas. Bestia deputo debilito spoliatio voco viriliter vestigium conscendo. Subiungo sonitus summopere.

Visit: {{url}}
"""
}
```
