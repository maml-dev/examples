# 049 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "sublime"
  shortcuts: [
    {
      key: "Ctrl+Shift+F"
      action: "search_project"
      when: "global"
    }

    {
      key: "Ctrl+/"
      action: "find_references"
      when: "terminal"
    }

    {
      key: "Ctrl+B"
      action: "toggle_sidebar"
      when: "global"
    }

    {
      key: "Ctrl+Shift+F"
      action: "toggle_comment"
      when: "global"
    }

    {
      key: "Ctrl+Shift+F"
      action: "format"
      when: "terminal"
    }
  ]
}
```
