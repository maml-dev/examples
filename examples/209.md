# 209 - SSH Config

SSH client host configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  host: "dev-server"
  hostname: "29.253.212.195"
  user: "flavio81"
  port: 22
  identity_file: "~/.ssh/id_rsa"

  forward_agent: false
  server_alive_interval: 60
  compression: true
}
```
