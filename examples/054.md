# 054 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "editor"
      permissions: [
        "read"
        "manage_users"
        "view_analytics"
        "export_data"
      ]
      max_seats: 37
    }

    {
      name: "billing"
      permissions: [
        "manage_billing"
        "write"
        "read"
      ]
      max_seats: null
    }

    {
      name: "moderator"
      permissions: [
        "write"
        "delete"
        "read"
      ]
      max_seats: null
    }
  ]
}
```
