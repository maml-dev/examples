# 040 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "swing-web"
    namespace: "staging"
    labels: {
      app: "embossing"
      tier: "frontend"
    }
  }

  # Deployment spec
  spec: {
    replicas: 3
    container: {
      image: "bench:5.4.5"
      port: 8000
      resources: {
        cpu_limit: "250m"
        memory_limit: "1Gi"
      }
    }

    health_check: {
      path: "/healthz"
      interval: 15
      timeout: 3
    }
  }
}
```
