# 008 - Proxy Config

Reverse proxy settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  listen: 8080
  server_name: "bony-spring.com"
  upstream: "http://127.0.0.1:4124"

  # TLS settings
  ssl: true
  http2: true

  gzip: true
  gzip_min_length: 256
  client_max_body: "18mb"
}
```
