# 085 - API Mock

Mock API response payloads for testing. This is an example of a [MAML](https://maml.dev) document.

```maml
# Mock API definition
{
  endpoint: "/api/v1/users"
  method: "GET"
  status: 200
  # JSON response body
  response: """
[
  {
    "id": 1,
    "name": "Reginald Boyer",
    "email": "Sister_Becker14@gmail.com",
    "role": "admin"
  },
  {
    "id": 2,
    "name": "Clifford Quigley",
    "email": "Lelia77@hotmail.com",
    "role": "user"
  }
]
"""

    Content-Type: "application/json"

    X-Request-Id: "113bcfdf-86b9-4364-a1ee-c2666e9d8440"
  }
  delay_ms: 451 # simulated latency
}
```
