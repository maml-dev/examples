# 240 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "vivacious-tuba.com"
  inbound: {
    mx_records: [
      "mx1.vivacious-tuba.com"
      "mx2.vivacious-tuba.com"
    ]
    spam_filter: {
      enabled: true
      threshold: 5
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.vivacious-tuba.com"
    port: 587
    tls: true
    rate_limit: {
      per_minute: 201
      per_hour: 7379
    }
  }
}
```
