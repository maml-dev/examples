# 295 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@elementary-pneumonia.info"
  subject: "Debilito suus deleniti cunae centum."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Depono deinde rerum adulatio corrigo odio conor. Allatus congregatio vulgo consuasor magnam ver. Suscipio ultra autus curo sordeo aduro.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Sulum quaerat perferendis. Barba ciminatio utpote conservo articulus modi volva cum suasoria. Aequus laboriosam creator defungo thorax cui solus delectatio spargo.

Visit: {{url}}
"""
}
```
