# 024 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "considerate-lace.biz"
  inbound: {
    mx_records: [
      "mx1.considerate-lace.biz"
      "mx2.considerate-lace.biz"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "tag"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 2048
    }
  }

  outbound: {
    relay: "smtp.considerate-lace.biz"
    port: 25
    tls: true
    rate_limit: {
      per_minute: 371
      per_hour: 2929
    }
  }
}
```
