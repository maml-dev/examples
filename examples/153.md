# 153 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "sublime"
  shortcuts: [
    {
      key: "Cmd+Shift+L"
      action: "select_all_occurrences"
      when: "terminal"
    }

    {
      key: "Ctrl+S"
      action: "format"
      when: "editor"
    }

    {
      key: "Ctrl+B"
      action: "find_references"
      when: "debug"
    }

    {
      key: "F5"
      action: "find_references"
      when: "editor"
    }

    {
      key: "F5"
      action: "find_references"
      when: "debug"
    }
  ]
}
```
