# 124 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "redis"
  connection: {
    host: "mq.internal"
    port: 5672
    vhost: "/staging"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 8
      retry: 3
      timeout: 48785 # ms
    }

    priority: {
      concurrency: 7
      retry: 3
      timeout: 5811 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 6
    queue: "dead-letter"
  }
}
```
