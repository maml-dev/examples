# 140 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.homely-adaptation.org"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "1m"
    max_requests: 2795
    by: "user"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://powerless-vision.com"
      "https://app.lost-doorpost.org"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 7200
  }

  cache: {
    enabled: true
    ttl: 163
    store: "memory"
  }
}
```
