# 103 - Logger Config

Structured logging configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  level: "debug"
  format: "logfmt"
  timestamp: true
  caller: true # include file:line

  output: "stderr"
  rotation: false
  max_size_mb: 204
  max_age_days: 20
  compress: true
}
```
