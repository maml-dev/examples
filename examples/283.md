# 283 - Quoted Keys

Configuration using special characters in keys. This is an example of a [MAML](https://maml.dev) document.

```maml
# HTTP Headers - keys need quoting
{
  Content-Type: "application/json"
  X-Request-ID: "fb00397c-4619-4d7e-b351-73ea4c73017d"
  Accept-Language: "ja-JP"
  Cache-Control: "public, max-age=86400"
  X-Powered-By: "Express"

  # Keys with spaces get quoted
  "headers with spaces": {
    "custom header": "claro"
    "another one": "volaticus"
  }
}
```
