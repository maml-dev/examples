# 058 - DNS Records

DNS zone record entries. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  zone: "stormy-extension.org"
  records: [
    {
      type: "MX"
      name: "*"
      value: "live-retrospectivity.name"
      ttl: 300
    }

    {
      type: "NS"
      name: "www"
      value: "scary-custody.com"
      ttl: 86400
    }

    {
      type: "A"
      name: "mail"
      value: "needy-fellow.info"
      ttl: 3600
    }

    {
      type: "TXT"
      name: "www"
      value: "young-necklace.org"
      ttl: 86400
    }
  ]
}
```
