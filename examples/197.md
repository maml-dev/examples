# 197 - Quoted Keys

Configuration using special characters in keys. This is an example of a [MAML](https://maml.dev) document.

```maml
# HTTP Headers - keys need quoting
{
  Content-Type: "application/xml"
  X-Request-ID: "81f71365-df32-45d1-be92-2cf9779f8090"
  Accept-Language: "en-US"
  Cache-Control: "no-cache"
  X-Powered-By: "Express"

  # Keys with spaces get quoted
  "headers with spaces": {
    "custom header": "quidem"
    "another one": "curso"
  }
}
```
