# 051 - Cron Jobs

Scheduled task definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  timezone: "UTC"
  jobs: [
    {
      name: "refresh-cache"
      schedule: "*/15 * * * *"
      command: "pg_dump $DATABASE_URL"
      enabled: false
    }

    {
      name: "rotate-logs"
      schedule: "0 * * * *"
      command: "./bin/sync --all"
      enabled: true
    }

    {
      name: "backup-db"
      schedule: "0 * * * *"
      command: "python manage.py report"
      enabled: false
    }

    {
      name: "refresh-cache"
      schedule: "0 2 * * 0"
      command: "./bin/sync --all"
      enabled: false
    }
  ]
}
```
