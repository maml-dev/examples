# 133 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "sqs"
  connection: {
    host: "localhost"
    port: 6379
    vhost: "/"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 2
      retry: 4
      timeout: 16478 # ms
    }

    priority: {
      concurrency: 8
      retry: 3
      timeout: 9870 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 8
    queue: "dead-letter"
  }
}
```
