# 205 - Locale Config

Regional and locale settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  language: "zh"
  country: "ZW"
  timezone: "America/Araguaina"
  currency: "RWF"

  date_format: "DD/MM/YYYY"
  time_format: "12h"
  first_day_of_week: "sunday"
  decimal_separator: ","
}
```
