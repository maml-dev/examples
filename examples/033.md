# 033 - Terraform Resource

Infrastructure as code resource definition. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  provider: "aws"
  region: "asia-southeast1"
  resources: {
    # Network
    vpc: {
      cidr_block: "10.1.0.0/16"
      enable_dns: true
      tags: {
        Name: "farm-vpc"
        Environment: "staging"
      }
    }

    # Compute
    instance: {
      ami: "ami-affeef13"
      type: "m5.large"
      key_name: "pile-key"
      monitoring: true
      root_volume: {
        size: 50
        type: "gp2"
        encrypted: true
      }
    }

    # Security
    security_group: {
      name: "allow-web"
      ingress_port: 443
      ingress_cidr: "0.0.0.0/0"
    }
  }
}
```
