# 134 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.straight-help.biz"
    port: 8443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "1m"
    max_requests: 4703
    by: "user"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://handy-packaging.net"
      "https://app.menacing-experience.biz"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 86400
  }

  cache: {
    enabled: true
    ttl: 2691
    store: "memory"
  }
}
```
