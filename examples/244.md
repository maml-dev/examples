# 244 - Font Stack

Typography font family definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  fonts: [
    {
      family: "Inter"
      category: "serif"
      weights: [
        100
        300
        600
        900
      ]
      variable: true
    }

    {
      family: "Geist"
      category: "monospace"
      weights: [
        300
        400
        800
        900
      ]
      variable: true
    }

    {
      family: "Berkeley Mono"
      category: "display"
      weights: [
        300
        500
        600
        900
      ]
      variable: true
    }
  ]
}
```
