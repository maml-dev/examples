# 039 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "metrics"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 10 # seconds
    labels: {
      env: "staging"
      region: "eu-west"
      team: "computers"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 76
    memory_threshold: 75
    latency_p99_ms: 117
    error_rate: 0.01
    notification: {
      channel: "#incidents"
      severity: "info"
    }
  }
}
```
