# 121 - Terraform Resource

Infrastructure as code resource definition. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  provider: "google"
  region: "eu-west-1"
  resources: {
    # Network
    vpc: {
      cidr_block: "10.64.0.0/16"
      enable_dns: true
      tags: {
        Name: "tomatillo-vpc"
        Environment: "production"
      }
    }

    # Compute
    instance: {
      ami: "ami-bcbeacac"
      type: "m5.large"
      key_name: "yin-key"
      monitoring: true
      root_volume: {
        size: 20
        type: "io1"
        encrypted: true
      }
    }

    # Security
    security_group: {
      name: "allow-all"
      ingress_port: 22
      ingress_cidr: "0.0.0.0/0"
    }
  }
}
```
