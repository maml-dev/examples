# 253 - Roles & Permissions

Role-based access control definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  roles: [
    {
      name: "viewer"
      permissions: [
        "read"
        "view_analytics"
      ]
      max_seats: 16
    }

    {
      name: "editor"
      permissions: [
        "manage_billing"
        "delete"
      ]
      max_seats: 43
    }

    {
      name: "editor"
      permissions: [
        "write"
        "view_analytics"
      ]
      max_seats: null
    }

    {
      name: "admin"
      permissions: [
        "view_analytics"
        "export_data"
        "manage_users"
      ]
      max_seats: null
    }
  ]
}
```
