# 158 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v2"
  routes: [
    {
      method: "DELETE"
      path: "/products"
      handler: "listOrders"
      auth: false
    }

    {
      method: "GET"
      path: "/posts/:id"
      handler: "updateOrders"
      auth: true
    }

    {
      method: "PATCH"
      path: "/comments/:id"
      handler: "deletePosts"
      auth: true
    }
  ]
}
```
