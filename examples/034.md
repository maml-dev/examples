# 034 - API Gateway

API gateway routing and middleware configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  gateway: {
    host: "api.impartial-plastic.biz"
    port: 443
    protocol: "https"
  }

  # Rate limiting
  rate_limit: {
    enabled: true
    window: "15m"
    max_requests: 8625
    by: "api_key"
  }

  # CORS policy
  cors: {
    enabled: true
    origins: [
      "https://punctual-roundabout.info"
      "https://app.nautical-jazz.name"
    ]
    methods: [
      "GET"
      "POST"
      "PUT"
      "DELETE"
    ]
    max_age: 7200
  }

  cache: {
    enabled: false
    ttl: 499
    store: "memory"
  }
}
```
