# 252 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v3"
  routes: [
    {
      method: "GET"
      path: "/posts/:id"
      handler: "getProducts"
      auth: false
    }

    {
      method: "PATCH"
      path: "/products"
      handler: "createPosts"
      auth: true
    }

    {
      method: "POST"
      path: "/posts"
      handler: "listPosts"
      auth: true
    }

    {
      method: "DELETE"
      path: "/users"
      handler: "createUsers"
      auth: true
    }

    {
      method: "POST"
      path: "/products"
      handler: "createProducts"
      auth: true
    }

    {
      method: "PUT"
      path: "/posts"
      handler: "updateUsers"
      auth: false
    }
  ]
}
```
