# 126 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "experiences"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 10 # seconds
    labels: {
      env: "prod"
      region: "eu-west"
      team: "electronics"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 93
    memory_threshold: 77
    latency_p99_ms: 1928
    error_rate: 0.05
    notification: {
      channel: "#ops-alerts"
      severity: "info"
    }
  }
}
```
