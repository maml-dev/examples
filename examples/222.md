# 222 - Backup Strategy

Backup schedule and retention policy. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  source: {
    type: "mongodb"
    host: "localhost"
    databases: [
      "yak"
      "reach"
    ]
  }

  # Backup schedule
  schedule: {
    full: "0 2 * * 0" # weekly Sunday 2am
    incremental: "0 2 * * 1-6" # daily except Sunday
    snapshot: "0 */4 * * *" # every 4 hours
  }

  retention: {
    daily: 11
    weekly: 12
    monthly: 7
  }

  destination: {
    type: "gcs"
    path: "s3://backups-vein"
    encryption: true
    compression: "zstd"
  }
}
```
