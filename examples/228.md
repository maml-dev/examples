# 228 - Theme Config

Design system theme configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  name: "shadowy Theme"
  # Color palette
  colors: {
    primary: "#0b3962"
    secondary: "#f5adc6"
    accent: "#0ea1cd"
    background: "#f8f9fa"
    text: "#1a1a1a"

    error: "#dc3545"
    warning: "#ffc107"
    success: "#28a745"
  }

  # Typography
  typography: {
    font_family: "Inter, sans-serif"
    base_size: 14
    line_height: 1.5
    headings: {
      font_family: "\"Playfair Display\", serif"
      weight: 800
    }
  }

  spacing: {
    unit: 4
    scale: [
      0
      4
      8
      16
      24
      32
      48
    ]
  }
  border_radius: 4
}
```
