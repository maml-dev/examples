# 260 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v2"
  routes: [
    {
      method: "DELETE"
      path: "/products"
      handler: "getUsers"
      auth: true
    }

    {
      method: "PUT"
      path: "/comments"
      handler: "deleteUsers"
      auth: false
    }

    {
      method: "POST"
      path: "/products/:id"
      handler: "deleteOrders"
      auth: true
    }
  ]
}
```
