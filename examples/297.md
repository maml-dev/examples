# 297 - Quoted Keys

Configuration using special characters in keys. This is an example of a [MAML](https://maml.dev) document.

```maml
# HTTP Headers - keys need quoting
{
  Content-Type: "text/html"
  X-Request-ID: "28630821-7de1-4dc9-85de-ff4917597102"
  Accept-Language: "en-US"
  Cache-Control: "no-cache"
  X-Powered-By: "Hono"

  # Keys with spaces get quoted
  "headers with spaces": {
    "custom header": "trepide"
    "another one": "vitiosus"
  }
}
```
