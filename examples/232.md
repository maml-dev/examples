# 232 - Email Routing

Email routing and filtering rules. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  domain: "scary-story.org"
  inbound: {
    mx_records: [
      "mx1.scary-story.org"
      "mx2.scary-story.org"
    ]
    spam_filter: {
      enabled: true
      threshold: 8.5
      action: "quarantine"
    }

    dkim: {
      enabled: true
      selector: "default"
      key_size: 1024
    }
  }

  outbound: {
    relay: "smtp.scary-story.org"
    port: 25
    tls: true
    rate_limit: {
      per_minute: 263
      per_hour: 7627
    }
  }
}
```
