# 200 - Quoted Keys

Configuration using special characters in keys. This is an example of a [MAML](https://maml.dev) document.

```maml
# HTTP Headers - keys need quoting
{
  Content-Type: "text/html"
  X-Request-ID: "23b58e5c-8fc7-4b20-a8c3-23e34bb1ae2a"
  Accept-Language: "fr-FR"
  Cache-Control: "max-age=3600"
  X-Powered-By: "Koa"

  # Keys with spaces get quoted
  "headers with spaces": {
    "custom header": "stips"
    "another one": "color"
  }
}
```
