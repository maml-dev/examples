# 001 - Logger Config

Structured logging configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  level: "debug"
  format: "logfmt"
  timestamp: true
  caller: false # include file:line

  output: "stderr"
  rotation: true
  max_size_mb: 289
  max_age_days: 54
  compress: true
}
```
