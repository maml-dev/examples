# 149 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v2"
  routes: [
    {
      method: "DELETE"
      path: "/products/:id"
      handler: "createOrders"
      auth: true
    }

    {
      method: "POST"
      path: "/comments/:id"
      handler: "deleteUsers"
      auth: true
    }

    {
      method: "DELETE"
      path: "/orders"
      handler: "listProducts"
      auth: true
    }

    {
      method: "PUT"
      path: "/orders/:id"
      handler: "deleteProducts"
      auth: false
    }

    {
      method: "POST"
      path: "/posts"
      handler: "updatePosts"
      auth: false
    }
  ]
}
```
