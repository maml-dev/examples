# 123 - Queue Config

Message queue and worker configuration. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  broker: "kafka"
  connection: {
    host: "mq.internal"
    port: 9092
    vhost: "/"
  }

  # Queue definitions
  queues: {
    default: {
      concurrency: 6
      retry: 3
      timeout: 7939 # ms
    }

    priority: {
      concurrency: 8
      retry: 4
      timeout: 5172 # ms
    }
  }

  dead_letter: {
    enabled: true
    max_retries: 3
    queue: "dead-letter"
  }
}
```
