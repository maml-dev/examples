# 030 - Backup Strategy

Backup schedule and retention policy. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  source: {
    type: "mysql"
    host: "db.internal"
    databases: [
      "tribe"
      "case"
    ]
  }

  # Backup schedule
  schedule: {
    full: "0 2 * * 0" # weekly Sunday 2am
    incremental: "0 2 * * 1-6" # daily except Sunday
    snapshot: "0 */4 * * *" # every 4 hours
  }

  retention: {
    daily: 12
    weekly: 12
    monthly: 17
  }

  destination: {
    type: "s3"
    path: "/mnt/backups/retention"
    encryption: true
    compression: "lz4"
  }
}
```
