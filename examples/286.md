# 286 - Quoted Keys

Configuration using special characters in keys. This is an example of a [MAML](https://maml.dev) document.

```maml
# HTTP Headers - keys need quoting
{
  Content-Type: "application/json"
  X-Request-ID: "1f0fffea-2f3e-4cfa-8b82-ec39a5eb5338"
  Accept-Language: "fr-FR"
  Cache-Control: "no-cache"
  X-Powered-By: "Koa"

  # Keys with spaces get quoted
  "headers with spaces": {
    "custom header": "cupiditas"
    "another one": "aestivus"
  }
}
```
