# 108 - Proxy Config

Reverse proxy settings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  listen: 80
  server_name: "zealous-hyena.com"
  upstream: "http://localhost:5349"

  # TLS settings
  ssl: false
  http2: true

  gzip: true
  gzip_min_length: 256
  client_max_body: "23mb"
}
```
