# 029 - K8s Deployment

Kubernetes deployment specification. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  apiVersion: "apps/v1"
  kind: "Deployment"
  metadata: {
    name: "blight-worker"
    namespace: "staging"
    labels: {
      app: "footrest"
      tier: "data"
    }
  }

  # Deployment spec
  spec: {
    replicas: 5
    container: {
      image: "birdbath:2.20.19"
      port: 8080
      resources: {
        cpu_limit: "500m"
        memory_limit: "256Mi"
      }
    }

    health_check: {
      path: "/ready"
      interval: 15
      timeout: 5
    }
  }
}
```
