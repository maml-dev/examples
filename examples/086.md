# 086 - HTML Templates

Email HTML templates with raw string content. This is an example of a [MAML](https://maml.dev) document.

```maml
# Email template configuration
{
  from: "noreply@remarkable-viability.org"
  subject: "Tunc adficio beatus."
  # HTML email body
  template: """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Welcome</title>
</head>
<body>
  <h1>Hello, {{name}}!</h1>
  <p>Cunae curia at repellat adfectus coniuratio cotidie coma canis. Celebrer contigo vulgaris. Facilis temptatio subnecto.</p>
  <a href="{{url}}">Get Started</a>
</body>
</html>
"""

  # Plain text fallback
  plain_text: """
Hello, {{name}}!

Verto coerceo asperiores. Velit deprecator confido creta creator enim votum. Fuga administratio dolorem natus aureus consequuntur timor chirographum ultio aegre.

Visit: {{url}}
"""
}
```
