# 031 - Monitoring Config

Application monitoring and alerting setup. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  service: "e-commerce"
  # Prometheus metrics
  metrics: {
    endpoint: "/metrics"
    interval: 10 # seconds
    labels: {
      env: "prod"
      region: "us-east"
      team: "grocery"
    }
  }

  # Alert thresholds
  alerts: {
    cpu_threshold: 86
    memory_threshold: 88
    latency_p99_ms: 1511
    error_rate: 0.01
    notification: {
      channel: "#incidents"
      severity: "info"
    }
  }
}
```
