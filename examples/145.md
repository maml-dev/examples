# 145 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v1"
  routes: [
    {
      method: "GET"
      path: "/users"
      handler: "deleteProducts"
      auth: false
    }

    {
      method: "DELETE"
      path: "/users/:id"
      handler: "listUsers"
      auth: false
    }

    {
      method: "POST"
      path: "/users"
      handler: "createProducts"
      auth: true
    }

    {
      method: "PUT"
      path: "/comments/:id"
      handler: "getUsers"
      auth: false
    }

    {
      method: "PUT"
      path: "/products"
      handler: "listOrders"
      auth: false
    }
  ]
}
```
