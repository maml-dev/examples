# 257 - Keyboard Shortcuts

Editor keyboard shortcut bindings. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  editor: "emacs"
  shortcuts: [
    {
      key: "Ctrl+/"
      action: "toggle_sidebar"
      when: "terminal"
    }

    {
      key: "Alt+Up"
      action: "search_project"
      when: "debug"
    }

    {
      key: "Ctrl+`"
      action: "search_project"
      when: "terminal"
    }

    {
      key: "Ctrl+/"
      action: "save"
      when: "editor"
    }

    {
      key: "Ctrl+D"
      action: "search_project"
      when: "terminal"
    }

    {
      key: "Alt+F7"
      action: "select_all_occurrences"
      when: "global"
    }
  ]
}
```
