# 156 - API Routes

REST API route definitions. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  base_url: "/api/v3"
  routes: [
    {
      method: "PUT"
      path: "/orders"
      handler: "deleteUsers"
      auth: true
    }

    {
      method: "GET"
      path: "/posts/:id"
      handler: "getPosts"
      auth: false
    }

    {
      method: "GET"
      path: "/products/:id"
      handler: "getPosts"
      auth: false
    }
  ]
}
```
