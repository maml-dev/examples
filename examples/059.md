# 059 - User Directory

A list of user accounts with roles. This is an example of a [MAML](https://maml.dev) document.

```maml
{
  users: [
    {
      id: 1
      name: "Bonnie Purdy"
      email: "Alisa_Kassulke@gmail.com"
      role: "moderator"
      active: false
    }

    {
      id: 2
      name: "Myra Considine"
      email: "Stella15@yahoo.com"
      role: "moderator"
      active: false
    }

    {
      id: 3
      name: "Angelica Terry MD"
      email: "Andrew6@yahoo.com"
      role: "viewer"
      active: true
    }
  ]
}
```
